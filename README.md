# 🧙‍♂️ Tech Daily Digest

Daily curated summaries from top tech sources, compiled by Yoda.

## Structure
```
tech-daily-digest/
├── YYYY/MM/DD/                   ← 일일 다이제스트 (자동)
│   ├── README.md                 ← 인덱스
│   ├── hackernews/{slug}.md      ← HN 5건
│   ├── geeknews/{slug}.md       ← GeekNews 10건
│   └── github-trending/{slug}.md ← GitHub Trending 10건
│
├── articles/                     ← 별도 요청 심층 분석 (영구 보관)
│   ├── ai-engineering/           ← AI 에이전트, 하네스, AInD
│   ├── dev-tools/                ← 개발 도구, 메타 도구
│   ├── on-device-ai/             ← 온디바이스/엣지 AI
│   ├── productivity/             ← 생산성, 전략, 리더십
│   └── README.md
│
└── scripts/                      ← 생성 가이드
```

## Daily Digest
- **Schedule:** 매일 08:00 KST
- **Sources:** Hacker News (5), GeekNews (10), GitHub Trending (10)
- **Format:** 개별 .md 파일, 15줄+ 상세 요약

## Articles (별도 요청)
- **Trigger:** Soo의 요청으로 생성
- **Location:** `articles/{대분류}/{날짜}_{제목}.md`
- **Lifecycle:** 매월 말 리스팅 → Soo 확인 → 삭제/보관 결정
