# 🧙‍♂️ Tech Daily Digest

Daily curated summaries from top tech sources, compiled by Yoda.

## Sources
- **Hacker News** (news.ycombinator.com) — Top 5 stories
- **GeekNews** (news.hada.io) — Top 10 stories (Korean tech community)
- **GitHub Trending** — Top 10 repositories

## Structure
```
YYYY/MM/DD/
├── README.md              ← 인덱스 (전체 목차)
├── hackernews/            ← HN 상위 5건 (개별 파일)
│   └── {제목-slug}.md
├── geeknews/              ← GeekNews 상위 10건 (개별 파일)
│   └── {제목-slug}.md
└── github-trending/       ← GitHub Trending 상위 10건 (개별 파일)
    └── {제목-slug}.md
```

## Format Standard
- 각 기사별 개별 .md 파일
- 최소 15줄 이상 상세 요약
- 출처 링크, 토론 링크, 태그 포함
- "Soo에게 의미 있는 이유" 섹션 포함
- 2일 이상 연속 트렌딩 시 🔥 표시 및 성장률 비교

## Schedule
Updated daily at 08:00 KST (Asia/Seoul).
