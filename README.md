- **GeekNews** (news.hada.io) — Korean tech community
- **GitHub Trending** — Trending repositories

## Structure
```
YYYY/
  MM/
```

No folder exceeds 10 documents — subcategories are created as needed.

## Schedule
Updated daily at 08:00 KST (Asia/Seoul).
