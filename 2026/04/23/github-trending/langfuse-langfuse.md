## 상세 요약

Langfuse는 **오픈소스 LLM 엔지니어링 플랫폼**으로, 팀이 협업하여 LLM 애플리케이션을 관찰, 디버그, 평가, 개선할 수 있게 해준다. Y Combinator W23 출신으로, ClickHouse 기반으로 구축되었다.

**핵심 기능:**
- **트레이싱**: LLM 호출, 에이전트 실행, RAG 파이프라인의 전체 추적
- **평가**: 자동화된 평가 파이프라인으로 모델 품질 측정
- **프롬프트 관리**: 버전 관리, A/B 테스트, 롤백
- **비용 분석**: 모델별, 사용자별, 기능별 토큰 비용 추적
- **데이터셋**: 평가 데이터셋 관리 및 벤치마킹
- **셀프 호스팅**: MIT 라이선스, Docker로 자체 배포 가능

최근 오픈소스에 더욱 집중하겠다는 발표와 함께 다시 트렌딩에 올랐다. Python과 JavaScript SDK 모두 지원.

## Soo에게 의미 있는 이유

AInD 컨설팅에서 LLM 기반 프로덕트의 관찰성(observability)은 필수 인프라다. 고객에게 "AI를 도입한 후 어떻게 모니터링하고 개선할 것인가"에 대한 답으로 Langfuse를 제안할 수 있다.
