# koala73/worldmonitor — 실시간 글로벌 인텔리전스 대시보드

- **GitHub**: [koala73/worldmonitor](https://github.com/koala73/worldmonitor)
- **태그**: `#OSINT` `#대시보드` `#뉴스` `#지정학` `#TypeScript`

## 상세 요약

AI 기반 뉴스 집계, 지정학적 모니터링, 인프라 추적을 통합한 **실시간 상황 인식 인터페이스**. 단일 코드베이스에서 5개 사이트 변형(world, tech, finance, commodity, happy)을 운영한다.

**주요 기능:**
- **500+ 큐레이팅된 뉴스 피드**: 15개 카테고리, AI 요약 생성
- **듀얼 맵 엔진**: 3D 글로브(globe.gl) + WebGL 플랫 맵(deck.gl), 45개 데이터 레이어
- **크로스 스트림 상관분석**: 군사, 경제, 재난, 에스컬레이션 신호 수렴 분석
- **Country Intelligence Index**: 12개 신호 카테고리의 복합 위험 점수
- **금융 레이더**: 92개 증권거래소, 원자재, 암호화폐, 7-신호 시장 복합지수
- **로컬 AI**: Ollama로 전부 구동 가능, API 키 불필요
- **네이티브 데스크톱 앱**: Tauri 2 기반 macOS/Windows/Linux
- **21개 언어** 지원 (RTL 포함)

```bash
git clone https://github.com/koala73/worldmonitor.git
cd worldmonitor && npm install && npm run dev
```

## Soo에게 의미 있는 이유

기술 트렌드 모니터링과 글로벌 뉴스 추적을 자동화하는 도구. AInD 컨설팅에서 시장 분석, 경쟁사 동향 파악에 활용 가능하며, "AI 기반 인텔리전스 대시보드" 자체가 고객에게 제안할 수 있는 프로젝트 유형.
