## 상세 요약

홍콩대학교(HKUDS)에서 개발한 **차세대 멀티모달 RAG 시스템**으로, LightRAG를 기반으로 텍스트, 이미지, 테이블, 수식을 모두 처리하는 통합 RAG 프레임워크다.

**핵심 기능:**
- **멀티모달 쿼리**: 텍스트, 이미지, 테이블, 수식을 포함한 문서 처리
- **VLM-Enhanced Query**: 이미지가 포함된 문서에서 VLM을 활용한 멀티모달 분석
- **컨텍스트 인식 처리**: 관련 문맥 정보를 지능적으로 통합
- **LightRAG 기반**: 경량 그래프 RAG 아키텍처
- PyPI 패키지로 쉬운 설치 (`pip install raganything`)
- uv 지원

arXiv 논문(2510.12323)으로 기술적 상세가 공개되어 있으며, GitHub 1k+ 스타를 달성한 검증된 프로젝트.

## Soo에게 의미 있는 이유

RAG는 AInD 컨설팅의 핵심 기술 스택 중 하나. 특히 멀티모달 RAG는 기업 문서(PDF, 프레젠테이션, 보고서)에 이미지와 표가 혼재하는 현실적 시나리오를 해결한다. 고객 프로젝트에서 바로 적용 가능한 프레임워크.
