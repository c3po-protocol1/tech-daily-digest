# AIDC-AI/Pixelle-Video — AI 전자동 숏폼 비디오 생성 엔진

- **GitHub**: [AIDC-AI/Pixelle-Video](https://github.com/AIDC-AI/Pixelle-Video)
- **태그**: `#영상생성` `#AI` `#ComfyUI` `#TTS` `#자동화`

## 상세 요약

주제 하나만 입력하면 **자동으로 완전한 숏폼 비디오를 생성**하는 오픈소스 AI 엔진. 문안 작성, AI 이미지/비디오 생성, 음성 합성, BGM 추가, 최종 합성까지 원스톱으로 처리한다.

**생성 파이프라인:**
1. ✍️ 비디오 스크립트 작성 (GPT, Qwen, DeepSeek, Ollama 등 지원)
2. 🎨 AI 이미지/비디오 생성 (WAN 2.1, FLUX 등)
3. 🗣️ 음성 합성 (Edge-TTS, Index-TTS 등)
4. 🎵 BGM 추가
5. 🎬 최종 비디오 합성

**추가 기능:**
- 디지털 휴먼 립싱크 (아바타 기반 발표)
- 이미지-to-비디오 파이프라인
- 모션 트랜스퍼 (참조 비디오의 동작을 다른 이미지에 적용)
- ComfyUI 기반 모듈식 아키텍처 (워크플로우 커스터마이징 가능)
- Web UI 제공, Windows 통합 패키지
- RunningHub 클라우드 GPU 지원

**최근 업데이트:**
- 2026-01: 모션 트랜스퍼, 디지털 휴먼, 다국어 TTS
- 2025-12: 커스텀 소재 업로드, 배치 작업, 역사 기록

## Soo에게 의미 있는 이유

AI 기반 콘텐츠 자동 생성은 마케팅, 교육, 내부 커뮤니케이션에서 수요가 폭발적. AInD 컨설팅에서 "AI로 콘텐츠 생산 비용을 90% 줄이기" 시나리오의 실증 도구로 활용 가능.
