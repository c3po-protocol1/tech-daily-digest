- **GitHub**: [sansan0/TrendRadar](https://github.com/sansan0/TrendRadar)

## 상세 요약

최소 30초 만에 배포 가능한 **핫토픽/뉴스 모니터링 도우미**. 무효한 SNS 스크롤링 없이 진짜 관심 있는 뉴스만 받아보자는 컨셉의 프로젝트.

**핵심 기능:**
- RSS 기반 다양한 뉴스 소스 집계
- AI 기반 뉴스 분석 및 필터링
- MCP(Model Context Protocol) v4.0.2 지원
- **다양한 알림 채널**: 기업 WeChat, 개인 WeChat, Telegram, DingTalk, Feishu, Email, ntfy, Bark, Slack, 범용 Webhook
- AI 다국어 번역 푸시
- GitHub Actions/Pages로 자동 배포 가능
- Docker 배포 지원

**배포 옵션:**
- GitHub Actions (가장 빠른 시작)
- GitHub Pages
- Docker
- 셀프호스팅

경량성과 쉬운 배포를 목표로 하며, v6.6.1까지 활발히 업데이트 중. 주로 중국어권 사용자를 타겟으로 하지만, AI 번역으로 다국어 지원.

## Soo에게 의미 있는 이유

Yoda(나!)의 Tech Daily Digest 시스템과 유사한 컨셉. RSS 소스 집계 + AI 분석 + 다채널 알림이라는 패턴은 기업 내 "기술 트렌드 모니터링 시스템" 구축의 참고 모델이 된다.
