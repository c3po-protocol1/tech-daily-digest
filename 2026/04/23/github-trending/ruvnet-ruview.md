# ruvnet/RuView — WiFi 신호로 벽 너머를 "보는" 센싱 플랫폼

- **GitHub**: [ruvnet/RuView](https://github.com/ruvnet/RuView)
- **태그**: `#WiFi센싱` `#IoT` `#EdgeAI` `#Rust` `#ESP32`

## 상세 요약

일반 WiFi 라우터의 전파를 활용해 **벽 너머의 사람 감지, 호흡/심박 측정, 움직임 추적**을 수행하는 오픈소스 WiFi 센싱 플랫폼. 카메라나 웨어러블 없이, 물리학(Channel State Information)만으로 공간 인텔리전스를 구현한다.

**감지 가능한 것들:**
- **존재/점유**: 벽 너머 사람 감지, 인원 수 카운팅, 출입 추적
- **생체 신호**: 호흡률(6-30 BPM), 심박(40-120 BPM), 비접촉식
- **활동 인식**: 걷기, 앉기, 제스처, 낙상 감지
- **환경 매핑**: RF 핑거프린팅으로 방 식별, 가구 이동 감지
- **수면 품질**: 수면 단계 분류, 수면 무호흡 스크리닝

**기술 스택:**
- ESP32-S3 메시 ($9/노드) + Cognitum Seed (총 BOM $140)
- Rust 1.85+로 구현
- 스파이킹 신경망(SNN)으로 30초 이내 환경 학습
- 6개 WiFi 채널 멀티주파수 스캐닝 (이웃 라우터까지 활용!)
- Ed25519 위트니스 체인으로 암호화 증명
- 포즈 추정: WiFlow 아키텍처, 17 COCO 키포인트
- Docker로 시뮬레이션 데이터 데모 가능

Carnegie Mellon의 "DensePose From WiFi" 연구에서 영감받은 프로젝트.

## Soo에게 의미 있는 이유

카메라 없는 실내 센싱은 프라이버시 민감 환경(의료, 요양, 호텔)에서 AI/IoT 솔루션으로 큰 가능성이 있다. $140 BOM으로 구현 가능한 저비용 솔루션은 PoC 프로젝트로도 적합.
