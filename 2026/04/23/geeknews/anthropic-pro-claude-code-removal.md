## 상세 요약

Anthropic이 월 $20짜리 **Claude Pro 플랜 신규 가입자에게서 Claude Code 접근을 제외**한 것으로 보인다는 보도가 나왔다. 기존에는 Pro/Max 플랜에서 Claude Code 사용이 가능했으나, 신규 가입자에게는 이 기능이 빠져있다는 것.

이는 코딩 에이전트의 **비용 구조 문제**를 반영한다:
- Claude Code는 대량의 토큰을 소비하는 서비스
- $20/월 정액제로는 수지가 맞지 않을 가능성
- Max 플랜($100+/월)이나 API 사용으로 유도하려는 전략으로 해석

GeekNews에서는 14개의 활발한 댓글이 달리며, "합리적 사업 결정" vs "기존 사용자 기대를 저버리는 것"에 대한 토론이 이어졌다. 코딩 에이전트 시장의 가격 모델이 아직 확립되지 않았음을 보여주는 사례.

## Soo에게 의미 있는 이유

AI 코딩 에이전트의 비용 모델이 안정되지 않은 상태에서, API 키 기반 사용이 가장 예측 가능한 경로라는 것을 재확인. OpenClaw의 멀티 프로바이더 전략(Anthropic + OpenAI + Qwen 등)이 리스크 분산 측면에서 올바른 접근이다.
