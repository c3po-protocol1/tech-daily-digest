# claude-code-best — 실행 가능한 Claude Code 원본 복원

- **GitHub:** https://github.com/claude-code-best/claude-code
- **스타:** 10,401 ⭐
- **언어:** TypeScript
- **태그:** #claude-code #runnable #typescript #fork

## 상세 요약

유출된 Claude Code 소스를 실제로 빌드하고 실행할 수 있도록 정리한 프로젝트다. "원즙원맛(原汁原昧)" — 원래의 맛 그대로를 표방하며, TypeScript 타입 전체 수정, 기업급 신뢰성, 안전 무독성, lock 파일 보존을 특징으로 한다.

`bun i`로 설치하고 `bun run dev`로 바로 실행 가능. 유출 코드의 다양한 포크 중 가장 "원본에 충실한" 실행 가능 버전을 지향한다.

## Soo에게 의미 있는 이유

Claude Code의 내부를 직접 실행하며 동작을 관찰할 수 있어, 에이전트 아키텍처 학습과 실험에 유용하다. 다만 법적/윤리적 관점에서 유출 코드 기반 프로젝트의 사용은 주의가 필요하다.
