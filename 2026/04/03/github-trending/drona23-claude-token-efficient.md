## 상세 요약

CLAUDE.md 파일 하나를 프로젝트에 드롭하기만 하면 Claude의 응답이 간결해지고 무거운 워크플로우에서 출력 장황함이 줄어든다. 코드 변경 없이 드롭인으로 적용 가능.
