## 상세 요약

코딩 에이전트에 대한 연구 자료를 체계적으로 정리한 큐레이션 저장소다. Claude Code 유출 사건 이후 에이전트 아키텍처에 대한 관심이 폭증하면서, 코딩 에이전트의 동작 원리, 설계 패턴, 벤치마크, 연구 논문 등을 학습할 수 있는 리소스가 한곳에 모여 있다.

약 1만 스타를 빠르게 달성한 것은 "에이전트를 사용만 하는 게 아니라 이해하고 싶다"는 개발자들의 수요를 반영한다. 에이전트 하네스 엔지니어링, 도구 호출, 컨텍스트 관리, 멀티에이전트 조정 등 다양한 주제를 다룬다.

## Soo에게 의미 있는 이유

AInD 컨설팅의 교육 커리큘럼이나 내부 학습 자료를 구성할 때 출발점으로 활용할 수 있는 큐레이션 리소스다.
