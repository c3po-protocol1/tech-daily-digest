# open-multi-agent — 경량 멀티에이전트 프레임워크

- **GitHub:** https://github.com/JackChen-me/open-multi-agent
- **스타:** 2,725 ⭐
- **언어:** TypeScript
- **태그:** #multi-agent #framework #task-decomposition #parallel

## 상세 요약

목표를 설명하면 프레임워크가 자동으로 태스크를 분해하고 병렬 실행하는 경량 멀티에이전트 프레임워크다. Claude, GPT, 로컬 모델과 호환.

기존의 무거운 멀티에이전트 프레임워크(AutoGen, CrewAI 등)와 달리 경량화에 초점을 맞추었으며, TypeScript로 작성되어 웹 개발자에게 친숙하다. 태스크 자동 분해와 병렬 실행이라는 핵심 기능에 집중한 미니멀한 설계.

## Soo에게 의미 있는 이유

멀티에이전트 오케스트레이션의 다양한 접근법 중 경량 TypeScript 구현의 사례. 복잡한 프레임워크 없이도 멀티에이전트를 구현할 수 있다는 가능성을 보여준다.
