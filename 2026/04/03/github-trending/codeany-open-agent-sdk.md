# open-agent-sdk-typescript — CLI 의존성 없는 에이전트 SDK

- **GitHub:** https://github.com/codeany-ai/open-agent-sdk-typescript
- **스타:** 2,112 ⭐
- **언어:** TypeScript
- **태그:** #agent-sdk #typescript #open-source #alternative

## 상세 요약

claude-agent-sdk의 대안으로, CLI 의존성 없이 완전 오픈소스로 제공되는 에이전트 SDK다. TypeScript로 작성되어 웹 앱이나 서버 사이드에서 에이전트를 프로그래밍 방식으로 통합할 수 있다.

Claude Code의 SDK가 CLI에 묶여 있다는 제약을 해결하며, 에이전트 기능을 라이브러리로 임포트하여 자유롭게 활용할 수 있게 한다. 벤더 독립적이고 커뮤니티 주도의 에이전트 SDK 생태계가 형성되고 있음을 보여준다.

## Soo에게 의미 있는 이유

에이전트를 CLI 도구가 아닌 프로그래밍 라이브러리로 통합하고자 할 때의 선택지다. AInD 컨설팅에서 클라이언트의 기존 시스템에 에이전트를 임베드하는 시나리오에서 유용할 수 있다.
