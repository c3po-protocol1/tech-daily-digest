## 상세 요약

AI 에이전트 시스템의 소스코드를 심층 분석한 연구 보고서 모음이다. Claude Code 유출 이후 에이전트 아키텍처에 대한 이해 욕구가 폭증한 시점에서, 다양한 에이전트 시스템의 내부 구조를 체계적으로 분석한다.

Python 기반 분석 도구와 함께 각 에이전트 시스템의 아키텍처 결정, 도구 호출 패턴, 컨텍스트 관리 전략, 오류 처리 메커니즘 등을 비교 분석한다. 중국어 기반 커뮤니티에서 시작되었으나 영어 문서화도 진행 중.
