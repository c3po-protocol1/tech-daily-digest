## 상세 요약

Anysphere가 Cursor 3를 발표했다. "에이전트와 함께 소프트웨어를 만드는 통합 워크스페이스"로, 기존 VS Code 포크에서 한 단계 더 나아가 에이전트를 중심으로 처음부터 새로 설계한 인터페이스를 공개했다.

**주요 신기능:**

1. **모든 에이전트를 한 곳에서:** 새 인터페이스는 본질적으로 멀티 워크스페이스를 지원하며, 사람과 에이전트가 여러 저장소에 걸쳐 작업할 수 있다. 모바일, 웹, 데스크톱, Slack, GitHub, Linear에서 시작한 로컬/클라우드 에이전트 모두 사이드바에 표시된다.

2. **다수 에이전트 병렬 실행:** 여러 에이전트를 동시에 돌리며 작업을 분산할 수 있다. 클라우드 에이전트는 작업 결과의 데모와 스크린샷을 생성하여 검증을 용이하게 한다.

3. **로컬-클라우드 핸드오프:** 클라우드에서 로컬로 이동해 직접 편집 및 테스트하거나, 반대로 로컬에서 클라우드로 이동해 오프라인 중에도 작업을 계속할 수 있다. Composer 2(자체 프론티어 코딩 모델)로 빠른 반복 가능.

4. **커밋에서 머지 PR까지:** 새로운 diff 뷰로 변경 사항을 더 빠르게 편집 및 리뷰. 스테이지, 커밋, PR 관리까지 일관된 흐름.

5. **통합 브라우저:** 에이전트가 빌트인 브라우저로 로컬 웹사이트를 열고 탐색 가능.

6. **Cursor 마켓플레이스 플러그인:** MCP, 스킬, 서브에이전트 등을 확장하는 수백 개의 플러그인. 팀 전용 프라이빗 마켓플레이스도 설정 가능.

## Soo에게 의미 있는 이유

Cursor 3는 "소프트웨어 개발의 세 번째 시대"(에이전트 플릿이 자율적으로 개선사항을 배포하는 시대)를 향한 구체적 행보다. 에이전트 기반 개발이 IDE 수준에서 본격 통합되는 모습은 AInD 컨설팅의 핵심 주제이며, 클라이언트에게 "AI 코딩 도구의 현재와 미래"를 설명할 때 가장 좋은 사례가 된다.
