## 요약

Claude Code를 터미널, VSCode 확장, Discord를 통해 무료로 사용할 수 있게 해주는 프로젝트. OpenClaw과 유사한 방식으로 Claude Code 인터페이스를 제공한다.

### 핵심 기능
- **터미널 CLI**: Claude Code와 동일한 터미널 인터페이스
- **VSCode 확장**: IDE 내에서 직접 사용
- **Discord 봇**: Discord 채널을 통한 Claude Code 접근
- 다양한 프로바이더(OpenRouter, Together AI 등) 지원
- 자체 API 키로 운영 가능

### 프로젝트 구조
```
api/        # API 서버
cli/        # 터미널 클라이언트
config/     # 설정
messaging/  # Discord 등 메시징 통합
providers/  # LLM 프로바이더 추상화
tests/      # 테스트
```

### 주의사항
Claude Code의 공식 프로젝트가 아닌 커뮤니티 프로젝트다. Anthropic의 이용 약관 준수 여부는 사용자 책임.

## Soo에게 의미 있는 이유

5.5K 스타를 받을 만큼 Claude Code에 대한 수요가 크다는 것을 보여준다. AInD 컨설팅에서 코딩 에이전트 도입 비용이 진입 장벽인 고객에게 대안적 접근법을 제시할 때 참고할 수 있다.
