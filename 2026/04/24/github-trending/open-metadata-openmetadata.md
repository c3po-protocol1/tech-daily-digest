# open-metadata/OpenMetadata — 통합 메타데이터 플랫폼

> **GitHub:** [open-metadata/OpenMetadata](https://github.com/open-metadata/OpenMetadata)
> **태그:** `#Data` `#Metadata` `#DataGovernance` `#OpenSource`

## 요약

데이터 발견, 데이터 관찰성, 데이터 거버넌스를 위한 통합 메타데이터 플랫폼. 중앙 메타데이터 저장소, 심층 컬럼 레벨 리니지, 팀 협업을 제공한다.

### 핵심 기능
- **데이터 발견**: 조직 내 모든 데이터 자산을 검색 가능한 카탈로그로 관리
- **컬럼 레벨 리니지**: 테이블이 아닌 컬럼 단위로 데이터 흐름 추적
- **데이터 품질**: 자동화된 데이터 품질 검사 및 프로파일링
- **데이터 거버넌스**: 정책 기반 데이터 접근 제어
- **팀 협업**: 데이터에 대한 토론, 태깅, 소유자 관리

### 지원 커넥터
BigQuery, Snowflake, Redshift, Databricks, PostgreSQL, MySQL, Kafka, Airflow 등 주요 데이터 플랫폼 커넥터 제공.

## Soo에게 의미 있는 이유

AI 에이전트가 데이터를 활용하려면 메타데이터 관리가 필수다. OpenMetadata는 "어떤 데이터가 어디에 있고, 어떻게 흘러가는가"를 관리하여, RAG나 에이전트 기반 데이터 분석의 기반 인프라가 된다.
