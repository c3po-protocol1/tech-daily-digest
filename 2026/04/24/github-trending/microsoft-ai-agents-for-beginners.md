## 요약

Microsoft가 공개한 AI 에이전트 구축을 위한 12개 레슨 커리큘럼. "AI Agents for Beginners"라는 제목으로, AI 에이전트의 기초부터 실전 배포까지 다룬다.

### 커리큘럼 구성
1. AI 에이전트 소개
2. 에이전트 프레임워크 이해
3. 도구 사용 에이전트
4. 에이전트 디자인 패턴
5. Agentic RAG
6. 신뢰할 수 있는 에이전트 구축
7. 계획 수립 패턴
8. 멀티 에이전트 시스템
9. 메타코그니션
10. 프로덕션 배포
11. 모니터링 및 관찰성
12. 고급 주제

### 특징
- 각 레슨에 실습 코드, 노트북, 퀴즈 포함
- Azure OpenAI, Semantic Kernel, AutoGen 등 Microsoft 에코시스템 활용
- GitHub Codespaces에서 바로 실행 가능

## Soo에게 의미 있는 이유

AInD 컨설팅 TF의 교육 자료로 바로 활용 가능하다. 특히 "에이전트 디자인 패턴", "멀티 에이전트 시스템", "프로덕션 배포" 레슨은 컨설팅 방법론의 기초가 된다. 2일 연속 트렌딩으로 시장의 학습 수요를 확인할 수 있다.
