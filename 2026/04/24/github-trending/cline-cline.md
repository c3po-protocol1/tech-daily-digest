# cline/cline — IDE 내 자율 코딩 에이전트

> **GitHub:** [cline/cline](https://github.com/cline/cline)
> **태그:** `#AI` `#CodingAgent` `#VSCode` `#IDE`

## 요약

VSCode 내에서 동작하는 자율 코딩 에이전트. 파일 생성/편집, 명령 실행, 브라우저 사용 등을 수행하며, 매 단계마다 사용자 허가를 받는다.

### 핵심 특징
- **IDE 통합**: VSCode 확장으로 설치
- **자율 작업**: 파일 생성/편집, 터미널 명령 실행, 브라우저 자동화
- **안전 우선**: 모든 작업에 사용자 승인 필요 (human-in-the-loop)
- **다중 프로바이더**: OpenAI, Anthropic, Google, AWS Bedrock 등 지원
- **MCP 통합**: Model Context Protocol로 외부 도구 연결

### 왜 트렌딩?
코딩 에이전트 시장이 급성장하면서 Cline도 재조명. Claude Code(터미널), Cursor(에디터), Cline(VSCode 확장)이 각각의 인터페이스에서 경쟁 중.

## Soo에게 의미 있는 이유

코딩 에이전트 시장의 경쟁 구도를 파악하는 데 유용하다. 터미널(Claude Code) vs 에디터(Cursor) vs VSCode 확장(Cline)이라는 인터페이스 전쟁이 벌어지고 있으며, AInD 컨설팅에서 고객의 워크플로우에 맞는 도구를 추천하는 데 필요한 배경 지식이다.
