## 요약

AI 기반 코드 리뷰 에이전트. PR을 자동으로 리뷰하고 피드백을 제공한다.

### 핵심 기능
- GitHub PR에 대한 자동 코드 리뷰
- 보안 취약점, 코드 품질, 성능 이슈 자동 탐지
- 컨텍스트 인식: 프로젝트의 코딩 컨벤션과 아키텍처 패턴 이해
- 인라인 코멘트와 전체 리뷰 요약 제공

### 포지셔닝
기존 정적 분석 도구(SonarQube, CodeClimate)를 넘어, LLM 기반의 의미론적 코드 리뷰를 제공한다. 단순 린팅이 아닌, "이 코드가 의도대로 동작하는가?"에 대한 리뷰.

## Soo에게 의미 있는 이유

코드 리뷰 자동화는 AInD의 가장 즉각적인 ROI를 보여줄 수 있는 영역이다. 에이전트가 코드를 작성하는 것에서 한 발 더 나아가, 다른 에이전트의 코드를 리뷰하는 "에이전트 간 피어 리뷰" 패턴이 형성되고 있다.
