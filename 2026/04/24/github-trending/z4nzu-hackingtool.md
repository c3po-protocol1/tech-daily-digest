# Z4nzu/hackingtool — 올인원 해킹 도구 모음

> **GitHub:** [Z4nzu/hackingtool](https://github.com/Z4nzu/hackingtool)
> **태그:** `#Security` `#Pentesting` `#Tools` `#Python`

## 요약

다양한 보안 테스트 도구를 하나의 인터페이스로 통합한 Python 기반 프로젝트. 펜테스팅, 정보 수집, 네트워크 분석 등의 도구를 메뉴 방식으로 제공한다.

### 포함 도구 카테고리
- 익명성 도구 (Tor, ProxyChains)
- 정보 수집 (Nmap, Shodan, theHarvester)
- 워드리스트 생성
- 무선 네트워크 공격
- SQL 인젝션 도구
- 피싱 공격 도구
- 웹 해킹
- 포스트 익스플로잇
- 포렌식 도구

### 주의사항
교육 및 합법적 보안 테스트 목적으로만 사용해야 한다. 비인가 시스템에 대한 사용은 불법이다.

### 왜 트렌딩?
Bitwarden 공급망 공격 뉴스와 맞물려 보안 도구에 대한 관심이 높아진 것으로 보인다.

## Soo에게 의미 있는 이유

보안 인식 향상 차원에서 참고. AInD 환경의 보안 테스트 시 알아두면 좋은 도구 모음이다.
