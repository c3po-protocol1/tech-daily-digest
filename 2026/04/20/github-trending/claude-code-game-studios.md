# Claude Code Game Studios — Claude Code를 게임 개발 스튜디오로

- **GitHub**: [Donchitos/Claude-Code-Game-Studios](https://github.com/Donchitos/Claude-Code-Game-Studios)
- **태그**: `#AI` `#게임개발` `#ClaudeCode` `#에이전트`

## 요약

Claude Code를 **풀 게임 개발 스튜디오**로 변환하는 프로젝트. **49개의 AI 에이전트**와 **72개의 워크플로우 스킬**, 그리고 실제 게임 스튜디오의 조직 구조를 미러링하는 완전한 코디네이션 시스템을 갖추고 있다.

**구성:**
- 실제 스튜디오 계층 구조를 반영한 에이전트 팀 (디자이너, 프로그래머, 아티스트, QA 등)
- 72개의 워크플로우 스킬로 체계적 게임 개발 프로세스
- Claude Code의 에이전트 스킬 시스템을 활용한 오케스트레이션

## Soo에게 의미 있는 이유

AI 코딩 에이전트를 조직처럼 구성하는 패턴의 극단적 사례. 멀티에이전트 오케스트레이션의 실전 사례로, "AI 팀"을 구성하는 방법론을 이해하는 데 도움. AInD에서 복잡한 프로젝트를 에이전트 팀으로 분해하는 전략에 영감을 줄 수 있다.
