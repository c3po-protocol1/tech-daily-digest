# Paperless-ngx — AI 기반 문서 관리 시스템

- **GitHub**: [paperless-ngx/paperless-ngx](https://github.com/paperless-ngx/paperless-ngx)
- **태그**: `#문서관리` `#OCR` `#셀프호스팅` `#오픈소스`

## 요약

커뮤니티 주도의 강력한 문서 관리 시스템. 물리 문서를 스캔하여 검색 가능한 디지털 아카이브로 변환한다. OCR, 자동 태깅, 매칭 알고리즘을 통해 문서를 지능적으로 분류하고 관리한다.

**주요 기능:**
- OCR을 통한 자동 텍스트 추출
- 머신러닝 기반 자동 분류 및 태깅
- 다국어 지원
- Docker 기반 셀프호스팅
- REST API 제공
- 모바일 친화적 웹 인터페이스

## Soo에게 의미 있는 이유
AI 기반 문서 처리의 실전 사례. 기업 내부 문서 디지털화에 관심 있는 AInD 고객사에 추천 가능한 오픈소스 솔루션.
