## 요약

OpenAI가 공식 제공하는 경량 멀티에이전트 워크플로우 프레임워크. 에이전트 정의, 핸드오프, 가드레일, 트레이싱 등 프로덕션 에이전트 시스템 구축에 필요한 핵심 기능을 제공한다. 23.1k 스타로 높은 인기.

**주요 기능:**
- Agent 클래스 기반 선언적 에이전트 정의
- 에이전트 간 핸드오프 (Handoff) 메커니즘
- 입출력 가드레일 (Guardrails)
- MCP(Model Context Protocol) 서버 통합
- OpenTelemetry 기반 트레이싱
