## 상세 요약

Google Workspace의 Google Vids에 새로 추가된 Chrome 확장 프로그램으로, 별도 앱 없이 브라우저 어느 탭에서나 클릭 한 번으로 화면 녹화를 시작할 수 있다.

### 핵심 기능
- Chrome 확장 툴바에서 원클릭 녹화 시작
- **최대 30분** 녹화 가능
- 녹화 후 미리보기 → Google Vids에서 편집·공유
- 워크플로 중단 없이 모든 탭에서 사용 가능

### 대상
- Google Workspace 고객 전체 + 개인 Google 계정 사용자 무료
- 관리자가 Admin console에서 강제 설치 및 자동 고정 가능

### 활용 시나리오
- 빠른 워크스루 캡처
- 버그 리포트 녹화
- 프레젠테이션 녹화

### 배포 상태
Chrome Web Store에서 즉시 사용 가능 (Rapid Release / Scheduled Release 모두).

## Soo에게 의미 있는 이유
간단한 도구이지만, 팀 협업에서 비동기 커뮤니케이션(영상 메시지, 버그 재현 녹화)에 유용. 별도 도구 설치 없이 브라우저만으로 해결.
