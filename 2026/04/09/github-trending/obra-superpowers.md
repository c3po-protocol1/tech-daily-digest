# obra/superpowers — 코딩 에이전트를 위한 완전한 소프트웨어 개발 워크플로우

- **GitHub:** [obra/superpowers](https://github.com/obra/superpowers)
- **태그:** `#코딩에이전트` `#워크플로우` `#ClaudeCode` `#Codex` `#TDD`

## 상세 요약

코딩 에이전트(Claude Code, Codex, Cursor 등)에 "슈퍼파워"를 부여하는 워크플로우 시스템. Jesse Vincent(obra) 개발.

### 핵심 작동 방식
1. **스펙 먼저:** 에이전트가 코드부터 쓰지 않고, 무엇을 만들려는지 대화로 끌어냄
2. **소화 가능한 단위로 분할:** 스펙을 읽을 수 있는 크기로 보여줌
3. **구현 계획:** "열정적이지만 판단력이 부족한 주니어 엔지니어"도 따를 수 있는 명확한 플랜
4. **서브에이전트 주도 개발:** "go" 명령 후 서브에이전트가 태스크를 처리하고, 상위 에이전트가 검사·리뷰
5. **자동 트리거:** 스킬이 자동 활성화되므로 특별한 조작 불필요

### 강조하는 원칙
- 진짜 Red/Green TDD
- YAGNI (You Aren't Gonna Need It)
- DRY

### 지원 플랫폼
Claude Code (공식 마켓플레이스), Cursor, Codex, OpenCode — 각각 설치 방법 제공.

### 특징
Claude가 2시간 이상 자율적으로 계획에서 벗어나지 않고 작업하는 것이 일반적이라고 주장.

## Soo에게 의미 있는 이유
에이전트가 "코드를 쓰는" 것에서 "소프트웨어를 만드는" 것으로 진화하는 패턴. agent-skills와 함께 AInD 표준 워크플로우 구축의 핵심 참고자료.
