# mem0ai/mem0 — AI를 위한 지능형 메모리 레이어

- **GitHub:** [mem0ai/mem0](https://github.com/mem0ai/mem0)
- **태그:** `#AI메모리` `#개인화` `#RAG` `#YCombinator` `#에이전트`

## 상세 요약

AI 어시스턴트와 에이전트에 지능형 메모리 레이어를 추가하는 프레임워크. Y Combinator S24 배치. "mem-zero"로 발음.

### 연구 성과
- **OpenAI Memory 대비 +26% 정확도** (LOCOMO 벤치마크)
- **91% 빠른 응답** (전체 컨텍스트 대비)
- **90% 토큰 절감** (전체 컨텍스트 대비)

### 핵심 아이디어
사용자 선호도를 기억하고, 개인 요구에 적응하며, 시간이 지남에 따라 지속적으로 학습. 고객 지원 챗봇, AI 어시스턴트, 자율 시스템에 적합.

### v1.0.0 릴리스
API 현대화, 벡터 스토어 지원 개선, GCP 통합 강화.

## Soo에게 의미 있는 이유
우리 에이전트 시스템의 MEMORY.md/memory/ 구조와 개념적으로 동일한 문제를 프레임워크 수준에서 해결. 메모리 아키텍처 설계 시 참고 가능.
