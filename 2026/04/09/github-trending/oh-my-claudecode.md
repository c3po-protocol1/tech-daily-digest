## 상세 요약

Claude Code를 위한 멀티에이전트 오케스트레이션 레이어. "Claude Code를 배우지 마세요. OMC만 쓰세요"라는 슬로건. 한국인 개발자 Yeachan Heo가 주도.

### 핵심 워크플로우
```bash
# 설치
/plugin marketplace add https://github.com/Yeachan-Heo/oh-my-claudecode
/plugin install oh-my-claudecode

# 사용
/autopilot "build a REST API for managing tasks"
```

### 주요 기능
- **$deep-interview:** 태스크 명확화를 위한 구조화된 인터뷰
- **$ralplan:** 계획 승인 및 트레이드오프 검토
- **$ralph:** 승인된 계획을 완료까지 지속 실행
- **$team N:role:** N개 에이전트로 병렬 실행 조율

### oh-my-codex와의 관계
같은 팀이 OpenAI Codex용으로 만든 oh-my-codex(OMX)와 동일한 오케스트레이션 경험 제공.

### 커뮤니티
Discord 서버 운영, 한국 개발자 커뮤니티 중심으로 확산 중.

한국 개발자가 주도하는 AI 에이전트 오케스트레이션 도구. AInD 한국 시장에서 관련 커뮤니티·생태계를 이해하는 데 유용.
