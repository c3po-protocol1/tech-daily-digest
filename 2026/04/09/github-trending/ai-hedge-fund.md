## 상세 요약

19개의 AI 에이전트가 협업하여 트레이딩 의사결정을 내리는 AI 헤지펀드 개념 증명(PoC). **교육·연구 목적만을 위한 프로젝트** — 실제 거래는 수행하지 않음.

### 에이전트 구성 (19개)
**투자 전설급 에이전트 13개:**
- Aswath Damodaran (밸류에이션), Ben Graham (가치투자), Bill Ackman (행동주의), Cathie Wood (성장투자), Charlie Munger (우량주), Michael Burry (역발상), Mohnish Pabrai (저위험 더블), Nassim Taleb (테일 리스크), Peter Lynch (텐배거), Phil Fisher (스컬버팅), Rakesh Jhunjhunwala (인도 대불), Stanley Druckenmiller (매크로), Warren Buffett (오마하의 현인)

**기능 에이전트 6개:**
- Valuation Agent (내재가치 계산), Sentiment Agent (시장 심리), Fundamentals Agent (기본적 분석), Technicals Agent (기술적 분석), Risk Manager (리스크 메트릭), Portfolio Manager (최종 의사결정)

### 작동 방식
각 투자 전설 에이전트가 자신의 투자 철학에 따라 시그널 생성 → 기능 에이전트가 종합 분석 → Portfolio Manager가 최종 주문 결정.

## Soo에게 의미 있는 이유
멀티에이전트 오케스트레이션의 도메인 특화 사례. 에이전트에 "페르소나"를 부여해 다양한 관점을 종합하는 패턴은 AInD 컨설팅의 의사결정 지원 시스템에 직접 적용 가능.
