# alibaba/page-agent — 웹페이지 내에서 자연어로 UI를 제어하는 에이전트

- **GitHub:** [alibaba/page-agent](https://github.com/alibaba/page-agent)
- **태그:** `#GUI에이전트` `#웹자동화` `#알리바바` `#자연어UI` `#MCP`

## 상세 요약

Alibaba가 공개한 "웹페이지 안에 사는 GUI 에이전트". 브라우저 확장이나 Python, 헤드리스 브라우저 없이 **순수 in-page JavaScript**로 동작.

### 핵심 차별점
- **스크린샷 불필요:** 텍스트 기반 DOM 조작 → 멀티모달 LLM이나 특별 권한 불필요
- **한 줄 통합:** `<script>` 태그 하나로 모든 웹페이지에 추가 가능
- **자체 LLM 연결:** Bring Your Own LLM

### 활용 사례
- **SaaS AI 코파일럿:** 몇 줄 코드로 프로덕트에 AI 코파일럿 탑재. 백엔드 재작성 불필요
- **스마트 폼 입력:** 20클릭 워크플로를 한 문장으로. ERP, CRM, 어드민 시스템에 적합
- **접근성:** 자연어로 모든 웹앱 접근 가능. 음성 명령, 스크린 리더
- **멀티페이지 에이전트:** Chrome 확장으로 브라우저 탭 간 에이전트 확장
- **MCP 서버:** 외부 에이전트 클라이언트에서 브라우저 제어

### 설치
```bash
npm install page-agent
```

## Soo에게 의미 있는 이유
GUI 에이전트를 가장 가벼운 방식(순수 JS, 스크린샷 없음)으로 구현한 사례. 기존 웹앱에 AI 자동화를 추가하는 AInD 컨설팅 시나리오에 직접 활용 가능.
