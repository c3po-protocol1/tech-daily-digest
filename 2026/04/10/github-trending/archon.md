## 상세 요약

"AI 코딩을 결정적이고 반복 가능하게 만드는" 최초의 오픈소스 하네스 빌더. 에이전트 하네스를 직접 구축하고 커스터마이즈할 수 있는 도구를 제공.

TypeScript 기반이며, 에이전트 코딩의 비결정성 문제를 해결하는 것이 핵심 목표. "하네스 엔지니어링" 개념의 구체적 구현체.

## Soo에게 의미 있는 이유

Harness Engineering 개념의 오픈소스 구현. Prompt → Context → Harness로 이어지는 AI 개발 패러다임 전환의 실체.
