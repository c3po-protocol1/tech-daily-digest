# OpenBMB/VoxCPM — 토크나이저 없는 다국어 TTS

- **출처**: [GitHub](https://github.com/OpenBMB/VoxCPM) ⭐ 7.6k | 오늘 +460
- **태그**: `#tts` `#speech` `#multilingual` `#voice-cloning`

## 상세 요약

VoxCPM2는 토크나이저 없이 다국어 음성을 생성하는 TTS 모델. 다국어 음성 생성, 창의적 음성 디자인, 실물과 같은 음성 클로닝을 지원.

OpenBMB(청화대학 연구그룹)에서 개발. 기존 TTS의 토크나이저 병목을 제거하여 더 자연스러운 음성 생성을 목표로 함.

## Soo에게 의미 있는 이유

음성 AI의 최전선. 토크나이저 프리 접근법은 TTS 아키텍처의 새로운 방향을 제시.
