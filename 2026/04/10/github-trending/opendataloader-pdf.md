## 상세 요약

AI 파이프라인에 최적화된 PDF 파서. PDF의 구조적 데이터(테이블, 헤더, 리스트 등)를 보존하면서 AI가 소화할 수 있는 형식으로 변환. PDF 접근성 자동화도 지원.

Java 기반이며, Python 예제도 제공. RAG 파이프라인에서 PDF 데이터 품질이 결과를 좌우하는 만큼, 고품질 파싱 도구의 수요가 크다.

## Soo에게 의미 있는 이유

RAG 시스템의 데이터 품질은 파싱에서 시작. 기업 문서(PDF)를 AI에 먹이는 파이프라인 구축 시 핵심 도구.
