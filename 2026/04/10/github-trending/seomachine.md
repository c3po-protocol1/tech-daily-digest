# TheCraigHewitt/seomachine — Claude Code 워크스페이스 기반 SEO 콘텐츠 생성

- **출처**: [GitHub](https://github.com/TheCraigHewitt/seomachine) | 오늘 +725
- **태그**: `#seo` `#content` `#claude-code` `#marketing`

## 상세 요약

Claude Code 워크스페이스를 특화하여 장문의 SEO 최적화 블로그 콘텐츠를 생성하는 시스템. 리서치→작성→분석→최적화 파이프라인을 에이전트 워크플로우로 자동화.

"어떤 비즈니스든" 적용 가능하다고 표방. Claude Code의 스킬/워크스페이스 구조를 마케팅 도메인에 적용한 사례.

## Soo에게 의미 있는 이유

에이전트 하네스의 도메인 특화 사례. 코딩이 아닌 마케팅/콘텐츠 영역에서의 에이전트 활용.
