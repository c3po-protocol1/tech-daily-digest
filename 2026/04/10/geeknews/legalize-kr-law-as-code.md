# legalize-kr — 법령을 코드처럼, 판례도 코드처럼

- **출처**: [legalize.kr](https://legalize.kr)
- **GeekNews**: [토론](https://news.hada.io/topic?id=28353) (14 points)
- **태그**: `#legal-tech` `#open-data` `#korea` `#markdown`

## 상세 요약

법제처의 국가법령정보 공동활용 API를 활용하여 대한민국 법령과 판례를 Markdown 파일로 변환하고, 모든 개정과 판결을 Git 커밋으로 관리하는 프로젝트. 법령을 코드처럼 diff하고, 히스토리를 추적하고, PR처럼 개정을 리뷰할 수 있게 만든다.

개발자가 법령 데이터를 RAG 파이프라인이나 AI 분석에 쉽게 활용할 수 있도록 하는 것이 목표.

## Soo에게 의미 있는 이유

한국 법률 데이터를 AI-ready 포맷으로 만드는 프로젝트. LegalTech + AI의 교차점에서 유용한 데이터 소스.
