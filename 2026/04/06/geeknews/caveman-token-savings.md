## 요약

LLM에게 **원시인 말투로 응답하도록 강제**하여 평균 65~75%의 출력 토큰을 절감하는 재미있으면서도 실용적인 스킬이다.

### 핵심 기능

- **3단계 압축 강도**: Lite, Full, Ultra로 조절
- **기술적 정확성 유지**: 짧고 효율적인 답변을 생성하면서도 기술적 내용의 정확성을 보존
- **실제 벤치마크**: React 코드 질문에서 기존 응답 대비 토큰 사용량 65~75% 감소 확인

### 동작 원리

시스템 프롬프트에 "원시인처럼 말하라"는 지시를 추가하면, LLM이 불필요한 수식어, 반복적 설명, 정중한 서두/마무리를 제거하고 핵심만 전달한다. "Me think bug in useEffect. You no pass dependency array. Fix: add [count] to array. Bug go away." 식의 응답이 나온다.

### 재미와 실용의 결합

유머러스한 접근이지만 토큰 비용 절감 효과는 실제적이며, rtk(위 참조)와 함께 사용하면 입출력 양쪽 모두에서 토큰을 최적화할 수 있다.

## Soo에게 의미 있는 이유

프롬프트 엔지니어링의 창의적 사례이자, AI 비용 최적화의 비전통적 접근법이다. 컨설팅에서 "토큰 비용 절감 전략"을 논의할 때 이런 창의적 사례를 포함하면 발표에 활기를 더할 수 있다. 또한 LLM 출력의 장황함이 실제로 비용에 얼마나 영향을 미치는지를 수치로 보여주는 좋은 예다.
