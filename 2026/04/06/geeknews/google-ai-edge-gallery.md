Google이 공개한 온디바이스 생성형 AI 실행 앱으로, iOS/Android 모두 지원하며 완전 오프라인·프라이빗 환경에서 LLM을 구동한다.

### 핵심 기능 상세

- **Gemma 4 공식 지원**: 서버 전송 없이 고급 추론·논리·창작 기능 테스트 가능
- **Agent Skills**: 위키피디아, 인터랙티브 맵, 비주얼 요약 카드로 모델 기능 확장. URL이나 GitHub 커뮤니티에서 모듈형 스킬 로드 가능
- **Thinking Mode**: 모델의 단계별 추론 과정을 실시간 관찰 (Gemma 4 지원)
- **Ask Image**: 멀티모달 — 카메라/갤러리 사진으로 객체 인식, 비주얼 퍼즐, 상세 설명
- **Audio Scribe**: 실시간 음성→텍스트 변환 및 번역
- **Prompt Lab**: 온도, top-k 등 세밀한 파라미터 조절 가능한 프롬프트 테스트 공간
- **Mobile Actions**: FunctionGemma 270m 파인튠 기반의 오프라인 기기 제어
- **모델 관리 & 벤치마크**: 오픈소스 모델 다운로드, 커스텀 모델 로딩, 하드웨어별 성능 측정
- **100% 온디바이스 프라이버시**: 모든 추론이 기기에서 수행

### 기술 스택

LiteRT(경량 런타임), Google AI Edge API, Hugging Face 통합
