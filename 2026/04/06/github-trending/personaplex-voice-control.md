# PersonaPlex: 풀 듀플렉스 음성 AI의 페르소나 제어 (by NVIDIA)

- **출처**: [GitHub - NVIDIA/personaplex](https://github.com/NVIDIA/personaplex)
- **스타**: ⭐ 6,978 | **언어**: Python
- **태그**: `#음성AI` `#NVIDIA` `#풀듀플렉스` `#페르소나`

## 요약

NVIDIA의 **실시간 풀 듀플렉스 음성-음성 대화 모델**. 텍스트 기반 역할 프롬프트와 오디오 기반 음성 컨디셔닝을 통해 **페르소나 제어**를 가능하게 한다.

### 핵심 특징

- **풀 듀플렉스**: 양방향 동시 대화 — AI가 말하는 도중에도 사용자가 끼어들 수 있음
- **페르소나 제어**: 텍스트 역할 프롬프트로 성격 설정 + 오디오 샘플로 음성 스타일 설정
- **Moshi 아키텍처 기반**: 합성 대화와 실제 대화의 조합으로 학습
- **저지연 자연스러운 대화**: 일관된 페르소나를 유지하면서 자연스러운 구어체 대화
- **7B 파라미터**: HuggingFace에서 가중치 공개
- **로컬 실행 가능**: GPU 메모리 부족 시 CPU 오프로드 옵션

### 사용법

서버를 시작하고 브라우저에서 WebUI 접속. 음성 프롬프트와 역할 프롬프트를 설정하면 실시간 대화 가능. 오프라인 평가를 위한 WAV 파일 입출력도 지원.

## Soo에게 의미 있는 이유

음성 AI의 차세대 모습을 보여주는 NVIDIA의 연구. 특히 "페르소나 제어"는 고객 서비스 챗봇, 가상 어시스턴트 등 엔터프라이즈 음성 AI 애플리케이션에서 핵심 요구사항이다. 풀 듀플렉스 대화의 실현은 AI 에이전트의 UX를 한 단계 끌어올린다.
