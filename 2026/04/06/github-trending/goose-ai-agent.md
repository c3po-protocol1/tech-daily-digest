# Goose: 오픈소스 로컬 AI 에이전트 (by Block)

- **출처**: [GitHub - block/goose](https://github.com/block/goose)
- **스타**: ⭐ 37,554 | **언어**: Rust
- **태그**: `#AI에이전트` `#MCP` `#오픈소스` `#Rust`

## 요약

Block(구 Square)이 개발한 **로컬 실행 AI 에이전트**. 코드 제안을 넘어 프로젝트 전체를 처음부터 빌드하고, 코드를 작성·실행하며, 디버깅하고, 외부 API와 상호작용하는 자율 에이전트다.

### 핵심 특징

- **완전한 자율성**: 설치, 실행, 편집, 테스트를 스스로 수행
- **모든 LLM 지원**: 특정 모델에 종속되지 않으며, 멀티모델 구성으로 성능과 비용 최적화
- **MCP 서버 통합**: Model Context Protocol 서버와 원활한 연동
- **데스크톱 앱 + CLI**: 두 가지 인터페이스 제공
- **Rust로 작성**: 고성능과 안정성
- **커스텀 디스트리뷰션**: 미리 구성된 프로바이더, 확장, 브랜딩으로 자체 goose 배포판 빌드 가능

### 사용 시나리오

프로토타이핑, 기존 코드 리팩토링, 복잡한 엔지니어링 파이프라인 관리 등 개발 워크플로 전반에 적용 가능.

## Soo에게 의미 있는 이유

Claude Code, Codex와 경쟁하는 오픈소스 AI 코딩 에이전트로, "벤더 종속 없는 AI 에이전트" 옵션으로 주목할 만하다. 특히 커스텀 디스트리뷰션 기능은 엔터프라이즈 환경에서 자체 AI 에이전트 배포를 고려하는 조직에 매력적이다.
