# Immich: 고성능 셀프호스팅 사진/비디오 관리

- **출처**: [GitHub - immich-app/immich](https://github.com/immich-app/immich)
- **스타**: ⭐ 96,518 | **언어**: TypeScript
- **태그**: `#셀프호스팅` `#사진관리` `#GooglePhotos대안` `#Flutter`

## 요약

Google Photos의 오픈소스 셀프호스팅 대안으로, 96K+ 스타를 보유한 대형 프로젝트. 사진과 비디오를 자체 서버에서 관리할 수 있다.

### 핵심 특징

- **자동 백업**: 모바일 앱에서 자동으로 사진/비디오 백업
- **얼굴 인식 & 분류**: ML 기반의 자동 분류
- **모바일 앱**: Flutter 기반 iOS/Android 앱
- **웹 UI**: SvelteKit 기반의 반응형 웹 인터페이스
- **NestJS 백엔드**: TypeScript 기반 안정적인 서버
- **지속적 업데이트**: 2022년부터 활발한 개발 지속

## Soo에게 의미 있는 이유

96K 스타의 대형 셀프호스팅 프로젝트. 개인 데이터 주권의 중요성이 커지는 시대에, "클라우드 의존도를 줄이면서 동등한 경험을 제공하는" 사례로 참고할 만하다.
