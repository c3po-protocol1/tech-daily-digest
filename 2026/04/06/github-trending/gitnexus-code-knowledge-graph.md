# GitNexus: 제로서버 코드 인텔리전스 엔진

- **출처**: [GitHub - abhigyanpatwari/GitNexus](https://github.com/abhigyanpatwari/GitNexus)
- **스타**: ⭐ 22,858 | **언어**: TypeScript
- **태그**: `#코드분석` `#지식그래프` `#RAG` `#브라우저`

## 요약

GitNexus는 **완전히 브라우저에서 실행되는 클라이언트 사이드 지식 그래프 생성기**다. GitHub 레포나 ZIP 파일을 드롭하면 인터랙티브한 지식 그래프와 내장된 Graph RAG 에이전트를 제공한다.

### 핵심 기능

- **서버 불필요**: 모든 처리가 브라우저에서 이루어져 코드가 외부로 전송되지 않음
- **코드 탐색용 지식 그래프**: 코드베이스의 구조, 의존성, 관계를 시각적 그래프로 표현
- **Graph RAG 에이전트**: 생성된 그래프 위에서 자연어 질문으로 코드베이스 탐색
- **즉시 사용 가능**: GitHub URL이나 ZIP 파일만 제공하면 됨

### 규모

2.6k 포크, 575 커밋으로 활발한 개발 중. Claude 플러그인과 Cursor 통합도 지원.

## Soo에게 의미 있는 이유

코드 이해와 온보딩을 돕는 AI 도구. 새 프로젝트에 투입될 때 코드베이스를 빠르게 파악하거나, 클라이언트의 레거시 코드를 분석할 때 활용 가능하다. 서버 없이 브라우저에서 동작하므로 보안 우려 없이 사용할 수 있다.
