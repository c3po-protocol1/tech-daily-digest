# Google AI Edge Gallery: 온디바이스 AI 갤러리 앱

- **출처**: [GitHub - google-ai-edge/gallery](https://github.com/google-ai-edge/gallery)
- **스타**: ⭐ 17,455 | **언어**: Kotlin
- **태그**: `#온디바이스AI` `#Gemma4` `#모바일` `#Google`

## 요약

🔥 **HN + GeekNews + GitHub 3곳 동시 트렌딩**

Google의 온디바이스 AI 모델 갤러리 앱. 상세 내용은 GeekNews 항목 참조. iOS/Android 앱으로 Gemma 4를 포함한 다양한 오픈소스 LLM을 완전 오프라인으로 실행, Agent Skills, Thinking Mode, 멀티모달, 음성 변환, 벤치마크 등 제공.

## Soo에게 의미 있는 이유

3개 플랫폼 동시 트렌딩은 온디바이스 AI에 대한 업계의 뜨거운 관심을 반영. PoC 데모 도구로 최적.
