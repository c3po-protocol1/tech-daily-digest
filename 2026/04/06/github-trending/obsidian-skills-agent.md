# Obsidian Skills: Obsidian용 AI 에이전트 스킬

- **출처**: [GitHub - kepano/obsidian-skills](https://github.com/kepano/obsidian-skills)
- **스타**: ⭐ 20,071 | **언어**: Markdown
- **태그**: `#Obsidian` `#에이전트스킬` `#마크다운` `#CLI`

## 요약

Obsidian 창업자 Kepano가 만든 **AI 에이전트를 위한 Obsidian 스킬 모음**. Agent Skills 스펙을 따르며 Claude Code, Codex CLI, OpenCode 등 다양한 에이전트와 호환된다.

### 제공 스킬

| 스킬 | 설명 |
|------|------|
| obsidian-markdown | 위키링크, 임베드, 콜아웃, 프로퍼티 등 Obsidian Flavored Markdown 생성/편집 |
| obsidian-bases | Obsidian Bases(.base) 뷰, 필터, 수식, 요약 |
| json-canvas | JSON Canvas(.canvas) 노드, 엣지, 그룹 |
| obsidian-cli | Obsidian CLI를 통한 볼트 상호작용, 플러그인/테마 개발 |
| defuddle | Defuddle CLI로 웹페이지에서 깨끗한 마크다운 추출 (토큰 절약) |

### 설치 방법

마켓플레이스, npx skills, 또는 수동 복사로 설치 가능. 여러 에이전트 플랫폼에 대한 구체적인 설치 가이드 제공.

## Soo에게 의미 있는 이유

Agent Skills 스펙의 실전 적용 사례. 에이전트가 특정 도구와 어떻게 통합되는지의 패턴을 보여주며, 우리 자체 에이전트 스킬(OpenClaw skills)과 비교 분석할 수 있다. 에이전트 생태계의 표준화 흐름을 이해하는 데 중요하다.
