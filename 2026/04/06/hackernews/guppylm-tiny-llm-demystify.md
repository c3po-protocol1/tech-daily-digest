# GuppyLM: 작은 LLM을 직접 만들어 언어모델의 작동 원리를 이해하기

- **출처**: [GitHub - arman-bd/guppylm](https://github.com/arman-bd/guppylm)
- **HN 토론**: [Hacker News](https://news.ycombinator.com/item?id=47655408) (526 points, 53 comments)
- **태그**: `#LLM` `#교육` `#ML` `#오픈소스`

## 요약

GuppyLM은 약 900만(~9M) 파라미터의 초소형 언어 모델로, "Guppy"라는 이름의 물고기 캐릭터처럼 대화하도록 설계되었다. 이 프로젝트의 핵심 목적은 **LLM 학습이 마법이 아님을 증명하는 것**이다.

### 핵심 특징

- **Google Colab에서 5분 만에 훈련 가능**: 박사 학위나 대형 GPU 클러스터 없이, 노트북 하나만 실행하면 동작하는 LLM을 처음부터 직접 만들 수 있다
- **전체 파이프라인을 직접 경험**: 데이터 생성 → 토크나이저 구축 → 모델 아키텍처 설계 → 훈련 루프 → 추론까지 모든 단계를 직접 수행
- **아키텍처 상세**: 6개 레이어, 384 히든 차원, 6 어텐션 헤드, 768 FFN(ReLU), 4,096 BPE 보캡, 최대 128 토큰 시퀀스, LayerNorm, 학습된 포지션 임베딩
- **훈련 데이터**: 60개 주제에 걸친 6만 개의 합성 대화로 학습
- **브라우저에서 실행 가능**할 만큼 작은 모델 크기

### 대화 예시

물고기 캐릭터로서 물, 음식, 빛, 수조 생활에 대해 이야기하며, "삶의 의미가 뭐냐"라는 질문에 "음식. 답은 항상 음식이야"라고 답하는 등 캐릭터 일관성을 유지한다.

### 학습 가치

이 프로젝트를 통해 대형 모델들이 더 이상 블랙박스처럼 느껴지지 않게 되며, 데이터 → 가중치 → 생성된 출력까지 모든 과정을 투명하게 이해할 수 있다.

## Soo에게 의미 있는 이유

AInD 컨설팅 TF를 이끄는 입장에서, 클라이언트에게 LLM의 기본 원리를 설명할 때 이런 교육용 프로젝트가 매우 유용하다. 복잡한 대형 모델의 작동 방식을 미니어처로 체험하게 함으로써, "AI는 마법이 아니라 엔지니어링"이라는 핵심 메시지를 전달할 수 있다. 또한 사내 교육 자료로도 활용 가능하다.
