## 상세 요약

한국 개발자가 만든 올인원 크롬 확장 프로그램 LunaTools가 공개 1주년을 맞았다. 처음 5개 기능으로 시작해 현재 18가지 기능을 갖추고 있다.

**주요 기능 18가지:**
- **마우스 제스처**: 뒤로/앞으로/닫기 3가지 핵심 제스처만 구현 (도넛Q, 웹마2에서 영감)
- **키보드 페이지 이동**: 게시판 등에서 좌우키로 페이지 이동
- **중복 탭 방지**: 같은 URL 탭이 여러 개 열리는 것을 방지
- **탭 URL 기준 정렬**: 많은 탭을 URL 기준으로 체계적으로 정렬
- **창 통합**: 여러 창의 탭을 하나로 통합
- **PiP (Picture in Picture)**: 미지원 사이트에서도 PiP 강제 활성화
- **환율 변환**: 텍스트 블록 지정 후 Alt+Z로 원화 환율 변환 (오페라의 기본 기능을 구현)
- **세션 매니저**: 탭 세션 저장/복원
- **볼륨 증폭**: 시스템 볼륨 이상으로 미디어 볼륨 증폭

기존에 분산된 여러 확장 프로그램의 기능을 하나로 통합한 것이 특징이며, 크롬 탭을 많이 사용하는 사용자에게 특히 유용하다. PiP 기능은 기존 확장 프로그램 개발자의 허락을 받아 참고했다고 밝히고 있다.

## Soo에게 의미 있는 이유

단일 개발자가 사용자 니즈를 하나씩 해결하며 기능을 확장해나간 방식은 AI 시대에도 여전히 유효한 "자기 가려운 곳을 긁는" 인디 개발 전략의 좋은 사례다. 브라우저 확장 프로그램은 AI 에이전트의 브라우저 제어와도 관련이 깊어, MCP 기반 브라우저 자동화의 기초 이해에 도움이 된다.
