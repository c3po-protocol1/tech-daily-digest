## 상세 요약

Paperclip과 Gastown에서 영감을 받아 개발된 오픈소스 AI 에이전트 프레임워크. 핵심 컨셉은 "토큰만 있으면 AI가 스스로 일을 이어가며 제품을 키워나가는 영구기관(perpetual engine)"이다.

**핵심 아이디어:**
- AI를 단발성 호출이 아니라, **지속적으로 작동하는 시스템**으로 바라봄
- 작업을 스스로 생성 → 우선순위 조정 → 결과 평가 → 다음 액션 이어감
- 사람이 매번 개입하지 않아도 "일이 계속 굴러가는 구조"

**차별화 포인트 — 가시성(Visibility):**
- AI가 만든 결과물을 단순 텍스트 로그가 아닌 시각적으로 확인 가능
- UI 시안, 문서, 리서치 결과를 구조적으로 볼 수 있음
- AI가 현재 어떤 방향으로 일하고 있는지, 무엇을 만들고 있는지 한눈에 파악

**스타트업 맥락:**
- 리서치 → 구현 → 테스트 → 개선 사이클 자동화
- 제한된 리소스(토큰) 안에서 최대 성과를 내도록 설계
- "Paperclip Maximizer" 개념을 실용적으로 구현한 프로젝트

한국어 README가 제공되며, 한국 개발자 커뮤니티에서의 관심이 높다.

## Soo에게 의미 있는 이유

자율 에이전트의 "지속적 작동" 패턴은 현재 AI 에이전트 아키텍처의 핵심 트렌드다. OpenClaw의 heartbeat/cron 시스템과도 맥이 통하며, AInD 컨설팅에서 "에이전트 자율성의 적정 수준"을 논의할 때 구체적 사례로 활용 가능하다.
