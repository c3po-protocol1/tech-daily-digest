# Voicebox — 오픈소스 음성 합성 스튜디오

- **출처**: [GitHub](https://github.com/jamiepine/voicebox)
- **태그**: `#TTS` `#voice-cloning` `#open-source` `#local-AI`

## 상세 요약

오픈소스 음성 합성 스튜디오. 음성 복제, 음성 생성, 이펙트 적용, 음성 기반 앱 개발까지 모든 것을 **로컬 머신에서** 실행한다.

**핵심 기능:**
- **Voice Cloning**: 기존 음성을 복제하여 새로운 텍스트를 해당 음성으로 합성
- **Speech Generation**: 텍스트를 음성으로 변환
- **Effects**: 다양한 음성 효과 적용
- **API**: 음성 기반 앱 개발을 위한 프로그래밍 인터페이스
- **완전 로컬 실행**: 클라우드 의존 없이 사용자 머신에서 모든 처리

**앱 구성:**
- 데스크톱 앱 형태 (스크린샷 기반으로 깔끔한 UI)
- 웹사이트 및 문서 제공 (voicebox.sh, docs.voicebox.sh)
- 트러블슈팅 가이드 포함

**프로젝트 상태:**
- 활발한 릴리스 (GitHub Releases)
- 커뮤니티 관심도 높음 (별 수 증가 추세)
- DeepWiki 통합으로 코드베이스 탐색 지원

## Soo에게 의미 있는 이유

로컬 TTS는 AI 에이전트의 음성 출력을 구현할 때 프라이버시와 비용 측면에서 클라우드 API의 대안이 될 수 있다. VoxCPM2와 함께 오늘 트렌딩에 TTS 프로젝트가 2개 올라온 것은 음성 AI에 대한 관심이 급증하고 있음을 보여준다.
