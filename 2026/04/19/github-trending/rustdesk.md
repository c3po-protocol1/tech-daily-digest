# 🔥 RustDesk — Rust로 만든 오픈소스 원격 데스크톱

- **출처**: [GitHub](https://github.com/rustdesk/rustdesk)
- **태그**: `#remote-desktop` `#Rust` `#open-source` `#self-hosted`

## 상세 요약

Rust로 작성된 오픈소스 원격 데스크톱 솔루션. TeamViewer/AnyDesk의 오픈소스 대안으로, 설정 없이 바로 작동하며 데이터에 대한 완전한 통제권을 제공한다.

**핵심 특징:**
- **제로 설정**: 설치 후 바로 작동, 복잡한 설정 불필요
- **데이터 주권**: 자체 서버 구축 가능, 보안 우려 없음
- **크로스플랫폼**: F-Droid, Flathub 등 다양한 배포 채널
- **자체 rendezvous/relay 서버** 구축 가능 또는 공식 서버 사용

**기술 스택:**
- Rust (성능·안정성·보안)
- Flutter (모바일 클라이언트)
- 자체 rendezvous/relay 프로토콜

**GitHub 트렌딩 재등장 이유:**
- 원격 근무·AI 에이전트 시대에 안전한 원격 접근 도구의 수요 증가
- TeamViewer/AnyDesk의 보안 우려 및 가격 정책에 대한 대안 수요

프로젝트는 이미 수년간 개발되어 안정화 단계에 있으며, RustDesk Server Pro라는 엔터프라이즈 버전도 제공한다.

## Soo에게 의미 있는 이유

AI 에이전트가 원격 서버를 제어하거나, 원격 개발 환경에 접근하는 시나리오에서 RustDesk 같은 도구는 인프라 계층의 핵심이 될 수 있다. 자체 서버를 통한 데이터 주권 확보는 기업 보안 정책과도 부합한다.
