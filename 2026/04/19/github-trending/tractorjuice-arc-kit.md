# ArcKit — AI 기반 엔터프라이즈 아키텍처 거버넌스 툴킷

- **출처**: [GitHub](https://github.com/tractorjuice/arc-kit)
- **태그**: `#enterprise-architecture` `#governance` `#AI-agent` `#Claude-Code`

## 상세 요약

엔터프라이즈 아키텍트를 위한 AI 지원 거버넌스 툴킷. 흩어진 아키텍처 문서를 체계적이고 AI가 지원하는 워크플로로 전환한다.

**지원 워크플로 (68개 명령):**
- 🏛️ 아키텍처 원칙 수립·시행
- 👥 이해관계자 드라이버, 목표, 성과 분석
- 🛡️ 리스크 관리 (HM Treasury Orange Book)
- 💼 비즈니스 케이스 정당화 (HM Treasury Green Book SOBC)
- 📋 요구사항 문서 작성
- 🗄️ 데이터 모델링 (ERD, GDPR, 데이터 거버넌스)
- 🔬 기술 리서치 (build vs buy 분석, 웹 검색 지원)
- ☁️ Azure 리서치 (Microsoft Learn MCP 활용)
- 🗺️ Wardley Mapping 전략 계획
- 📊 Mermaid 다이어그램 자동 생성
- 🤝 벤더 RFP 및 선정 프로세스
- ✅ 설계 리뷰 (HLD/DLD)

**통합 플랫폼:**
- **Claude Code** (프리미어): 플러그인 마켓플레이스, 10개 자율 리서치 에이전트, 5개 자동화 훅
- **Gemini CLI**: 제로 설정 확장
- **GitHub Copilot**: VS Code 프롬프트 파일

Claude Code v2.1.112+ 필요 — Opus 4.7의 `xhigh` 에포트 티어, Auto 모드 등 최신 기능 활용.

## Soo에게 의미 있는 이유

"AI 에이전트가 실제 엔터프라이즈 업무에 어떻게 적용되는가"의 가장 구체적인 사례 중 하나다. Claude Code 플러그인 시스템을 활용한 68개 명령의 실무 도구는 AInD 컨설팅에서 "에이전트 스킬 설계 패턴"의 레퍼런스가 될 수 있다.
