# GenericAgent — 자가 진화하는 자율 에이전트 프레임워크

- **출처**: [GitHub](https://github.com/lsdefine/GenericAgent)
- **태그**: `#AI-agent` `#self-evolving` `#automation` `#autonomous`

## 상세 요약

**~3K 라인의 코드**로 구성된 미니멀한 자가 진화 자율 에이전트 프레임워크. 9개의 원자적 도구와 ~100줄의 Agent Loop로 LLM에게 시스템 수준의 컴퓨터 제어권을 부여한다.

**9개 원자적 도구가 제어하는 영역:**
- 브라우저 (로그인 세션 유지하며 실제 브라우저에 주입)
- 터미널
- 파일시스템
- 키보드/마우스 입력
- 화면 비전
- 모바일 디바이스 (ADB)

**자가 진화 메커니즘 (핵심 차별점):**
```
[새 작업] → [자율 탐색] (의존성 설치, 스크립트 작성, 디버그·검증)
→ [실행 경로를 스킬로 결정화] → [메모리 레이어에 기록]
→ [다음 유사 작업에서 직접 호출]
```

**실제 예시:**
| 명령 | 첫 번째 실행 | 이후 |
|---|---|---|
| "위챗 메시지 읽어줘" | 의존성 설치→DB 역공학→읽기 스크립트 작성→스킬 저장 | **한 줄 호출** |
| "주식 모니터링 알림" | mootdx 설치→선별 플로 구축→cron 설정→스킬 저장 | **한 줄 시작** |
| "Gmail로 파일 전송" | OAuth 설정→전송 스크립트 작성→스킬 저장 | **바로 사용** |

**자가 부트스트랩 증명**: Git 설치부터 `git init`, 모든 커밋 메시지까지 GenericAgent가 자율적으로 완료. 저자는 터미널을 한 번도 열지 않았다.

**토큰 효율**: <30K 컨텍스트 윈도우 (다른 에이전트의 200K-1M 대비 극히 적음). 계층적 메모리로 노이즈·환각 감소, 성공률 향상.

## Soo에게 의미 있는 이유

"스킬 결정화(skill crystallization)"는 AI 에이전트의 가장 흥미로운 패턴 중 하나다. 사용할수록 스킬 트리가 성장하여 고유한 에이전트 인스턴스가 되는 개념은 OpenClaw의 스킬 시스템과도 맥이 통한다. 3K 라인의 미니멀한 설계는 에이전트 아키텍처 학습에 최적의 코드베이스다.
