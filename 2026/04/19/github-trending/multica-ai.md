# Multica — 코딩 에이전트를 진짜 팀원으로 만드는 플랫폼

- **출처**: [GitHub](https://github.com/multica-ai/multica)
- **태그**: `#AI-agent` `#team-management` `#managed-agents` `#open-source`

## 상세 요약

"당신의 다음 10명의 채용은 인간이 아닐 것이다"라는 슬로건을 가진 오픈소스 관리형 에이전트 플랫폼. 코딩 에이전트를 실제 팀원처럼 작업을 할당하고, 진행 상황을 추적하며, 스킬을 축적하는 시스템이다.

**핵심 기능:**
- **에이전트 = 팀원**: 이슈를 동료에게 할당하듯 에이전트에게 할당. 프로필 보유, 보드에 표시, 댓글 작성, 이슈 생성, 블로커 보고
- **자율 실행**: set-and-forget. 작업 전체 생명주기 관리(대기열→청구→시작→완료/실패). WebSocket으로 실시간 진행 스트리밍
- **재사용 가능 스킬**: 모든 솔루션이 팀 전체가 재사용 가능한 스킬로 축적. 배포, 마이그레이션, 코드 리뷰 등의 스킬이 시간이 지날수록 강화
- **통합 런타임**: 로컬 데몬과 클라우드 런타임 통합 대시보드. CLI 자동 감지, 실시간 모니터링
- **멀티 워크스페이스**: 팀별 격리된 환경

**지원 에이전트 런타임:**
Claude Code, Codex, **OpenClaw**, OpenCode, Hermes, Gemini, Pi, Cursor Agent

**설치:** `brew install multica-ai/tap/multica`

## Soo에게 의미 있는 이유

OpenClaw를 지원 런타임으로 포함하고 있어 직접적으로 관련된 프로젝트다. "에이전트를 팀원으로 관리한다"는 패러다임은 AInD 컨설팅에서 "AI 팀 구성 전략"의 핵심 레퍼런스가 될 수 있다. 특히 재사용 가능 스킬 축적 메커니즘은 에이전트 시스템의 장기적 가치를 보여준다.
