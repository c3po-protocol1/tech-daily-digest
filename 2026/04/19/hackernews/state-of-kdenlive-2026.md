## 상세 요약

오픈소스 비디오 편집기 Kdenlive의 2026년 현황 보고서가 HN에서 399점을 기록하며 큰 관심을 받았다. 2025년 동안 Kdenlive 팀은 새 기능 추가, 버그 수정, UI 개선, 성능 향상을 균형 있게 진행했으며, 안정성을 기능 확장보다 우선시했다.

**주요 릴리스 하이라이트:**

1. **25.04.0**: SAM2 모델 기반 자동 마스킹 도구(Object Segmentation) 추가 — 배경에서 선택한 객체를 자동 분리. OpenTimelineIO C++ 라이브러리를 활용한 프로젝트 교환 기능 재작성. 오디오 파형 생성 **300% 성능 향상**.

2. **25.08.0**: 안정화에 집중, 300+ 커밋, 15개 이상의 크래시 수정. 오디오 믹서 UI 재설계(HiDPI 지원 개선). 마커/가이드 대폭 개선.

3. **25.12.0**: 첫 실행 화면(Welcome Screen) 추가. 새로운 도킹 시스템(위젯 그룹화, 레이아웃 파일 저장/공유). 프로젝트 모니터 파형 인터페이스 개편.

**향후 로드맵:**
- **26.04**: 모니터 미러링, 애니메이션 전환 미리보기, 재생 속도 일괄 변경
- **중장기**: MLT 프레임워크에 10/12비트 색상 지원, 재생 최적화(디코딩), OpenFX 지원 추가
- **Dopesheet**: 키프레임 시스템 리팩터링, 다중 이펙트 키프레임 관리 전용 타임라인 (NLnet/NGI Zero Commons 그랜트 지원)

웹사이트도 새 CMS로 재구축하여 2002년까지의 역사적 콘텐츠를 복원했다.

## Soo에게 의미 있는 이유

오픈소스 프로젝트의 지속 가능한 성장 전략을 보여주는 좋은 사례다. SAM2(Meta AI의 세그먼테이션 모델) 통합은 오픈소스 데스크톱 앱에 AI 모델을 녹여내는 방법론의 참고 사례가 되며, KDE 릴리스 사이클(연 3회 메이저 + 유지보수)은 체계적인 오픈소스 운영의 벤치마크다.
