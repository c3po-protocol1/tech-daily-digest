## 상세 요약

Bitwarden 호환 비밀번호 관리 서버의 비공식 경량 구현체로, Rust로 작성되었다. 59,000개 이상의 스타로 셀프 호스팅 비밀번호 관리 솔루션 중 가장 인기 있는 프로젝트다.

최근 Bitwarden CLI의 공급망 공격(Socket.dev, JFrog에서 보고)이 발생한 직후 트렌딩에 오른 것으로, Bitwarden 사용자들이 대안을 찾고 있을 가능성이 있다.

**특징**: Docker 기반 간편 배포, Bitwarden 공식 클라이언트와 완벽 호환, 공식 서버 대비 매우 적은 리소스 사용.

## Soo에게 의미 있는 이유

보안 인프라의 셀프 호스팅은 기업 보안 전략의 중요한 옵션이다. 특히 AI 에이전트가 API 키, 비밀 자격 증명을 다루는 상황에서, 안전한 시크릿 관리 인프라의 필요성이 커진다.
