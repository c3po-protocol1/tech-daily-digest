# google/osv-scanner — Google의 오픈소스 취약점 스캐너

- **GitHub**: [google/osv-scanner](https://github.com/google/osv-scanner)
- **스타**: 9,473 ⭐ | **언어**: Go
- **태그**: `#보안` `#취약점` `#OSV` `#Google` `#DevSecOps`

## 상세 요약

Google이 만든 오픈소스 취약점 스캐너로, osv.dev 데이터베이스를 활용하여 프로젝트의 의존성 취약점을 검사한다.

**주요 기능**:
- Go로 작성된 고성능 바이너리
- osv.dev의 포괄적인 취약점 데이터베이스 활용
- 다양한 패키지 매니저 지원 (npm, pip, cargo, go modules 등)
- SBOM(소프트웨어 자재 명세서) 생성 및 분석
- CI/CD 파이프라인 통합 가능
- 보안 감사 및 보고서 생성

9,400개 이상의 스타와 보안 도구로서의 입지가 확립되어 있다. AI 코딩 에이전트가 생성하는 코드의 보안 검증에도 활용 가능하다.

## Soo에게 의미 있는 이유

AI가 생성한 코드의 보안 검증 파이프라인에서 필수적인 도구다. AInD 컨설팅에서 "AI 코딩 도입 시 보안 가이드라인"을 설계할 때 osv-scanner를 CI/CD에 통합하는 것을 권장할 수 있다.
