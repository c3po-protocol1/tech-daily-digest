## 상세 요약

데이터 디스커버리, 데이터 관측 가능성, 데이터 거버넌스를 위한 통합 메타데이터 플랫폼이다. 중앙 메타데이터 저장소, 컬럼 레벨 리니지, 팀 협업 기능을 제공한다.

**주요 기능**:
- 데이터 카탈로그: 조직 내 모든 데이터 자산의 디스커버리
- 데이터 리니지: 컬럼 레벨까지의 데이터 흐름 추적
- 데이터 품질: 자동화된 데이터 품질 검사 및 프로파일링
- 데이터 계약: 팀 간 데이터 인터페이스 관리
- MCP 서버 지원: AI 에이전트와의 통합을 위한 MCP 서버 제공

13,000개 이상의 스타와 MCP 서버 지원은 이 프로젝트가 AI 시대의 데이터 거버넌스에 적극 대응하고 있음을 보여준다.

## Soo에게 의미 있는 이유

AI 프로젝트의 성공은 데이터 품질에 달려 있다. 조직의 데이터 자산을 체계적으로 관리하는 메타데이터 플랫폼은 AInD 도입의 기반 인프라로, 컨설팅에서 "AI 준비도 평가"의 핵심 항목이 될 수 있다.
