## 상세 요약

Zilliz(Milvus 벡터DB 개발사)가 만든 코드 검색 MCP 서버로, 전체 코드베이스를 코딩 에이전트의 컨텍스트로 만들어준다.

**핵심 기능**:
- 코드베이스의 시맨틱 검색 및 인덱싱
- Merkle 트리 기반의 효율적 변경 감지
- Voyage AI 임베딩을 활용한 의미적 코드 검색
- Claude Code, Cursor, Gemini CLI, VS Code 등 다양한 에이전트와 호환
- 벡터 데이터베이스를 활용한 Agentic RAG 구현

**기술 스택**: TypeScript, Node.js, 임베딩 기반 검색, MCP 프로토콜

8,977개의 스타와 20개의 토픽 태그가 이 프로젝트의 성숙도와 범용성을 보여준다.

## Soo에게 의미 있는 이유

대규모 코드베이스에서 AI 코딩 에이전트의 효율을 높이는 핵심 인프라다. 컨텍스트 윈도우의 물리적 한계를 RAG로 극복하는 접근은, 기업 규모의 코드베이스에서 AI 코딩을 도입할 때 필수적이다.
