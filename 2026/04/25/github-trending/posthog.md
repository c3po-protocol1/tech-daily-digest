# PostHog — 올인원 개발자 제품 분석 플랫폼

- **GitHub**: [PostHog/posthog](https://github.com/PostHog/posthog)
- **스타**: 33,079 ⭐ | **언어**: Python
- **태그**: `#분석` `#제품` `#세션리플레이` `#피처플래그` `#오픈소스`

## 상세 요약

PostHog는 성공적인 제품 구축을 위한 올인원 개발자 플랫폼이다. 제품 분석, 웹 분석, 세션 리플레이, 에러 추적, 피처 플래그, 실험(A/B 테스트), 설문조사, 데이터 웨어하우스, CDP, AI 제품 어시스턴트를 하나의 스택에서 제공한다.

33,000개 이상의 스타와 활발한 개발이 이루어지고 있으며, 최근에는 AI 분석 기능이 추가되었다.

**핵심 가치**: 모든 사용 데이터와 고객 데이터를 하나의 스택에서 관리. 여러 SaaS를 조합하는 대신 단일 플랫폼으로 통합.

## Soo에게 의미 있는 이유

AI 제품의 사용자 행동 분석에 PostHog를 활용할 수 있다. 특히 AI 기능의 A/B 테스트, 사용 패턴 분석 등은 AInD 프로젝트에서 제품 의사결정에 필수적이다.
