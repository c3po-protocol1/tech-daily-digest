# 법망 — PostgreSQL 기반 한국 법령 전체를 JSON으로 제공하는 에이전트용 API

- **출처**: [GeekNews (Show GN)](https://news.hada.io/topic?id=28050)
- **태그**: `#한국법령` `#API` `#AI에이전트` `#PostgreSQL` `#pgvector`

## 상세 요약

로스쿨 학생이 개발한 한국 법령 정보 API로, AI 에이전트가 법령 데이터에 효율적으로 접근할 수 있도록 설계되었다. 법제처 공식 API 대비 원하는 정보에 도달하는 시간과 토큰을 1/10 수준으로 줄였다고 한다.

**핵심 기능:**
- XML, HWP, PDF 사전 파싱 — 표 데이터 포함 모든 출력을 JSON으로 변환
- 국가법령정보센터 법령 99.9% 이상 수록, 매주 토요일 최신 동기화
- PostgreSQL + pgvector 기반 주요 조문 20만건 임베딩(768d)으로 시맨틱 검색 지원
- 인증키 불필요한 REST API, rate limit 100회/분
- 4분 내 500 동시접속 20K+ 요청 처리 완료

**4가지 기본 Action:**
`law.search`, `law.get`, `law.diff`, `law.history` — Claude 또는 Codex에서 MCP 설정 없이 프롬프트 한 줄로 바로 사용 가능.

**프라이버시:**
로그인, IP, 쿼리 등 로깅 없음. 익명 엔드포인트 호출 빈도만 집계.

**반응:**
공개 24시간 내 누적 2만 건 이상 API 호출이 발생했다.

## Soo에게 의미 있는 이유

AI 에이전트를 위한 도메인 특화 API의 훌륭한 사례다. 법률 도메인에서 비정형 문서(HWP, PDF)를 JSON으로 변환하고 pgvector로 시맨틱 검색을 지원하는 파이프라인은 AInD 컨설팅에서 다른 도메인(의료, 금융 등)에도 적용 가능한 패턴이다.
