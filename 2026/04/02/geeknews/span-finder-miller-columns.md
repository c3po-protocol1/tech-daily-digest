# SpanFinder — 윈도우 탐색기 대안, 맥 스타일 Miller Columns 파일 탐색기

- **GitHub**: [LumiBearStudio/SpanFinder](https://github.com/LumiBearStudio/SpanFinder)
- **태그**: `#Windows` `#파일관리` `#UX` `#오픈소스` `#개발도구`

## 상세 요약

윈도우 순정 탐색기의 깊은 디렉토리 탐색 불편함을 해결하기 위해 제작된 맥 Finder의 Miller Columns(다단 뷰) 방식 파일 탐색기다.

**주요 기능:**
- 방향키만으로 상위/하위 폴더 즉시 탐색 (키보드 중심 설계)
- 스페이스바 Quick Look으로 파일 즉시 미리보기
- 10,000+ 파일 폴더에서도 비동기 썸네일 로딩으로 UI 멈춤 없음
- 30개 이상 언어 Syntax Highlighting + Hex 뷰어
- Git Status 뱃지가 폴더/파일 아이콘에 네이티브 표시
- 다중 탭(세션 자동 복원), 듀얼 페인(화면 분할), FTP/SFTP 기본 지원
- 가입/인증/사용자 데이터 수집 없음 (크래시 리포트만 옵셔널)
- 오픈소스(GPL v3), MS 스토어 100% 무료

## Soo에게 의미 있는 이유

개발 생산성 도구로서 관심을 가질 만하다. 깊은 프로젝트 디렉토리 탐색, Git 상태 확인, 코드 미리보기가 하나의 파일 탐색기에서 해결된다. macOS 사용자에겐 Finder가 이미 이 방식이지만, 윈도우 환경에서 개발하는 팀원에게 추천할 만하다.
