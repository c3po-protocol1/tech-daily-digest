# 🔥 openai/codex — OpenAI의 로컬 코딩 에이전트 CLI

- **GitHub**: [openai/codex](https://github.com/openai/codex)
- **언어**: Rust
- **태그**: `#AI` `#코딩에이전트` `#CLI` `#OpenAI` `#개발도구`

## 상세 요약

OpenAI의 Codex CLI는 로컬 컴퓨터에서 실행되는 코딩 에이전트다. Claude Code의 직접 경쟁자로, `npm i -g @openai/codex` 또는 `brew install --cask codex`로 설치 가능.

**주요 특징:**
- ChatGPT 플랜(Plus, Pro, Team, Edu, Enterprise)과 직접 연동 — Sign in with ChatGPT
- API 키를 별도 설정하면 API 사용도 가능
- macOS(Apple Silicon + x86_64), Linux(x64 + arm64), Windows(x64 + arm64) 전 플랫폼 지원
- Apache-2.0 라이선스 오픈소스
- 665개 이상의 릴리즈로 활발한 개발

**Claude Code와의 차별점:**
Claude Code가 Anthropic 모델 전용인 반면, Codex는 OpenAI 모델 기반. 하지만 OpenAI가 Claude Code용 Codex 플러그인까지 출시하면서 에이전트 간 협업이 가능해졌다. Codex는 추론에 강하고, Claude Code는 더 깨끗한 코드 생성에 강하다는 평가.

## Soo에게 의미 있는 이유

Claude Code와 함께 AI 코딩의 양대 산맥. Codex 플러그인을 통한 크로스-에이전트 협업은 멀티-모델 오케스트레이션의 실전 사례로, AInD 실무에서 도구 선택과 조합 전략의 참고가 된다.
