# f/prompts.chat — 세계 최대 오픈소스 AI 프롬프트 라이브러리

- **GitHub**: [f/prompts.chat](https://github.com/f/prompts.chat)
- **Stars**: 143K+
- **태그**: `#프롬프트` `#AI` `#ChatGPT` `#Claude` `#오픈소스`

## 상세 요약

원래 "Awesome ChatGPT Prompts"로 시작하여 세계 최대 오픈소스 프롬프트 라이브러리로 성장한 프로젝트다. ChatGPT, Claude, Gemini, Llama, Mistral 등 모든 AI 모델에서 사용 가능.

**주요 기능:**
- Forbes 피처, Harvard/Columbia 인용, 40개 이상 학술 논문 인용
- Hugging Face 가장 많은 좋아요 받은 데이터셋
- GitHub 143K+ 스타, GitHub Staff Pick
- "The Interactive Book of Prompting" — 25+ 챕터의 프롬프트 엔지니어링 가이드
- "Prompting for Kids" — 8-14세 아이들을 위한 게임형 AI 학습 교재

**최근 추가된 기능:**
- Claude Code 플러그인으로 설치 가능
- MCP 서버로도 사용 가능 (AI 도구에서 직접 접근)
- 셀프 호스팅 지원 — 커스텀 브랜딩, 테마, 인증(GitHub/Google/Azure AD)

## Soo에게 의미 있는 이유

AInD 컨설팅에서 프롬프트 엔지니어링 교육 자료로 활용 가능. 특히 MCP 서버로 제공되어 에이전트 워크플로우에서 프롬프트 라이브러리를 직접 조회할 수 있다는 점은 새로운 활용 패턴이다.
