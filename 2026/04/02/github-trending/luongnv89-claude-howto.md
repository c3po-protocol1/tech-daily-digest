## 상세 요약

"주말에 Claude Code 마스터하기"를 목표로 하는 체계적, 시각적, 예제 중심의 학습 가이드다. 공식 문서가 기능 설명에 그치는 반면, 이 가이드는 기능들을 실제로 조합하여 워크플로우를 구축하는 방법을 가르친다.

**구성:**
- 10개 튜토리얼 모듈 — 슬래시 커맨드부터 커스텀 에이전트 팀까지
- 복사-붙여넣기 설정 — 슬래시 커맨드, CLAUDE.md 템플릿, 훅 스크립트, MCP 설정, 서브에이전트 정의, 전체 플러그인 번들
- Mermaid 다이어그램으로 내부 동작 시각화
- 11-13시간 학습 경로로 초보자→파워유저 전환
- `/self-assessment`, `/lesson-quiz` 훅으로 자기 평가 가능

**문제 해결:**
공식 문서는 개별 기능을 설명하지만, 훅 + 메모리 + 서브에이전트를 조합한 코드 리뷰 파이프라인 구축 같은 실전 워크플로우는 없다. 이 가이드가 그 gap을 채운다.

## Soo에게 의미 있는 이유

Claude Code를 단순 코딩 보조가 아닌 에이전트 플랫폼으로 활용하는 구체적 방법을 배울 수 있다. AInD 컨설팅에서 고객사에 AI 코딩 도구 도입을 지원할 때 교육 자료로도 활용 가능.
