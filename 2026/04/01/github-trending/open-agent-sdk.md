## 핵심 요약

claude-agent-sdk의 **완전 오픈소스 대안**. CLI 의존성 없이 Agent SDK 기능을 제공. 582★.

### 핵심 차별점

- **CLI 의존성 제거**: claude-agent-sdk는 Claude Code CLI에 의존하지만, 이 SDK는 독립적으로 동작
- **완전 오픈소스**: claude-agent-sdk의 핵심 기능(세션 관리, 도구 호출, 오케스트레이션)을 오픈소스로 재구현
- **TypeScript 기반**: Node.js 생태계에서 바로 사용 가능
- **Claude Code 소스 유출 직후 출시**: 유출된 소스에서 배운 아키텍처 패턴을 반영

### Claude Code 유출의 영향

이 프로젝트는 소스 유출 사건이 어떻게 **오픈소스 생태계를 촉진**하는지 보여준다. 클로즈드 소스 도구의 내부를 이해하게 되면서, 대안 구현이 더 정확하고 빠르게 이루어짐.

## Soo에게 의미 있는 이유

오픈소스 에이전트 SDK의 등장은 벤더 록인 없는 에이전트 개발의 가능성을 열어준다. Holocron이 "source-agnostic"을 추구하는 것과 같은 맥락.
