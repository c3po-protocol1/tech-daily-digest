## 핵심 요약

Claude Code의 커뮤니티 기반 포크. 1,278★. 유출된 소스를 기반으로 커뮤니티가 자체적으로 유지보수하고 개선하려는 시도.

관리자 @paidev가 주도하며, 원본 Claude Code의 기능을 유지하면서 커뮤니티 기여를 받는 구조. Anthropic의 법적 대응 가능성에도 불구하고 빠르게 성장. 오픈소스 AI 코딩 도구에 대한 강력한 수요를 보여줌.

## Soo에게 의미 있는 이유

클로즈드 소스 AI 도구의 오픈소스 대안에 대한 커뮤니티 수요의 증거. 법적/윤리적 리스크와 기술 접근성의 긴장 관계.
