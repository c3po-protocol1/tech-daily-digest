## 핵심 요약

Claude Code 소스코드의 **심층 연구 보고서**. 955★. 중국어로 작성된 상세 분석 문서를 포함하여, 코드 구조, 에이전틱 루프, 도구 시스템, 컨텍스트 관리 등을 체계적으로 분석.

단순히 소스를 공개하는 것을 넘어 **각 모듈의 역할, 데이터 흐름, 설계 결정의 이유**까지 분석하는 연구 지향적 프로젝트. Claude Code의 내부 아키텍처를 이해하려는 개발자에게 가장 체계적인 참고 자료 중 하나.

## Soo에게 의미 있는 이유

코드만 보는 것보다 분석 보고서와 함께 보면 이해가 훨씬 빠르다. Harness 아키텍처의 설계 결정을 이해하는 데 유용.
