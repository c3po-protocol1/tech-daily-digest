## 상세 요약

"vibe coding에서 agentic engineering으로"를 표방하는 Claude Code 종합 베스트 프랙티스 가이드다. GitHub Trending #1을 기록했으며, 파키스탄에서 4번째로 많은 스타를 받은 레포다.

**정리된 Claude Code 개념들:**
- **Subagents**: 격리된 컨텍스트의 자율 에이전트 (`.claude/agents/<name>.md`)
- **Commands**: 기존 컨텍스트에 주입되는 워크플로우 오케스트레이션 템플릿
- **Skills**: 구성 가능하고 자동 발견되는 지식 유닛
- **Hooks**: 에이전틱 루프 외부에서 특정 이벤트에 실행되는 핸들러
- **MCP Servers**: 외부 도구/데이터베이스/API 연결

**최신 Hot 기능 정리:**
- Routines(클라우드 자동화), Devcontainers(보안 격리), Channels(Telegram/Discord 이벤트 주입)
- Ultraplan(클라우드 계획 수립), Auto Mode(자동 권한 판단), Agent Teams(병렬 에이전트)
- Agent SDK(Python/TypeScript), Ralph Wiggum Loop(자율 개발 루프)
- Chrome 통합, Code Review, GitHub Actions, Remote Control

각 개념마다 Best Practice 문서와 구현 예제가 포함되어 있다.

## Soo에게 의미 있는 이유

Claude Code 기능의 완전한 카탈로그이자 실전 가이드다. AInD 환경에서 Claude Code를 체계적으로 활용하기 위한 참고 자료로 최적이다. 특히 Subagent, Skills, Hooks의 조합 패턴은 에이전틱 개발 방법론 설계에 직접 활용 가능하다.
