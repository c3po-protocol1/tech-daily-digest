## 상세 요약

금융 시장의 "언어"를 이해하기 위한 파운데이션 모델이다. 금융 시계열 데이터를 처리하도록 특화되어 있다.

**핵심 특징:**
- 금융 시계열 데이터에 특화된 사전 학습 모델
- Hugging Face에서 모델 공개(NeoQuasar)
- 라이브 데모 제공(shiyu-coder.github.io/Kronos-demo/)
- 다국어 문서 지원(한국어 포함)

기존 범용 시계열 모델과 달리 금융 도메인에 최적화된 아키텍처와 학습 데이터를 사용하여 시장 패턴 인식과 예측에서 더 나은 성능을 목표로 한다.

## Soo에게 의미 있는 이유

도메인 특화 파운데이션 모델의 사례로, AI가 범용에서 특화 영역으로 진화하는 트렌드를 보여준다.
