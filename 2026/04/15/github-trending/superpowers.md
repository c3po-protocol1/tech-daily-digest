## 상세 요약

코딩 에이전트를 위한 완전한 소프트웨어 개발 워크플로우 프레임워크다. 조합 가능한 "스킬"과 초기 지시 사항으로 에이전트의 행동을 체계화한다.

**워크플로우:**
1. 에이전트가 즉시 코드를 작성하지 않고, 먼저 무엇을 하려는지 질문
2. 대화에서 스펙을 도출하고, 읽을 수 있는 크기로 분할하여 제시
3. 승인 후, "열정적이지만 판단력 없는 주니어 엔지니어"도 따를 수 있는 구현 계획 수립
4. TDD, YAGNI, DRY 원칙 강조
5. **서브에이전트 기반 개발**: 에이전트가 각 태스크를 서브에이전트에게 할당, 결과 검수, 계속 진행

Claude가 별도 설정 없이 수 시간 동안 자율적으로 작업 가능. Claude Code Official Marketplace에서 플러그인으로 설치 가능.

## Soo에게 의미 있는 이유

"에이전틱 소프트웨어 개발 방법론"의 구체적 구현체다. 서브에이전트 기반 개발, TDD 강제, 스펙 → 계획 → 구현 순서 등은 R2-D2를 활용할 때도 적용 가능한 패턴이다.
