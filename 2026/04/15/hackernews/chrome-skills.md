# Chrome Skills — AI 프롬프트를 원클릭 도구로

- **출처**: [blog.google](https://blog.google/products-and-platforms/products/chrome/skills-in-chrome/)
- **HN 토론**: [Hacker News](https://news.ycombinator.com/item?id=47768339) (69 points, 37 comments)
- **태그**: #Google #Chrome #AI #생산성

## 상세 요약

Google이 Chrome 브라우저에 **Skills** 기능을 출시했다. 자주 사용하는 AI 프롬프트를 저장하여 한 번의 클릭으로 재사용할 수 있게 하는 기능이다.

**핵심 기능:**
- 채팅 기록에서 유용했던 프롬프트를 Skill로 저장
- `/` 입력 또는 `+` 버튼으로 저장된 Skill 선택
- 현재 보고 있는 페이지뿐 아니라 다른 탭도 함께 선택하여 실행 가능
- 저장된 Skill은 언제든 편집하거나 새로 생성 가능

**사용 사례:**
- 건강/웰빙: 레시피에서 단백질 매크로 즉시 계산
- 쇼핑: 여러 탭에 걸쳐 스펙 비교표 자동 생성
- 생산성: 긴 문서에서 중요 정보 스캔

**Skills 라이브러리:** Google이 미리 만든 즉시 사용 가능한 Skills도 제공. 제품 성분 분석, 예산에 맞는 선물 추천 등. 마음에 드는 것은 저장 후 프롬프트를 커스터마이징 가능.

**보안:** Chrome의 보안/프라이버시 기반 위에 구축. 캘린더 추가, 이메일 전송 등 특정 작업 전에 확인 요청. 자동 레드티밍, 자동 업데이트 보호 적용. 데스크톱 Chrome에서 롤아웃 시작.

## Soo에게 의미 있는 이유

"프롬프트를 제품화하는" 트렌드의 브라우저 확장이다. AI 활용이 일회성 대화에서 **저장 가능한 워크플로우**로 진화하는 패턴을 보여준다. 기업용 AI 도구 설계 시 "사용자가 AI 워크플로우를 직접 정의하고 공유할 수 있는 구조"를 고려할 때 참고 사례가 된다.
