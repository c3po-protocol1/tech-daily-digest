## 요약

브라우저 기본 기능부터 JavaScript 미디어 쿼리까지, 다크 모드 구현을 점진적으로 확장하는 단계별 가이드. 가장 단순한 방식부터 시작해 필요에 따라 복잡도를 높여가는 실용적 접근.

**단계별 구현:**
- Level 0: 브라우저 기본 (아무것도 안 해도 `color-scheme: dark`으로 기본 다크 지원)
- Level 1-2: CSS 미디어 쿼리 `prefers-color-scheme`
- Level 3-4: CSS 커스텀 프로퍼티로 테마 변수 관리
- Level 5-6: JavaScript로 사용자 선호도 저장, 시스템/수동 전환 지원

**핵심 원칙:** 점진적 향상(Progressive Enhancement) — 간단한 것부터 시작하고 필요할 때만 복잡도를 추가.

## Soo에게 의미 있는 이유

프론트엔드 개발의 기초 체력. AI가 코드를 생성할 때도 이런 점진적 설계 원칙을 이해해야 좋은 코드를 판별할 수 있음.
