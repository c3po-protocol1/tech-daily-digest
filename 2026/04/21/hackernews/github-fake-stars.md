# GitHub의 가짜 스타 경제: 600만 개의 조작된 스타

- **출처**: [Awesome Agents](https://awesomeagents.ai/news/github-fake-stars-investigation/)
- **HN 토론**: [Hacker News](https://news.ycombinator.com/item?id=47831621) (711점, 348댓글)
- **태그**: `#GitHub` `#오픈소스` `#스타트업` `#VC` `#보안`

## 요약

Awesome Agents의 심층 조사 기사로, GitHub 스타 조작 생태계를 체계적으로 분석했다. CMU 연구팀의 ICSE 2026 논문을 기반으로 한 데이터 기반 탐사보도.

**핵심 수치:**
- CMU 연구(StarScout): 20TB GitHub 메타데이터 분석 → **600만 개 가짜 스타**, 18,617개 레포, 301,000개 계정
- 2024년 7월 기준 50개 이상 스타를 가진 레포의 **16.66%**가 가짜 스타 캠페인에 연루
- 가격: 개당 **$0.03~$0.85**, 최소 12개 이상의 판매 웹사이트 운영 중
- AI/LLM 레포가 비악성 카테고리 중 **최대 가짜 스타 수신자** (177,000개)

**VC 파이프라인과의 연결:**
- Redpoint Ventures 분석: 시드 라운드 시 중앙값 스타 수 **2,850개**, 시리즈 A는 **4,980개**
- $85~$285로 시드 중앙값 달성 가능, ROI **3,500x~117,000x**
- Runa Capital ROSS Index 1위 프로젝트(Union Labs)에서 **47.4% 가짜 스타** 탐지

**탐지 휴리스틱 — Fork-to-Star 비율:**
- 정상 프로젝트(Flask): 1,000스타당 235포크
- 조작 의심(Shardeum): 1,000스타당 22포크
- 극단적 사례(FreeDomain): 157,000스타, 168워처 — 워처/스타 비율 0.001

**법적 리스크:**
- FTC 2024년 규칙: 가짜 소셜 영향력 지표 금지, 위반당 **$53,088** 벌금
- SEC: HeadSpin CEO 메트릭 조작으로 기소 (최대 20년형)

## Soo에게 의미 있는 이유

오픈소스 프로젝트 평가 시 스타 수만으로 판단하면 안 된다는 것을 정량적으로 증명한 기사. AInD 컨설팅에서 기술 선택 시 **Fork/Star 비율, Watcher 비율, 실제 기여자 수** 등 다층적 메트릭을 활용해야 한다. 특히 AI 관련 레포가 가장 큰 조작 대상이라는 점은 도구 선택 시 각별한 주의가 필요함을 시사.
