# World Monitor — AI 기반 실시간 글로벌 인텔리전스 대시보드

- **GitHub**: [koala73/worldmonitor](https://github.com/koala73/worldmonitor)
- **태그**: `#OSINT` `#대시보드` `#뉴스집계` `#TypeScript` `#Tauri`

## 요약

AI 기반 뉴스 집계, 지정학 모니터링, 인프라 추적을 하나의 상황 인식 인터페이스로 통합한 대시보드.

**핵심 기능:**
- 15개 카테고리의 **500+ 뉴스 피드**를 AI로 브리핑 합성
- 듀얼 맵 엔진: 3D 글로브(globe.gl) + WebGL 평면 맵(deck.gl), 45개 데이터 레이어
- 교차 스트림 상관관계: 군사, 경제, 재난, 에스컬레이션 신호 수렴 분석
- **Country Intelligence Index**: 12개 신호 카테고리 기반 복합 리스크 스코어링
- 금융 레이더: 92개 증권거래소, 원자재, 암호화폐, 7-신호 시장 복합지표
- **로컬 AI**: Ollama로 API 키 없이 모든 것 실행 가능
- 5개 사이트 변형 (world, tech, finance, commodity, happy)을 단일 코드베이스에서 생성
- Tauri 2 네이티브 데스크톱 앱 (macOS/Windows/Linux)
- **21개 언어** 지원, RTL 포함

## Soo에게 의미 있는 이유

오픈소스 OSINT 도구의 완성도가 인상적. AI 에이전트 + 데이터 집계 + 시각화의 통합 사례로, AInD 컨설팅에서 인텔리전스 대시보드 설계 참고용.
