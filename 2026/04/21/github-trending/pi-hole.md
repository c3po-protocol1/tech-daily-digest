# Pi-hole — 네트워크 레벨 광고 차단기

- **GitHub**: [pi-hole/pi-hole](https://github.com/pi-hole/pi-hole)
- **태그**: `#DNS` `#광고차단` `#네트워크` `#프라이버시`

## 요약

DNS 싱크홀 방식으로 네트워크 전체의 광고와 트래커를 차단하는 오픈소스 프로젝트. 라즈베리 파이 등 저전력 디바이스에서 실행 가능. GitHub 트렌딩에 다시 등장한 것은 최근 업데이트나 커뮤니티 활동 증가를 시사.

## Soo에게 의미 있는 이유

네트워크 보안과 프라이버시의 기본 인프라. 홈랩이나 사무실 네트워크에 배포하면 전체 디바이스의 프라이버시 향상.
