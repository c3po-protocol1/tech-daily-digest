## 요약

OpenAI의 공식 멀티에이전트 워크플로우 프레임워크. 경량이면서 강력한 에이전트 오케스트레이션 지원. 프로바이더 불가지론적(100+ LLM 지원).

**핵심 개념:**
- **Agents**: 지시, 도구, 가드레일, 핸드오프로 구성된 LLM
- **Sandbox Agents**: 컨테이너와 연동하여 장시간 작업 수행
- **Tools**: 함수, MCP, 호스티드 도구 등 다양한 도구 지원
- **Guardrails**: 입출력 검증용 안전 체크
- **Human in the loop**: 인간 개입 메커니즘 내장
- **Sessions**: 에이전트 실행 간 대화 이력 자동 관리
- **Tracing**: 에이전트 실행 추적, 디버깅, 최적화
- **Realtime Agents**: gpt-realtime-1.5로 음성 에이전트 구축

## Soo에게 의미 있는 이유

에이전트 오케스트레이션의 사실상 표준 중 하나. AInD 컨설팅에서 멀티에이전트 시스템 설계의 레퍼런스 아키텍처.
