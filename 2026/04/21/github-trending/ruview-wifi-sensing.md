# RuView — WiFi 신호로 벽 너머 사람 감지·자세 추정

- **GitHub**: [ruvnet/RuView](https://github.com/ruvnet/RuView)
- **태그**: `#WiFi센싱` `#IoT` `#Rust` `#ESP32` `#헬스케어`

## 요약

일반 WiFi 신호를 공간 인텔리전스로 변환하는 플랫폼. 카메라나 웨어러블 없이 WiFi 채널 상태 정보(CSI)만으로 감지.

**감지 능력:**
- 벽 너머 **존재 감지 및 인원 수 파악** — 100% 정확도, 0.012ms 지연
- **비접촉 생체 신호**: 호흡수(6-30 BPM), 심박수(40-120 BPM)
- **활동 인식**: 걷기, 앉기, 제스처, 낙상 감지
- **17개 COCO 키포인트 자세 추정** (WiFlow 아키텍처) — 카메라 없이 92.9% PCK@20
- 수면 품질 모니터링: 수면 단계 분류, 무호흡증 스크리닝

**하드웨어:**
- ESP32 메시 ($9/노드) + Cognitum Seed
- 6개 WiFi 채널 멀티 주파수 스캐닝 — 이웃 라우터를 무료 레이더 조명기로 활용
- 스파이킹 신경망이 30초 내 환경 학습
- Ed25519 위트니스 체인으로 모든 측정값 암호학적 증명

## Soo에게 의미 있는 이유

카메라 없는 프라이버시 보호형 센싱은 헬스케어, 스마트홈, 보안 분야의 핵심 트렌드. $9 하드웨어로 이 수준의 감지가 가능하다는 것은 IoT/AI 컨설팅 시 중요한 참고 사항.
