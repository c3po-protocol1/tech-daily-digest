# ArcKit — 엔터프라이즈 아키텍처 거버넌스 툴킷

- **GitHub**: [tractorjuice/arc-kit](https://github.com/tractorjuice/arc-kit)
- **태그**: `#엔터프라이즈` `#아키텍처` `#거버넌스` `#ClaudeCode`

## 요약

엔터프라이즈 아키텍트를 위한 구조적 거버넌스 툴킷. Claude Code 플러그인으로 설치하여 AI 지원 워크플로우 제공.

**기능 범위:**
- 아키텍처 원칙 수립·시행, 이해관계자 분석
- HM Treasury Orange Book 기반 리스크 관리
- HM Treasury Green Book 기반 비즈니스 케이스 정당화
- 요구사항 문서 생성, 데이터 모델링(ERD, GDPR), 기술 리서치(빌드 vs 구매)
- Wardley Mapping, Mermaid 아키텍처 다이어그램
- 벤더 RFP·선정, 설계 리뷰(HLD/DLD), ServiceNow 서비스 관리 설계
- 68개 명령어, 10개 자율 리서치 에이전트, 5개 자동화 훅

**Claude Code 통합:** `/plugin marketplace add tractorjuice/arc-kit`으로 바로 설치. Claude Opus 4.7의 `xhigh` effort 티어 활용.

## Soo에게 의미 있는 이유

AI 지원 엔터프라이즈 아키텍처 거버넌스의 최전선. AInD 컨설팅에서 대기업 아키텍처 설계 프로세스에 AI를 통합하는 구체적 사례로 매우 관련성 높음.
