## 요약

Google Stitch가 도입한 DESIGN.md 개념을 활용한 오픈소스 컬렉션. AI 에이전트가 읽고 일관된 UI를 생성하기 위한 **텍스트 기반 디자인 문서** 60개 이상 수록.

### 핵심 개념

- **DESIGN.md**: AGENTS.md가 행동을 정의하듯, DESIGN.md는 **시각적 외형과 느낌**을 정의
- Figma, JSON 스키마 불필요 — **마크다운 파일 하나**를 프로젝트 루트에 복사하면 AI가 즉시 UI 스타일 인식

### 9개 표준 섹션

1. Visual Theme & Atmosphere — 무드, 밀도, 디자인 철학
2. Color Palette & Roles — 색상명 + HEX + 기능적 역할
3. Typography Rules — 폰트 패밀리, 전체 계층 테이블
4. Component Stylings — 버튼, 카드, 입력, 상태
5. Layout Principles — 간격, 그리드, 여백
6. Depth & Elevation — 그림자, 서피스 계층
7. Do's and Don'ts — 가이드라인 및 안티패턴
8. Responsive Behavior — 브레이크포인트, 축소 전략
9. Agent Prompt Guide — 색상 참조, 즉시 사용 프롬프트

### 수록 서비스 (60+)

- **AI/ML**: Claude, Cohere, ElevenLabs, Mistral AI, Ollama, xAI 등
- **개발 도구**: Cursor, Linear, Vercel, Sentry, Supabase 등
- **인프라**: ClickHouse, MongoDB, Stripe 등
- **디자인/생산성**: Figma, Notion, Miro, Airtable 등
- **엔터프라이즈**: Apple, BMW, NVIDIA, SpaceX, Spotify, Uber 등

### 사용법

```bash
# Stripe 스타일 적용
cp stripe/DESIGN.md ./DESIGN.md
# AI에게: "DESIGN.md를 참조해서 결제 페이지 만들어줘"
```

## Soo에게 의미 있는 이유

AGENTS.md와 함께 AI 코딩 에이전트 생태계의 새 표준으로 자리잡을 가능성. AInD 컨설팅에서 "바이브 코딩" 워크플로 설계 시 핵심 도구. 디자이너 없이도 프로페셔널 수준의 UI를 빠르게 프로토타이핑할 수 있게 해준다.
