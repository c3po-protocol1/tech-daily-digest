# 🔥 GitNexus — 코드베이스를 지식 그래프로 인덱싱하는 AI 에이전트 도구

- **GitHub**: [abhigyanpatwari/GitNexus](https://github.com/abhigyanpatwari/GitNexus)
- **태그**: `#코드분석` `#지식그래프` `#MCP` `#AI에이전트`
- 🔥 연속 트렌딩

## 요약

모든 코드베이스를 지식 그래프로 인덱싱하여 AI 에이전트가 의존성, 호출 체인, 클러스터, 실행 흐름을 놓치지 않도록 하는 도구.

### 핵심 기능

- **CLI + MCP**: 로컬 인덱싱 후 Cursor, Claude Code, Codex 등과 MCP 연결
- **Web UI**: 시각적 그래프 탐색 + AI 채팅 (설치 없이 gitnexus.vercel.app)
- **Tree-sitter 기반 파싱**: 네이티브 바인딩 또는 WASM
- **LadybugDB**: 전용 빠른 영속 저장소
- **100% 로컬**: 네트워크 전송 없음, 완전한 프라이버시

> "DeepWiki보다 더 깊이. DeepWiki은 코드를 이해하게 하고, GitNexus는 코드를 분석하게 한다"

### 사용 시나리오

- 소규모 모델도 전체 아키텍처 파악 가능
- 의존성 누락, 호출 체인 파손, 맹목적 편집 방지

## Soo에게 의미 있는 이유

AI 코딩 에이전트의 핵심 병목인 "코드베이스 이해"를 해결하는 도구. AInD 컨설팅에서 레거시 코드 현대화 프로젝트의 첫 단계로 활용 가능.
