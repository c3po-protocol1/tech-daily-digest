# obsidian-skills — Obsidian을 위한 Agent Skills

- **GitHub**: [kepano/obsidian-skills](https://github.com/kepano/obsidian-skills)
- **태그**: `#Obsidian` `#AgentSkills` `#ClaudeCode` `#Codex`

## 요약

Obsidian 창시자 Steph Ango(kepano)가 만든, Claude Code와 Codex CLI에서 Obsidian 볼트를 다루기 위한 Agent Skills 세트. agentskills.io 스펙 준수.

### 포함된 스킬

| 스킬 | 설명 |
|------|------|
| obsidian-markdown | 위키링크, 임베드, 콜아웃, 프로퍼티 등 Obsidian 마크다운 구문 |
| obsidian-bases | Obsidian Bases — 뷰, 필터, 수식, 요약 |
| json-canvas | JSON Canvas 파일 — 노드, 엣지, 그룹, 연결 |
| obsidian-cli | Obsidian CLI를 통한 볼트 상호작용, 플러그인/테마 개발 |
| defuddle | 웹 페이지에서 클린 마크다운 추출 (토큰 절약) |

### 설치 방법

- **Marketplace**: `/plugin marketplace add kepano/obsidian-skills`
- **npx**: `npx skills add git@github.com:kepano/obsidian-skills.git`
- **수동**: `.claude/` 또는 `~/.codex/skills`에 복사

## Soo에게 의미 있는 이유

Obsidian + AI 에이전트 통합의 공식 방법. LLM-Wiki 구축 시 필수 도구. agentskills.io 표준의 실제 구현 사례로, 스킬 기반 에이전트 생태계의 성장을 보여준다.
