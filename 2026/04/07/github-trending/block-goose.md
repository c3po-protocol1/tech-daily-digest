- **태그**: `#AI에이전트` `#오픈소스` `#MCP` `#개발자도구`
- 🔥 연속 트렌딩

## 요약

Block(구 Square)이 만든 온머신 AI 에이전트. 코드 제안을 넘어 전체 프로젝트 빌드, 코드 작성·실행, 디버깅, 워크플로 오케스트레이션, 외부 API 상호작용을 자율적으로 수행.

### 핵심 특징

- **모든 LLM 지원**: 멀티 모델 구성으로 성능·비용 최적화
- **MCP 서버 통합**: 최대 유연성
- **데스크탑 앱 + CLI**: 두 가지 인터페이스
- **커스텀 배포판**: 사전 구성된 프로바이더, 확장, 브랜딩으로 자체 goose 배포판 빌드 가능
- Apache 2.0 라이선스

## Soo에게 의미 있는 이유

Claude Code, Codex와 경쟁하는 오픈소스 대안. MCP 생태계의 확산을 보여주는 사례로, AInD 컨설팅에서 "벤더 종속 없는 AI 에이전트 선택지"를 설명할 때 활용.
