# OpenScreen — Screen Studio의 무료 오픈소스 대안

- **GitHub**: [siddharthvaddem/openscreen](https://github.com/siddharthvaddem/openscreen)
- **태그**: `#화면녹화` `#오픈소스` `#제품데모` `#macOS`

## 요약

Screen Studio($29/월)의 무료 오픈소스 대안. 제품 데모와 워크스루 영상을 아름답게 만들기 위한 핵심 기능을 제공.

### 핵심 기능

- **자동/수동 줌**: 깊이 조절, 지속시간, 위치 커스텀
- **마이크 + 시스템 오디오 녹음**
- **비디오 크롭**: 불필요한 부분 숨기기
- **배경 커스텀**: 배경화면, 단색, 그라데이션, 커스텀 배경
- **모션 블러**: 부드러운 팬/줌 효과
- **어노테이션**: 텍스트, 화살표, 이미지
- **트림 & 속도 조절**: 구간별 속도 커스텀
- **다양한 비율/해상도 내보내기**

### 지원 플랫폼

- macOS, Linux (Windows 추후)
- 100% 무료, 개인/상업 사용 모두 가능

### 참고

- Screen Studio의 1:1 클론이 아닌 "더 심플한 버전"
- beta 단계 — 약간의 버그 가능성

## Soo에게 의미 있는 이유

AInD 컨설팅 결과물을 데모 영상으로 제작할 때 활용 가능한 무료 도구. 제품 발표, 포트폴리오 영상 제작에 Screen Studio 구독 없이 사용 가능.
