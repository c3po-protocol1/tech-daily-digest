## 요약

Google Cloud AI 디렉터 Addy Osmani가 만든, AI 코딩 에이전트가 시니어 엔지니어 수준의 워크플로우를 따르도록 하는 **20개 구조화된 스킬** 모음.

**7개 슬래시 커맨드 (개발 생명주기 매핑):**
| 커맨드 | 원칙 |
|--------|------|
| `/spec` | 코드 전에 스펙부터 |
| `/plan` | 작고 원자적인 태스크 |
| `/build` | 한 번에 한 슬라이스 |
| `/test` | 테스트가 증명 |
| `/review` | 코드 건강도 개선 |
| `/code-simplify` | 명확성 > 영리함 |
| `/ship` | 빠른 것이 안전 |

**20개 스킬 카테고리:**
- **Define**: idea-refine, spec-driven-development
- **Plan**: planning-and-task-breakdown
- **Build**: incremental-implementation, test-driven-development
- **Verify/Review**: 검증 게이트와 반합리화(anti-rationalization) 테이블 포함

**지원 플랫폼:**
Claude Code (마켓플레이스 설치), Cursor (.cursor/rules/), Gemini CLI (네이티브 스킬), Windsurf, OpenCode, GitHub Copilot, Kiro IDE 등

스킬은 순수 Markdown — 시스템 프롬프트나 명령어 파일을 받는 모든 에이전트에서 동작한다.

## Soo에게 의미 있는 이유

하네스 엔지니어링의 실전 도구. AI 코딩 에이전트가 "시니어 엔지니어처럼" 일하게 만드는 구체적인 방법론을 제공한다. OpenClaw의 AGENTS.md에 이 스킬들을 적용하면 R2-D2의 코드 품질을 체계적으로 높일 수 있다.
