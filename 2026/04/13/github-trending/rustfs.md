# rustfs/rustfs — Rust 기반 고성능 분산 오브젝트 스토리지

> **GitHub**: [rustfs/rustfs](https://github.com/rustfs/rustfs)
> **태그**: `#Rust` `#스토리지` `#S3호환` `#인프라` `#오픈소스`

## 요약

MinIO의 단순함과 Rust의 메모리 안전성 + 성능을 결합한 **고성능 분산 오브젝트 스토리지 시스템**.

**핵심 특징:**
- **S3 완전 호환**: 기존 S3 호환 앱/도구와 원활한 통합
- **OpenStack Swift API**: Swift 프로토콜 + Keystone 인증 네이티브 지원
- **데이터 레이크 최적화**: 빅데이터 및 AI 워크로드에 최적화
- **Apache 2.0 라이선스**: AGPL 제한 없는 상업 사용 가능 (MinIO와의 차별점)

**벤치마크 환경** (2코어, 4GB RAM, 40GB×4 드라이브):
MinIO 대비 성능 우위를 보여주며, Rust의 GC 없는 특성으로 안정적인 레이턴시 제공.

**기능 상태:**
- ✅: S3 코어, 업로드/다운로드, 버전닝, 로깅, 이벤트 알림, K8s Helm, Bitrot 보호, 버킷 복제
- 🚧: 라이프사이클 관리, 분산 모드, KMS

ROSS Index Q4 2025 가장 빠르게 성장하는 오픈소스 프로젝트로 선정.

## Soo에게 의미 있는 이유

AI/ML 워크로드의 데이터 스토리지에서 S3 호환 시스템은 필수. MinIO의 AGPL 라이선스가 기업 도입의 걸림돌인 경우가 많은데, Apache 2.0 RustFS는 좋은 대안. 데이터 레이크/AI 파이프라인 아키텍처에서 참고.
