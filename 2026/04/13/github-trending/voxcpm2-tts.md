## 요약

VoxCPM2는 **토크나이저 없이** 연속 음성 표현을 직접 생성하는 end-to-end 확산 자기회귀 아키텍처 기반 TTS 시스템. 2B 파라미터, 200만 시간 이상의 다국어 음성 데이터로 훈련.

**핵심 하이라이트:**
- 🌍 **30개 언어** 지원 — 언어 태그 없이 입력 텍스트의 언어를 자동 감지
- 🎨 **Voice Design** — 자연어 설명만으로 새로운 음성 생성 (성별, 나이, 톤, 감정, 속도...)
- 🎛️ **제어 가능한 클로닝** — 짧은 참조 클립에서 음성 복제 + 스타일 가이던스로 감정/속도 제어
- 🎙️ **Ultimate Cloning** — 참조 오디오 + 스크립트 제공 시, 음색/리듬/감정/스타일까지 충실 재현
- 🔊 **48kHz 고품질 출력** — 16kHz 입력 → 48kHz 스튜디오급 출력 (내장 슈퍼레졸루션)
- ⚡ RTF ~0.3 (RTX 4090), Nano-VLLM 가속 시 ~0.13

MiniCPM-4 백본 기반, Apache 2.0 라이선스 (상업 사용 가능).

중국어 방언(사천, 광동, 동북 등 9개) 지원. HuggingFace와 ModelScope에서 가중치 제공.

## Soo에게 의미 있는 이유

오픈소스 TTS의 새로운 기준. "Voice Design"으로 자연어 설명만으로 음성을 생성하는 것은 AI 에이전트의 음성 인터페이스에 혁신적. 30개 언어 + 상업 가능 라이선스는 글로벌 서비스에 즉시 적용 가능.
