# OpenCode — 오픈소스 AI 코딩 에이전트 🔥

- **GitHub**: [anomalyco/opencode](https://github.com/anomalyco/opencode)
- **스타**: ⭐ 145,041 | **언어**: TypeScript
- **태그**: `#코딩에이전트` `#오픈소스` `#AI` `#CLI` 🔥

## 요약

**145K+ 스타**를 달성한 오픈소스 AI 코딩 에이전트. Claude Code, Codex와 경쟁하는 완전 오픈소스 대안이다.

### 특징

- **완전 오픈소스**: 소스 코드 전체 공개
- **다국어 README**: 한국어 포함 20+ 언어 README 제공
- **다양한 설치 방법**: npm, Homebrew, Scoop, Chocolatey, pacman, mise 등
- **터미널 UI**: 아름다운 TUI 인터페이스

### 설치

```bash
curl -fsSL https://opencode.ai/install | bash
# 또는
npm i -g opencode-ai@latest
brew install anomalyco/tap/opencode
```

### 위상

GitHub에서 가장 많은 스타를 받은 코딩 에이전트 프로젝트 중 하나로, **오픈소스 코딩 에이전트의 대표 주자**로 자리잡았다.

## Soo에게 의미 있는 이유

145K 스타는 오픈소스 코딩 에이전트에 대한 엄청난 수요를 보여준다. Claude Code(프로프라이어터리)와 OpenCode(오픈소스)의 경쟁은 **코딩 에이전트 시장의 양대 축**이 되어가고 있으며, 기업 클라이언트에게 선택지를 제공할 때 참고해야 한다.
