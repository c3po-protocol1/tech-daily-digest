# T3 Code — 코딩 에이전트를 위한 미니멀 웹 GUI

- **GitHub**: [pingdotgg/t3code](https://github.com/pingdotgg/t3code)
- **스타**: ⭐ 9,422 | **언어**: TypeScript
- **태그**: `#코딩에이전트` `#GUI` `#Codex` `#ClaudeCode`

## 요약

Codex와 Claude Code를 위한 **미니멀 웹 GUI**. 커맨드라인 없이 코딩 에이전트를 사용할 수 있게 해준다.

### 주요 특징

- `npx t3`로 설치 없이 즉시 실행
- **데스크톱 앱** 지원: Windows(winget), macOS(Homebrew), Arch Linux(AUR)
- Codex와 Claude Code 지원 (더 많은 프로바이더 추가 예정)
- 매우 초기 단계, 기여 아직 받지 않음

### 설치

```bash
npx t3                           # 설치 없이 실행
brew install --cask t3-code      # macOS
winget install T3Tools.T3Code    # Windows
```

## Soo에게 의미 있는 이유

T3 Stack으로 유명한 Theo(ping.gg)의 프로젝트. CLI 기반 코딩 에이전트에 GUI를 씌우는 트렌드가 확산되고 있으며, 9.4K 스타는 이 수요가 크다는 것을 보여준다.
