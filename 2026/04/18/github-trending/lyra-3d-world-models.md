## 요약

NVIDIA Spatial Intelligence Lab의 **오픈 3D 월드 생성 모델** 시리즈. Lyra 2.0은 **탐색 가능한 생성형 3D 세계**를 장기간 3D 일관성 있게 생성한다.

### 버전 비교

- **Lyra 1.0** (2025.09): 단일 이미지/비디오에서 3D/4D 씬 생성. 비디오 확산 모델 자기증류(self-distillation) 방식
- **Lyra 2.0** (2026.04): 탐색 가능한 3D 세계. 장기간 3D 일관성 있는 생성

### 기술 특징

- 논문, 프로젝트 페이지, HuggingFace 모델 체크포인트, 코드 모두 공개
- Apache 2.0 라이선스 (소스 코드)

## Soo에게 의미 있는 이유

3D 생성 AI는 게임, 시뮬레이션, 디지털 트윈 등 다양한 산업에 적용 가능하다. NVIDIA가 이를 오픈소스로 공개한 것은 3D 생성 기술의 민주화를 가속화한다.
