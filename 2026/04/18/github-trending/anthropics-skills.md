# Anthropic Agent Skills — 공식 스킬 저장소 🔥

- **GitHub**: [anthropics/skills](https://github.com/anthropics/skills)
- **스타**: ⭐ 119,582 | **언어**: Python
- **태그**: `#Anthropic` `#AgentSkills` `#Claude` `#AI에이전트` 🔥

## 요약

Anthropic의 **공식 Agent Skills 저장소**. Claude가 동적으로 로드하여 특화 작업의 성능을 높이는 지시, 스크립트, 리소스 폴더 모음이다. **119K+ 스타**로 엄청난 인기를 보이고 있다.

### Agent Skills란?

- SKILL.md 파일과 YAML 프론트매터로 구성된 자기 완결적 폴더
- 브랜드 가이드라인 적용, 조직 워크플로우 데이터 분석, 개인 작업 자동화 등
- **Claude Code, Claude.ai, Claude API** 모두에서 사용 가능

### 주요 스킬 카테고리

- **Creative & Design**: 아트, 음악, 디자인
- **Development & Technical**: 웹 앱 테스팅, MCP 서버 생성
- **Enterprise & Communication**: 커뮤니케이션, 브랜딩
- **Document Skills**: PDF, DOCX, PPTX, XLSX 생성·편집 (소스 공개, 프로덕션 사용 중)

### Claude Code에서 사용

```bash
/plugin marketplace add anthropics/skills
/plugin install document-skills@anthropic-agent-skills
```

Agent Skills는 Thoughtworks Radar에서도 "Adopt" 등급으로 올라간 핵심 기법이다. 119K 스타는 **에이전트 커스터마이제이션이 개발자들의 최대 관심사**임을 보여주며, AInD 컨설팅에서 클라이언트별 맞춤 스킬 설계는 핵심 서비스가 될 수 있다.
