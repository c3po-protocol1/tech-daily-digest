# Codex for (Almost) Everything — OpenAI Codex 대규모 업데이트

- **출처**: [OpenAI 블로그](https://openai.com/index/codex-for-almost-everything/)
- **HN 토론**: [Hacker News](https://news.ycombinator.com/item?id=47796469) (602 points)
- **태그**: `#AI` `#OpenAI` `#Codex` `#에이전트` `#코딩`

## 상세 요약

OpenAI가 매주 300만 명 이상의 개발자가 사용하는 Codex에 대규모 업데이트를 발표했다. 단순한 코딩 도구를 넘어 **소프트웨어 개발 전체 생명주기를 커버하는 종합 파트너**로 진화했다.

### 핵심 신기능
- **백그라운드 컴퓨터 사용**: Codex가 사용자의 Mac에서 **별도의 커서**로 앱을 보고, 클릭하고, 타이핑 가능. 여러 에이전트가 병렬로 작동하면서 사용자 작업을 방해하지 않음
- **인앱 브라우저**: 페이지에 직접 코멘트를 달아 에이전트에게 정확한 지시를 줄 수 있음. 프론트엔드/게임 개발에 유용
- **이미지 생성**: gpt-image-1.5를 활용한 이미지 생성/반복. 프로덕트 목업, UI 디자인을 같은 워크플로우 내에서 처리
- **90개 이상의 새 플러그인**: JIRA(Atlassian Rovo), CircleCI, CodeRabbit, GitLab Issues, Microsoft Suite, Neon, Render 등
- **PR 리뷰 코멘트 대응**: GitHub 리뷰 코멘트를 직접 처리
- **원격 devbox SSH 연결** (알파)
- **자동화 확장**: 기존 대화 스레드 재사용으로 컨텍스트 유지. 장기 작업을 위해 스스로 스케줄링하고 며칠~몇 주에 걸쳐 자동 기상
- **메모리 프리뷰**: 이전 경험에서 유용한 컨텍스트를 기억. 개인 선호, 수정사항, 정보를 학습
- **능동적 작업 제안**: 프로젝트, 플러그인, 메모리 컨텍스트를 활용해 하루 시작 시 우선순위 작업 제안

### 개발 라이프사이클 전체 커버
코드 작성 → 출력 확인 → 변경 리뷰 → 에이전트 협업을 하나의 워크스페이스에서 처리. 여러 터미널 탭, 파일 사이드바 프리뷰(PDF, 스프레드시트, 슬라이드, 문서), 요약 패인(에이전트 계획/소스/아티팩트 추적) 지원.

## Soo에게 의미 있는 이유
Codex가 "코딩 도우미"에서 "소프트웨어 개발 전체를 관리하는 자율 에이전트"로 진화하고 있다. 특히 백그라운드 컴퓨터 사용, 메모리, 장기 자동화 기능은 AInD 컨설팅에서 "AI 네이티브 개발"의 실체를 보여주는 핵심 사례다. Claude Code와의 직접 비교 포인트로도 중요하다.
