## 상세 요약

**GEP(Genome Evolution Protocol)**를 기반으로 AI 에이전트가 스스로 진화하는 엔진. 임시적인 프롬프트 조정을 감사 가능하고 재사용 가능한 진화 자산으로 변환한다.

### 핵심 특징
- **프롬프트 생성기**: 코드를 직접 패치하지 않음. 매 진화 주기마다 GEP 준수 프롬프트 생성
- **감사 가능**: 모든 진화 이벤트가 EvolutionEvent로 기록
- **오프라인 동작**: 인터넷 연결 없이 핵심 기능 완전 동작
- **EvoMap 네트워크**: evomap.ai에서 에이전트 진화 리더보드, 스킬 공유

### 진화 사이클
1. `memory/` 디렉토리의 런타임 로그, 에러 패턴, 시그널 스캔
2. `assets/gep/`에서 최적 Gene/Capsule 선택
3. 프로토콜 준수 GEP 프롬프트 생성
4. 추적 가능한 EvolutionEvent 기록

### 보안 모델
- 소스 코드 자동 편집 없음
- 임의의 쉘 명령 실행 없음
- Git을 활용한 롤백, 폭발 반경 계산

## Soo에게 의미 있는 이유
에이전트의 "자기 개선"을 프로토콜화하고 감사 가능하게 만든 접근이 흥미롭다. AInD 컨설팅에서 에이전트 거버넌스와 진화 관리 전략을 설계할 때 참고할 수 있다.
