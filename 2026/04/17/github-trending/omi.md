## 상세 요약

화면과 대화를 캡처하고, 실시간 전사, 요약, 액션 아이템 생성, 그리고 보고 들은 모든 것을 기억하는 **AI 제2의 뇌**. 30만 명 이상이 사용 중.

### 핵심 기능
- 화면 캡처 + 대화 전사 실시간 처리
- 요약 및 액션 아이템 자동 생성
- 보고 들은 모든 것을 기억하는 AI 채팅
- 데스크톱(macOS), 모바일(iOS/Android), 웨어러블 지원
- 웹 브라우저에서도 사용 가능
- 완전 오픈소스 (MIT 라이선스)

### 빠른 시작
```bash
git clone https://github.com/BasedHardware/omi.git
cd omi/desktop && ./run.sh --yolo
```
env 파일, 인증 정보, 로컬 백엔드 없이 바로 실행.

### 기술 스택
- macOS 14+, Xcode, Node.js 필요
- 백엔드: Rust
- 모바일: Flutter/Dart

## Soo에게 의미 있는 이유
"AI가 모든 컨텍스트를 기억하는" 패러다임은 AInD의 미래 방향과 일치한다. 화면+음성+텍스트를 통합한 멀티모달 메모리 시스템의 실제 구현 사례로 참고 가치가 크다.
