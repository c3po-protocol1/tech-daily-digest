## 상세 요약

**~3K 줄의 핵심 코드**에서 시작해 자율적으로 스킬 트리를 성장시키는 에이전트 프레임워크. 9개의 원자적 도구 + ~100줄의 Agent Loop로 LLM에게 시스템 수준의 제어권을 부여한다.

### 핵심 컨셉
- **자기 진화**: 새 작업을 해결할 때마다 실행 경로를 자동으로 스킬로 결정화
- **9개 원자적 도구**: 브라우저, 터미널, 파일시스템, 키보드/마우스, 화면 비전, 모바일(ADB) 제어
- **<30K 컨텍스트 윈도우**: 다른 에이전트의 200K~1M 대비 극적으로 적은 토큰 소비 (6배 절감)
- **계층적 메모리**: 적절한 지식이 항상 범위 내에 있도록 보장
- **셀프 부트스트랩 증명**: 리포의 모든 것(git init부터 커밋 메시지까지)이 에이전트가 자율적으로 완성

### 사용 예시
| 명령 | 첫 실행 | 이후 |
|---|---|---|
| "WeChat 메시지 읽어" | 의존성 설치 → DB 리버스 → 스크립트 작성 → 스킬 저장 | 한 줄 호출 |
| "주식 모니터링+알림" | mootdx 설치 → 선택 플로우 → cron 설정 → 스킬 저장 | 한 줄 시작 |
| "Gmail로 파일 전송" | OAuth 설정 → 전송 스크립트 → 스킬 저장 | 바로 사용 |
