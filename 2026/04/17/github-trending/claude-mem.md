## 상세 요약

Claude Code 세션 중 에이전트가 수행하는 **모든 작업을 자동으로 캡처하고 AI로 압축**하는 영속적 메모리 시스템.

### 핵심 기능
- Claude Code 플러그인 형태로 동작
- 코딩 세션의 모든 활동을 자동 기록
- Claude의 agent-sdk를 사용해 AI로 메모리 압축
- 세션 간 컨텍스트 유지
- v6.5.0 버전, Node.js 18+ 필요
- AGPL 3.0 라이선스

### 왜 필요한가
Claude Code는 세션마다 컨텍스트가 리셋된다. claude-mem은 이전 세션의 결정, 패턴, 학습을 자동으로 기억하여 다음 세션에서 더 나은 코드를 생성하게 한���.
