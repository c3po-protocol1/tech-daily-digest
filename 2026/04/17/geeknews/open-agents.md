## 상세 요약

Vercel Labs가 공개한 오픈소스 레퍼런스 앱으로, **백그라운드 코딩 에이전트**를 직접 구축하고 실행할 수 있는 환경을 제공한다.

### 핵심 특징
- Web → Agent Worker → 코드 실행의 완전한 파이프라인
- TypeScript 기반의 클라우드 에이전트 템플릿
- 에이전트 세션 관리, 상태 추적
- 다양한 LLM 프로바이더 지원
- 오픈소스로 에이전트 아키텍처 학습에 최적
