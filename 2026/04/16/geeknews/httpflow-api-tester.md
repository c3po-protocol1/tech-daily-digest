## 상세 요약

HTTPFlow는 `.http` 파일을 비주얼 노드 그래프로 변환하여 API 요청 흐름을 시각화하고 테스트할 수 있는 도구다. Show GN(GeekNews 프로젝트 쇼케이스)으로 공개되었다.

기존의 텍스트 기반 `.http` 파일을 노드 그래프 형태로 시각화하여, API 호출 간의 의존성과 데이터 흐름을 직관적으로 파악할 수 있게 해준다. Postman이나 Insomnia 같은 기존 API 테스트 도구와는 다른 접근 방식을 취한다.

## Soo에게 의미 있는 이유

API 중심 개발에서 요청 흐름의 시각화는 생산성을 높일 수 있다. AI 에이전트가 API를 호출하는 복잡한 워크플로우를 디버깅할 때도 유용할 수 있는 도구 유형이다.
