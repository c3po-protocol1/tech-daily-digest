## 상세 요약

아폴로 11호 달 착륙에 사용된 유도 컴퓨터(AGC)의 원본 소스코드를 공개한 저장소다. 명령 모듈(Command Module)과 달 착륙 모듈(Lunar Module)의 코드가 포함되어 있다.

1960년대 후반 MIT Instrumentation Lab에서 작성된 이 코드는 인류 최초의 달 착륙을 가능하게 한 소프트웨어다. 제한된 메모리(약 74KB)와 처리 능력으로 실시간 항법과 유도를 수행했다는 점에서 소프트웨어 엔지니어링의 걸작으로 평가된다.

주기적으로 GitHub 트렌딩에 올라오는 인기 저장소로, 오늘 다시 트렌딩에 진입했다.

## Soo에게 의미 있는 이유

극한의 제약 조건에서의 소프트웨어 엔지니어링 사례. AI 시대에 "코드의 효율성"과 "시스템 신뢰성"에 대해 생각하게 만드는 역사적 참조 자료.
