# Onyx — 오픈소스 AI 플랫폼

- **GitHub**: [https://github.com/onyx-dot-app/onyx](https://github.com/onyx-dot-app/onyx)
- **태그**: `ai-platform` `rag` `agents` `open-source` `self-hosted`

## 상세 요약

Onyx는 LLM을 위한 애플리케이션 레이어로, RAG, 웹 검색, 코드 실행, 파일 생성, 딥 리서치 등 풍부한 기능을 갖춘 오픈소스 AI 플랫폼이다.

### 핵심 기능
- **Agentic RAG**: 하이브리드 인덱스 + AI 에이전트 기반 최고 수준의 검색·답변 품질
- **Deep Research**: 다단계 리서치 플로우를 통한 심층 보고서 생성 (벤치마크 리더보드 1위, 2026.02 기준)
- **커스텀 에이전트**: 고유 지시사항, 지식, 액션을 갖춘 AI 에이전트 구축
- **웹 검색**: Serper, Google PSE, Brave, SearXNG 등 지원 + 자체 웹 크롤러
- **Artifacts**: 문서, 그래픽, 다운로드 가능한 아티팩트 생성
- **Actions & MCP**: 외부 앱과 상호작용, 유연한 Auth 옵션
- **코드 실행**: 샌드박스에서 데이터 분석, 그래프 렌더링, 파일 수정
- **음성 모드**: TTS/STT를 통한 대화
- **이미지 생성**: 프롬프트 기반 이미지 생성
- **LLM 분석**: LLM 앱의 trace, 생성, 지연, 비용 캡처
- **50+ 커넥터**: 기본 제공 인덱싱 커넥터 또는 MCP로 앱 연결

### 배포 모드
- **Onyx Lite**: 1GB 이하 메모리로 경량 Chat UI (테스트용/채팅+에이전트만 사용)
- **Standard Onyx**: 전체 기능 포함 풀스택 배포
- Docker, Kubernetes, Helm/Terraform, 주요 클라우드 프로바이더 지원
- 단일 명령 배포: `curl -fsSL https://onyx.app/install_onyx.sh | bash`

## Soo에게 의미 있는 이유

오픈소스 AI 플랫폼의 현재 수준을 보여주는 종합적 프로젝트. RAG + 에이전트 + 코드 실행 + MCP 등 AI 앱 개발의 전체 스택을 단일 플랫폼으로 제공하며, AInD 컨설팅에서 기업에 자체 호스팅 AI 플랫폼 옵션으로 추천할 수 있다.
