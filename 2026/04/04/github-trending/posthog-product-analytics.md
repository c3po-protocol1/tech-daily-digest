# PostHog — 올인원 오픈소스 프로덕트 분석 플랫폼

- **GitHub**: [https://github.com/PostHog/posthog](https://github.com/PostHog/posthog)
- **태그**: `analytics` `product` `open-source` `feature-flags` `session-replay`

## 상세 요약

PostHog는 성공적인 제품 구축을 위한 모든 도구를 하나의 오픈소스 플랫폼으로 제공하는 올인원 프로덕트 분석 도구이다.

### 제공 기능
- **Product Analytics**: 이벤트 기반 분석, 시각화, SQL 쿼리
- **Web Analytics**: GA와 유사한 대시보드로 웹 트래픽, 세션, 전환, Web Vitals 모니터링
- **Session Replays**: 실제 사용자 세션 녹화 및 재생 (웹/모바일)
- **Feature Flags**: 특정 사용자/코호트에 안전하게 기능 출시
- **Experiments (A/B 테스트)**: 변경 사항의 통계적 영향 측정, 노코드 실험 설정
- **Error Tracking**: 에러 추적, 알림, 이슈 해결
- **Surveys**: 노코드 설문 템플릿 + 커스텀 설문 빌더
- **Data Warehouse**: Stripe, Hubspot 등 외부 데이터 동기화 및 제품 데이터와 결합 쿼리
- **Data Pipelines**: 커스텀 필터/변환 + 25+ 도구로 실시간/배치 내보내기
- **LLM Analytics**: LLM 앱의 trace, 생성, 지연, 비용 캡처
- **Workflows**: 사용자 행동 기반 자동화 액션/메시지 전송

### 무료 티어
- 월 100만 이벤트, 5K 녹화, 1M 플래그 요청, 100K 예외, 1,500 설문 응답 무료
- US/EU 클라우드 또는 셀프 호스팅

## Soo에게 의미 있는 이유

AI 프로덕트의 분석 인프라로 사용할 수 있는 종합 도구. 특히 LLM Analytics 기능은 AI 앱의 성능과 비용을 추적하는 데 직접 활용 가능하며, AInD 프로젝트의 측정 가능한 성과 지표를 설정할 때 참고할 플랫폼이다.
