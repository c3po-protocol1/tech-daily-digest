# Baserow — 노코드 데이터베이스·자동화·AI 에이전트 플랫폼

- **GitHub**: [https://github.com/baserow/baserow](https://github.com/baserow/baserow)
- **태그**: `no-code` `database` `automation` `ai-agents` `airtable-alternative`

## 상세 요약

Baserow는 데이터베이스, 애플리케이션, 자동화, AI 에이전트를 코드 없이 구축할 수 있는 보안 중심 오픈소스 플랫폼이다. 15만 명 이상의 사용자가 사용하는 Airtable의 대안이다.

### 핵심 기능
- **스프레드시트-데이터베이스 하이브리드**: 스프레드시트의 사용 편의성 + 데이터베이스의 강력한 구조화
- **애플리케이션 빌더**: 앱과 포털 생성 후 자체 도메인에 배포
- **자동화**: 반복 워크플로 자동화
- **대시보드**: 데이터 시각화
- **Kuma AI 어시스턴트**: 자연어로 데이터베이스와 워크플로를 구축하는 AI 기반 빌더
- **규정 준수**: GDPR, HIPAA, SOC 2 Type II 인증
- **셀프 호스팅**: 스토리지 제한 없이 자체 서버에 배포 가능
- **API 퍼스트**: Headless 아키텍처로 기존 도구와 원활한 통합
- **기술 스택**: Django, Vue.js, PostgreSQL

### 배포
```bash
docker run -v baserow_data:/baserow/data -p 80:80 -p 443:443 baserow/baserow:2.1.6
```
Heroku 원클릭 배포도 지원.

### 라이선스
비프리미엄/비엔터프라이즈 기능은 MIT 라이선스로 상용/개인 사용 모두 허용.

## Soo에게 의미 있는 이유

AI 에이전트 + 노코드 + 데이터베이스의 융합 트렌드를 보여주는 플랫폼. 기업의 비기술 인력이 AI를 활용하여 데이터 기반 앱을 구축하는 패턴은 AInD 컨설팅의 핵심 제안 포인트가 될 수 있다.
