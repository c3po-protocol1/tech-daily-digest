# Snipe-IT — 오픈소스 IT 자산 관리 시스템

- **GitHub**: [https://github.com/grokability/snipe-it](https://github.com/grokability/snipe-it)
- **태그**: `asset-management` `it-ops` `laravel` `open-source`

## 상세 요약

Snipe-IT는 IT 운영팀을 위한 오픈소스 자산 관리 시스템으로, 노트북 할당, 구매 이력, 감가상각 추적, 소프트웨어 라이선스 관리 등을 처리한다.

### 핵심 특징
- **Laravel 11 기반**: PHP 프레임워크 Laravel로 구축된 웹 기반 소프트웨어
- **활발한 개발**: 빈번한 릴리스 (GitHub Releases에서 확인 가능)
- **웹 기반**: .exe 파일 없음, 웹 서버에서 실행하고 브라우저로 접근
- **크로스 플랫폼**: Mac OSX, Linux, Windows + Docker 이미지 제공
- **라이브 데모**: [snipeitapp.com/demo](https://snipeitapp.com/demo)에서 체험 가능
- **다국어 지원**: Crowdin 기반 번역 (331명 기여자)
- **보안**: 보안 취약점은 별도 이메일(security@snipeitapp.com)로 보고

### 용도
- 누가 어떤 노트북을 사용하는지 추적
- 구매 시점 기반 정확한 감가상각 계산
- 소프트웨어 라이선스 관리
- IT 자산의 전체 라이프사이클 관리

## Soo에게 의미 있는 이유

AI가 IT 운영을 자동화하는 시나리오에서 기초 데이터를 제공하는 시스템. AI 에이전트가 자산 정보를 조회하여 의사결정을 지원하는 패턴에서 Snipe-IT 같은 시스템이 데이터 소스가 된다.
