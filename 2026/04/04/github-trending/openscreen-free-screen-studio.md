# OpenScreen — Screen Studio의 무료 오픈소스 대안

- **GitHub**: [https://github.com/siddharthvaddem/openscreen](https://github.com/siddharthvaddem/openscreen)
- **태그**: `screen-recording` `video-editing` `open-source` `electron`

## 상세 요약

OpenScreen은 Screen Studio($29/월)의 무료 오픈소스 대안으로, 아름다운 제품 데모와 워크스루를 만들 수 있는 앱이다.

### 핵심 기능
- **화면 녹화**: 전체 화면 또는 특정 윈도우 녹화
- **자동/수동 줌**: 자동 줌 또는 커스터마이즈 가능한 수동 줌 (깊이 레벨 조절)
- **오디오 캡처**: 마이크 오디오 + 시스템 오디오 동시 캡처
- **배경 커스터마이징**: 벽지, 단색, 그라디언트, 커스텀 배경
- **모션 블러**: 부드러운 팬/줌 효과를 위한 모션 블러
- **주석(Annotations)**: 텍스트, 화살표, 이미지 추가
- **트리밍**: 클립의 특정 구간 트림
- **속도 조절**: 구간별 다른 속도 설정
- **다양한 내보내기**: 다양한 종횡비와 해상도로 내보내기
- **자르기(Crop)**: 비디오 녹화의 특정 부분 숨기기

### 플랫폼 지원
- macOS (Gatekeeper 우회 필요: `xattr -rd com.apple.quarantine`)
- Linux (AppImage)
- 100% 무료, 개인 및 상용 사용 가능

### 주의사항
- 베타 단계로 버그가 있을 수 있음
- Screen Studio의 1:1 클론이 아닌 심플한 버전

## Soo에게 의미 있는 이유

개발자를 위한 제품 데모 제작 도구. AI 프로젝트의 데모 영상을 만들 때 유용하며, 오픈소스이므로 비용 부담 없이 활용 가능하다.
