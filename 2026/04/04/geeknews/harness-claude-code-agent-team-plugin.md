## 상세 요약

"하네스 구성해줘" 한 마디로 도메인에 맞는 전문 에이전트 팀을 설계하고, 에이전트가 사용할 스킬까지 자동 생성해주는 Claude Code 메타 스킬(플러그인)이다.

### 핵심 개념
Harness는 Claude Code의 에이전트 팀 시스템을 활용하여 복잡한 작업을 전문화된 에이전트 팀으로 분해한다:
- **메타 스킬**: 도메인을 분석하여 필요한 에이전트 구성을 자동 설계
- **에이전트 팀 설계**: 각 에이전트의 역할, 전문 분야, 사용할 도구를 정의
- **스킬 자동 생성**: 설계된 에이전트가 사용할 스킬을 코드로 자동 생성

### 특징
- 자연어 명령 하나로 전체 에이전트 팀 아키텍처 생성
- 도메인 특화 에이전트 정의
- 에이전트별 전문 스킬 자동 생성
- Claude Code 플러그인으로 설치하여 바로 사용

이 프로젝트는 GeekNews에서 116 포인트로 매우 높은 관심을 받았다.

## Soo에게 의미 있는 이유

멀티 에이전트 팀의 자동 구성이라는 개념은 AI 에이전트 아키텍처의 미래 방향을 보여준다. "메타 스킬"이 하위 에이전트와 스킬을 자동 생성하는 패턴은 AInD에서 기업의 복잡한 워크플로를 에이전트화하는 데 직접 적용 가능한 접근법이다.
