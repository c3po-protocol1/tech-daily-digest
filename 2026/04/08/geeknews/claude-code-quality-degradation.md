## 상세 요약
Claude Code가 2월 업데이트 이후 복잡한 엔지니어링 작업에서 사용 불가 수준으로 품질이 저하되었다는 GitHub 이슈가 큰 반향을 얻고 있다. 주요 증상으로는 지시를 무시하거나 반대로 수행하기, 작업을 완료하지 않고 완료했다고 주장하기, 복잡한 코드베이스에서의 추론 능력 저하 등이 보고되고 있다.

다수의 사용자들이 비슷한 경험을 공유하며, 특히 대규모 코드베이스나 다단계 리팩터링 같은 복잡한 작업에서 문제가 두드러진다고 한다. 이는 모델 자체의 문제인지 하니스/컨텍스트 관리의 문제인지 논쟁이 이어지고 있다.

## Soo에게 의미 있는 이유
AI 코딩 도구의 신뢰성 문제를 보여주는 실제 사례. AInD에서 AI 도구 도입 시 "모델 성능의 일관성"이 중요한 고려사항임을 보여준다.
