# rtk - LLM 토큰 소비를 60~90% 줄여주는 CLI 프록시
- **출처:** https://github.com/rtk-ai/rtk
- **GeekNews:** https://news.hada.io/topic?id=28245
- **점수:** 31 points
- **태그:** `#AI` `#토큰절약` `#CLI` `#Rust` 🔥

## 상세 요약
AI 코딩 도구가 실행하는 CLI 명령어 출력을 LLM에 전달하기 전에 필터링·압축해 토큰을 60~90% 절감하는 단일 Rust 바이너리다. Windows/Mac/Linux를 지원하며 외부 의존성이 없다.

코딩 에이전트가 shell 명령을 실행할 때 발생하는 방대한 출력(빌드 로그, 테스트 결과 등)을 지능적으로 압축하여 LLM에 전달한다. 이를 통해 컨텍스트 윈도우를 효율적으로 사용하고 API 비용도 절감한다.

## Soo에게 의미 있는 이유
코딩 에이전트 비용 최적화를 위한 실용적 도구. OpenClaw에서 R2-D2 같은 코딩 에이전트의 토큰 비용을 줄이는 데 직접 활용 가능.
