# 📚 Claude Cookbooks — Anthropic 공식 개발 레시피 모음

- **저장소**: [anthropics/claude-cookbooks](https://github.com/anthropics/claude-cookbooks)
- **태그**: `#Claude` `#Anthropic` `#튜토리얼` `#API`

## 상세 요약

Anthropic의 공식 Claude 개발 레시피 모음으로, 복사하여 바로 사용할 수 있는 코드 스니펫과 가이드를 제공한다.

**주요 카테고리:**
- **분류**: 텍스트/데이터 분류 기법
- **RAG**: 외부 지식으로 Claude 응답 강화 (Pinecone, Wikipedia 등)
- **요약**: 효과적 텍스트 요약 기법
- **도구 사용**: 외부 도구/함수 통합 (고객 서비스 에이전트, 계산기, SQL 쿼리)
- **멀티모달**: 이미지 처리, 차트 해석, 폼 콘텐츠 추출
- **고급 기법**: 서브 에이전트, PDF 처리, 자동 평가, JSON 모드, 콘텐츠 모더레이션, 프롬프트 캐싱

Python 중심이지만 개념은 어떤 언어에도 적용 가능하다. Claude API 키만 있으면 바로 시작할 수 있다.

## Soo에게 의미 있는 이유

Claude API를 활용한 실전 패턴의 공식 레퍼런스다. AInD 컨설팅에서 Claude 기반 솔루션을 설계할 때 이 레시피들을 기초 빌딩 블록으로 활용할 수 있다.
