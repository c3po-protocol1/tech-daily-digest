## 상세 요약

Nous Research가 만든 **자기 개선 AI 에이전트**. 경험에서 스킬을 생성하고, 사용 중 스킬을 개선하며, 세션 간 연속성을 유지하는 것이 특징이다.

**핵심 차별점 — 학습 루프:**
- 에이전트가 큐레이팅한 메모리 + 주기적 넛지로 자율적 지식 축적
- 복잡한 작업 후 **자동 스킬 생성**, 사용 중 **스킬 자기 개선**
- FTS5 세션 검색 + LLM 요약으로 세션 간 회상
- Honcho 변증법적 사용자 모델링
- [agentskills.io](https://agentskills.io) 오픈 표준 호환

**멀티플랫폼:** Telegram, Discord, Slack, WhatsApp, Signal, CLI — 단일 게이트웨이

**어디서든 실행:** $5 VPS, GPU 클러스터, Docker, SSH, Daytona(서버리스), Modal, Singularity — 6가지 터미널 백엔드

**모델 자유:** Nous Portal, OpenRouter(200+), z.ai/GLM, Kimi, MiniMax, OpenAI 등 — `hermes model`로 전환

**OpenClaw 마이그레이션:** `hermes claw migrate` 명령으로 OpenClaw에서 이전 가능

## Soo에게 의미 있는 이유

OpenClaw과 직접 비교 가능한 에이전트 프레임워크다. 특히 자기 개선 루프, agentskills.io 호환, 멀티플랫폼 게이트웨이 등은 현재 Soo가 사용하는 에이전트 시스템과 유사하면서도 다른 접근법을 취한다. 경쟁 분석 및 아이디어 차용에 유용.
