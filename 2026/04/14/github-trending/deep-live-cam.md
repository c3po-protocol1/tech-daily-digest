## 상세 요약

단일 이미지만으로 실시간 얼굴 교체 및 비디오 딥페이크를 수행하는 도구다. v2.1.2 버전으로, 지속적으로 GitHub Trending에 오르고 있다.

**주요 기능:**
- **3클릭** 라이브 딥페이크: 얼굴 선택 → 카메라 선택 → Live!
- **Mouth Mask**: 원래 입 움직임 유지하면서 얼굴만 교체
- **Face Mapping**: 여러 대상에 동시에 다른 얼굴 적용
- **영화 모드**: 영화를 원하는 얼굴로 실시간 시청
- **라이브 쇼**: 실시간 공연 및 방송

**윤리적 장치:** 부적절한 미디어(노출, 폭력적 콘텐츠, 민감한 자료) 처리를 방지하는 내장 체크 포함. 실제 인물의 얼굴 사용 시 동의 획득 및 딥페이크 라벨링 요구.

NVIDIA GPU, AMD GPU, CPU, Mac Silicon 모두 지원.

## Soo에게 의미 있는 이유

딥페이크 기술의 접근성이 극도로 낮아지고 있다는 현실을 보여준다. AI 윤리와 미디어 진위 확인이 점점 중요해지는 맥락에서, AInD 컨설팅의 "AI 리스크 관리" 섹션에 참고할 기술 사례.
