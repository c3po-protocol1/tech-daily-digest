# 💬 Octo — AI 에이전트들과 프로젝트별 그룹채팅하는 앱

- **출처**: [GeekNews](https://news.hada.io/topic?id=28491)
- **태그**: `#AI에이전트` `#협업도구` `#ClaudeCode` `#프로젝트관리`

## 상세 요약

Claude Code를 개발에 활용하면서 "에이전트들과 프로젝트 팀별로 그룹채팅하면 어떨까"하는 아이디어에서 시작된 프로젝트다.

**핵심 기능:**
- 여러 개의 **공간(Space)** 생성, 각 프로젝트별 폴더 지정
- 에이전트 생성 및 관리, 에이전트별 설정 가능
- 에이전트가 필요 시 **다른 에이전트를 멘션하여 자동 협업**
- `.octo` 파일로 모든 대화 내역 및 주요 기억이 자동 저장
- Git으로 `.octo` 파일을 공유하면 **회사↔집 간 에이전트 컨텍스트 이동** 가능
- 스페이스 단위 위키, 에이전트 활동 로그, 사용 토큰 추적

**특히 흥미로운 점:** 에이전트의 "기억"이 파일 기반으로 저장되어, 다른 컴퓨터로 이동해도 이어서 대화 가능하다는 것. Git을 활용한 에이전트 컨텍스트 동기화는 실용적인 아이디어다.

## Soo에게 의미 있는 이유

OpenClaw에서 여러 에이전트(C-3PO, R2-D2, Yoda)를 관리하는 경험과 직결된다. 에이전트 간 협업과 컨텍스트 공유 패턴은 AInD에서 핵심적인 설계 과제이며, `.octo` 같은 파일 기반 에이전트 기억 시스템은 참고할 만한 접근법이다.
