# Manifest - 개인 AI를 위한 스마트 모델 라우터

- **GitHub**: [mnfst/manifest](https://github.com/mnfst/manifest)
- **언어**: TypeScript
- **오늘의 스타**: ⭐ 174/day
- **태그**: `#AI라우팅` `#비용최적화` `#LLM` `#인프라`

## 요약

OpenClaw이나 Hermes 같은 개인 AI 에이전트를 위한 스마트 모델 라우터다. 에이전트와 LLM 프로바이더 사이에서 각 요청의 복잡도를 점수화하고, 처리 가능한 가장 저렴한 모델로 라우팅한다.

**작동 원리:**
- `manifest/auto`로 보내는 모든 요청에 대해 23차원 스코어링 알고리즘 실행 (2ms 이내)
- 4개 티어로 분류: simple, standard, complex, reasoning
- 각 티어에서 연결된 프로바이더 중 최적 모델 선택

**핵심 가치:**
- **비용 절감 최대 70%**: 간단한 질문은 빠르고 저렴한 모델로, 어려운 문제는 비싼 모델로
- **자동 폴백**: 모델 실패 시 다음 모델이 자동 처리
- **예산 제한 설정**: 지출 한도 초과 방지
- **자동 모니터링**: 토큰, 비용, 모델, 응답시간 등 자동 기록

Docker 기반 자체 호스팅 또는 클라우드 버전(app.manifest.build) 제공.

## Soo에게 의미 있는 이유

AI 에이전트 운영 비용은 핵심 관심사다. 모델 라우팅으로 비용을 70%까지 절감할 수 있다는 것은 AInD 프로젝트의 ROI를 크게 개선할 수 있는 인프라 기술이다.
