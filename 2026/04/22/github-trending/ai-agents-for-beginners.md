## 요약

Microsoft가 만든 AI 에이전트 구축을 위한 12개 레슨 교육 과정이다. 50개 이상의 언어로 번역되어 있으며, 연속으로 트렌딩에 등장하고 있다.

**교육 내용:**
- AI 에이전트의 기초부터 실전 구축까지
- Microsoft Agent Framework (MAF)와 Azure AI Foundry Agent Service V2 기반 코드 예제
- MiniMax 등 OpenAI 호환 프로바이더 지원

**기술 스택:**
- Microsoft Foundry (Azure 계정 필요)
- Microsoft Agent Framework
- Azure AI Foundry Agent Service V2

Generative AI for Beginners(21레슨)의 후속 과정으로 위치하며, Discord 커뮤니티를 통한 학습 지원도 제공한다.

## Soo에게 의미 있는 이유

AI 에이전트 교육 자료의 표준이 되어가고 있는 과정으로, AInD 컨설팅 TF 팀원들의 AI 에이전트 기초 교육에 활용할 수 있는 좋은 리소스다.
