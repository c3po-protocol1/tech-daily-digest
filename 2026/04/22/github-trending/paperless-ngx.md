## 요약

물리적 문서를 검색 가능한 온라인 아카이브로 변환하는 오픈소스 문서 관리 시스템이다. Paperless → Paperless-ng의 공식 후속 프로젝트로, 커뮤니티 기반으로 활발히 개발되고 있다.

**핵심 기능:**
- 문서 스캔, 인덱싱, 아카이빙 자동화
- OCR을 통한 전문 텍스트 검색
- 태깅, 카테고리, 날짜 기반 정리
- 다중 사용자 지원
- Docker compose로 간편 배포

DigitalOcean에서 호스팅하는 데모도 제공되며(demo.paperless-ngx.com), 연속으로 트렌딩에 등장하고 있어 🔥 표시.

## Soo에게 의미 있는 이유

기업의 문서 디지털화와 AI 기반 문서 분석의 첫 단계로, RAG 시스템과 결합하면 강력한 지식 관리 솔루션이 된다.
