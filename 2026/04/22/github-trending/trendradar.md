# TrendRadar - AI 기반 여론/트렌드 모니터링 도구

- **GitHub**: [sansan0/TrendRadar](https://github.com/sansan0/TrendRadar)
- **언어**: Python
- **오늘의 스타**: ⭐ 584/day
- **태그**: `#트렌드모니터링` `#RSS` `#AI분석` `#알림`

## 요약

30초만에 배포 가능한 AI 기반 여론 모니터링 및 핫 토픽 필터링 도구다. 다중 플랫폼의 핫 토픽을 집계하고, AI로 뉴스를 필터링·분석·번역하여 스마트 알림을 보낸다.

**주요 기능:**
- 다중 플랫폼 핫 토픽 집계 + RSS 구독
- 키워드 기반 정밀 필터링
- AI 뉴스 필터링 + AI 번역 + AI 분석 브리핑
- MCP 아키텍처 지원: AI 자연어 대화 분석, 감정 분석, 트렌드 예측
- Docker 지원, 데이터 로컬/클라우드 자체 보관

**알림 채널**: WeChat, 飞书, DingTalk, Telegram, 이메일, ntfy, Bark, Slack 등 다양한 메신저 지원

GitHub Actions, GitHub Pages, Docker 등 다양한 배포 방식을 지원하며, v6.6.1까지 활발히 개발 중이다.

## Soo에게 의미 있는 이유

기술 트렌드 모니터링은 AInD 컨설팅의 기반이다. 이 도구를 커스터마이징하여 AI/ML 관련 뉴스를 자동 모니터링하는 데 활용할 수 있다.
