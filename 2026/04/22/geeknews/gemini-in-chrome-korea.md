# 제미나이 인 크롬(Gemini in Chrome) 한국 출시

- **출처**: [Google Korea Blog](https://blog.google/intl/ko-kr/products/android-play-hardware/gemini-in-chrome-launch-korea/)
- **GeekNews**: [topic](https://news.hada.io/topic?id=28743) (6p)
- **태그**: `#Google` `#Gemini` `#Chrome` `#AI브라우저`

## 요약

구글이 2026년 4월 20일, '제미나이 인 크롬(Gemini in Chrome)'을 포함한 크롬의 최신 AI 기능을 한국에 공식 출시했다. 최신 Gemini 3.1 모델을 기반으로 하며, 데스크톱 및 iOS 환경에서 먼저 제공된다.

**주요 기능:**
1. **탭 전환 없는 웹 콘텐츠 활용**: 사이드 패널에서 현재 페이지에 대해 AI와 대화. 긴 웹 콘텐츠 요약, 시험 예상문제 생성, 레시피 변환("이 레시피를 비건으로") 등 가능
2. **구글 서비스 통합**: 지메일, 지도, 캘린더, 유튜브 등을 브라우저 내에서 직접 연동. 페이지를 벗어나지 않고 이메일 작성·전송 가능
3. **멀티탭 크로스 작업**: 여러 열린 탭의 정보를 교차 확인하고 통합. 여러 사이트 제품 비교표 자동 생성
4. **나노 바나나 2 이미지 변환**: 크롬에 직접 탑재된 이미지 변환 모델. 파일 업로드 없이 브라우저 내에서 즉시 이미지 편집

**보안 설계:**
- 프롬프트 인젝션 방어를 위한 모델 훈련
- 민감한 작업(이메일 전송, 캘린더 추가) 전 확인 요청
- 자동화된 레드팀 훈련으로 보안 강화
- 자동 업데이트로 최신 위협 대응

## Soo에게 의미 있는 이유

브라우저에 AI가 네이티브로 통합되는 것은 사용자 경험의 근본적 변화다. 한국 출시는 Soo의 AInD 컨설팅 대상 기업들이 직접 체험할 수 있다는 의미이며, "AI가 일상 업무 도구에 어떻게 스며드는가"의 구체적 사례로 활용 가능하다.
