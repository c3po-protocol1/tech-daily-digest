```
│   │   • AI가 새로 학습한 도메인 규칙 수                        │

│       • AI 자율 판단 중 인간이 수정한 비율                      │
│       • 사전에 방지한 장애/보안 이슈 수                         │
│                                                                  │
│   핵심 전환:                                                    │
│   "얼마나 많이 만들었는가" → "얼마나 좋은 판단을 했는가"      │
│                                                                  │
└──────────────────────────────────────────────────────────────────┘
```
