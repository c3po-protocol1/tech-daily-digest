## 핵심 요약

제3자 AI API 릴레이/프록시 서비스의 **보안 감사 도구**. 숨겨진 데이터 수집, 요청 로깅, 프롬프트 스니핑 등을 탐지.

많은 개발자가 비용 절감이나 지역 제한 우회를 위해 제3자 AI API 프록시를 사용하는데, 이 프록시들이 **프롬프트와 응답을 몰래 로깅**할 수 있다는 보안 위험을 경고. 이 도구는 프록시 서비스를 감사하여 데이터 누출 여부를 검증.

## Soo에게 의미 있는 이유

AI API 보안은 기업 도입의 핵심 우려 사항. 제3자 프록시 사용 시 데이터 유출 위험을 인지하고 검증하는 것이 중요.
