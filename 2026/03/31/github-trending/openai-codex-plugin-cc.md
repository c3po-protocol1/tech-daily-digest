## 핵심 요약

OpenAI가 공식 출시한 **Claude Code 플러그인** — Codex를 Claude Code 내에서 사용하여 코드 리뷰나 작업 위임이 가능. 하루 만에 699★로 폭발적 관심.

경쟁사 제품(Claude Code) 안에서 자사 모델(Codex)을 사용할 수 있게 한다는 것 자체가 업계의 **도구 상호운용성 트렌드**를 보여준다. Harness 엔지니어링의 "최적의 harness가 모델이 훈련된 harness가 아닐 수 있다"는 원칙과 일맥상통.

## Soo에게 의미 있는 이유

멀티 모델 에이전트 오케스트레이션의 실전 사례. 하나의 harness에서 여러 모델을 사용하는 패턴이 공식화되고 있다.
