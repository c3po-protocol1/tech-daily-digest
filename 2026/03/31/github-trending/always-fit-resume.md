## 핵심 요약

폰트 크기와 행간을 **자동으로 조절**하여 이력서가 항상 정확히 한 페이지에 맞도록 하는 웹 기반 이력서 빌더.

이력서에 내용을 추가하거나 수정할 때마다 레이아웃이 자동으로 조정되어, 수동으로 폰트 크기를 줄이거나 여백을 조절하는 번거로움을 해결. "한 페이지 이력서" 규칙을 지키면서도 내용 타협 없이 모든 정보를 담을 수 있음.

99★로 많은 관심. 구직 시즌과 맞물려 실용적 도구에 대한 수요 반영.

## Soo에게 의미 있는 이유

자동 레이아웃 조정 알고리즘의 작은 규모 구현 사례. AI가 문서 레이아웃을 최적화하는 것의 단순한 전단계.
