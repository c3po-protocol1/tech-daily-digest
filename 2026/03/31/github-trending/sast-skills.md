## 핵심 요약

AI 코딩 에이전트(Claude Code 등)를 **SAST(Static Application Security Testing) 스캐너**로 변환하는 에이전트 스킬 모음.

에이전트 스킬(Claude Code Skills 포맷)로 제공되어, 에이전트가 코드를 작성하는 것뿐 아니라 보안 취약점을 **구조화된 방식으로 탐지**하도록 가르친다. SQL 인젝션, XSS, 인증 우회 등 일반적인 보안 취약점 패턴에 대한 스킬을 포함.

이것은 awesome-harness-engineering의 "Constraints & Guardrails" 카테고리와 직결 — 에이전트에게 코드를 만드는 것뿐 아니라 **보안 검증도 harness 안에서 수행**하게 하는 패턴.

## Soo에게 의미 있는 이유

에이전트 스킬을 활용한 보안 자동화의 구체적 사례. AInD 컨설팅에서 "AI가 보안 검증까지 자동화할 수 있는가"에 대한 답변 참고.
