## 핵심 요약

**VICE** — 웹 애플리케이션을 위한 보안 감사 CLI 도구. 46★.

웹 애플리케이션의 보안 취약점을 명령줄에서 빠르게 스캔하고 보고서를 생성. 일반적인 보안 취약점(XSS, CSRF, SQL Injection, 보안 헤더 미설정 등)을 자동 감지.

CLI 기반이므로 CI/CD 파이프라인에 통합 가능 — 코드 배포 전 자동 보안 검사에 활용. AI 에이전트가 코드를 생성한 후 자동 보안 검증 단계에 포함시킬 수 있음.

## Soo에게 의미 있는 이유

AI가 생성한 웹앱의 보안 검증 자동화 도구. Harness의 "검증 루프"에 포함시킬 수 있는 실용적 도구.
