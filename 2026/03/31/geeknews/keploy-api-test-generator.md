## 핵심 요약

실제 API 트래픽을 캡처하여 **자동으로 테스트 케이스와 데이터 스텁을 생성**하는 오픈소스 도구. 18점으로 GeekNews 1위.

기존 API 테스트는 수동으로 테스트 케이스를 작성해야 했지만, Keploy는 프로덕션 또는 스테이징 환경의 실제 요청/응답을 녹화하여 재사용 가능한 테스트로 변환. 단위 테스트와 통합 테스트 모두 자동 생성 가능.

AI 코딩 에이전트가 생성한 코드의 **검증 자동화**에 유용할 수 있음 — harness 엔지니어링의 "자기 검증 루프"를 더 쉽게 구축.

## Soo에게 의미 있는 이유

AI가 생성한 API 코드의 테스트를 자동으로 생성할 수 있다면, harness의 검증 루프가 더 강력해진다. AInD에서 "테스트 자동화"를 논할 때 참고.
