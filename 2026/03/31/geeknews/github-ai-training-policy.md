## 핵심 요약

GitHub이 개인정보 처리방침(Privacy Statement)과 서비스 약관(Terms of Service)을 업데이트. **2026년 4월 24일부터 Copilot Free, Pro, Pro+ 사용자의 상호작용 데이터 — 입력, 출력, 코드 스니펫, 관련 컨텍스트 — 를 AI 모델 훈련에 사용.**

### 영향 범위

| 플랜 | 영향 | 조치 |
|------|------|------|
| **Copilot Free** | ⚠️ 기본 허용 | 옵트아웃 필요 |
| **Copilot Pro** | ⚠️ 기본 허용 | 옵트아웃 필요 |
| **Copilot Pro+** | ⚠️ 기본 허용 | 옵트아웃 필요 |
| **Copilot Business** | ✅ 영향 없음 | 변경 없음 |
| **Copilot Enterprise** | ✅ 영향 없음 | 변경 없음 |

### 옵트아웃 방법

`github.com/settings/copilot`에서 설정 변경 가능. **별도 설정을 하지 않으면 4월 24일부터 자동으로 AI 훈련에 데이터 사용됨.**

### 주요 변경 사항

**개인정보 처리방침:**
- AI 기능 사용 시 수집되는 데이터 범위 명확화
- 생성 AI 기능의 데이터 처리 방식 구체화
- 서비스 데이터(디바이스 정보, 액세스 로그 등) 설명 업데이트

**서비스 약관:**
- 자동화된 에이전트 사용에 대한 조건 추가
- AI 생성 콘텐츠의 지적 재산권 관련 조항 명확화

## Soo에게 의미 있는 이유

**즉시 조치 필요.** 개인 및 팀의 GitHub 설정을 확인하여 옵트아웃할지 결정해야 한다. 기업 고객은 Business/Enterprise 플랜이라 괜찮지만, 개인 개발자들은 인지하지 못한 채 코드가 훈련 데이터로 사용될 수 있다. AInD 컨설팅에서 "데이터 거버넌스"를 논할 때 중요한 사례.
