## 핵심 요약

Claude Code를 **읽는 것이 아니라 실습으로 배우는** 인터랙티브 튜토리얼 사이트. 81점으로 개발자들의 관심을 끌었다.

문서를 읽는 대신 실제로 Claude Code를 사용하면서 기능을 하나씩 배워가는 실습형 접근. 초보자부터 고급 사용자까지 단계별로 학습 가능한 구조.

## Soo에게 의미 있는 이유

AInD 교육 자료로 활용 가능. "실습으로 배우기" 방식은 컨설팅 고객사 교육에서도 효과적인 접근법이다.
