## 핵심 요약

Claude Code를 **인터랙티브하게 실습으로 배우는** 웹 사이트. 문서를 읽는 대신 실제로 해보면서 배우는 접근. 81점으로 개발자 커뮤니티에서 호응.

### 사이트 구성

- **Learning Modules** (`/learn`) — 단계별 학습 모듈. 기본부터 고급까지 순차적으로
- **Find Your Level** (`/quiz`) — 퀴즈로 현재 수준 파악 후 적절한 시작점 추천
- **Playground** (`/playground`) — 실제로 Claude Code 기능을 실험할 수 있는 환경
- **Config Builder** (`/build`) — CLAUDE.md 등 설정 파일을 대화형으로 구축하는 도구
- **Cheat Sheet** (`/reference`) — 빠른 참조용 치트 시트
- **Feature Index** (`/catalog`) — Claude Code의 모든 기능을 카탈로그로 정리

### 차별점

대부분의 AI 도구 교육이 문서 기반인 반면, 이 사이트는 **"해보면서 배우기"**를 강조. Config Builder가 특히 실용적 — CLAUDE.md를 처음부터 직접 작성하기 어려운 사람에게 대화형 가이드 제공.

## Soo에게 의미 있는 이유

AInD 컨설팅에서 **고객사 교육 방법론**의 참고 자료. 문서 기반 교육보다 인터랙티브 실습이 효과적이라는 것은 교육 설계의 핵심 원칙. Config Builder 같은 도구를 고객사 맞춤형으로 만들 수 있다면 컨설팅 가치가 높아짐.
