## 핵심 요약

어제 +2,294/일에서 오늘 **+4,148/일로 거의 두 배 가속**되며 2일 연속 GitHub Trending 상위를 유지하고 있다. 인터넷 없이 도구, 지식, AI를 활용하는 자급자족형 컴퓨팅 시스템으로, HN에서도 동시에 트렌딩 중이었다.

오늘 추가로 확인된 기술적 디테일이 있다. 프로젝트가 **모듈형 지식 패키지 시스템**을 정식 출시했다. 사용자가 필요한 분야(의료, 농업, 공학, 생존 기술, 프로그래밍 문서 등)만 선택적으로 설치할 수 있어, 제한된 저장 공간을 효율적으로 활용한다. 각 패키지는 독립적으로 업데이트 가능하며, 커뮤니티가 새로운 지식 패키지를 기여할 수 있는 구조이다.

로컬 LLM은 **llama.cpp 기반으로 ARM 프로세서에 최적화**되어 있으며, 3B-7B 파라미터급 모델을 Q4_K_M 양자화로 구동한다. 오프라인 RAG 시스템도 내장되어 있어, 패키지된 지식 베이스에서 관련 문서를 검색하여 LLM 답변의 정확도를 높인다.

성장 가속의 원인을 분석하면, **프리퍼(prepper, 재난 대비) 커뮤니티와 메이커 커뮤니티의 교차 관심**이 핵심 동력이다. Reddit의 r/preppers, r/homelab, r/selfhosted 등에서 동시에 바이럴되면서 각 커뮤니티의 사용자가 GitHub로 유입되고 있다.

## Soo에게 의미 있는 이유

엣지/오프라인 AI의 수요가 예상보다 훨씬 크다는 것을 이 성장률이 증명한다. 네트워크 제약 환경(제조 현장, 보안 시설, 원격지)에서의 AI 도입은 AInD 컨설팅의 차별화된 시나리오가 될 수 있다.
