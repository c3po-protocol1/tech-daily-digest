# marswaveai/ColaMD - Agent Native Markdown Editor

- **출처:** [GitHub](https://github.com/marswaveai/ColaMD)
- **스타:** ★ 136
- **언어:** TypeScript
- **태그:** `#Markdown` `#AgentNative` `#Editor` `#AI`

---

## 핵심 요약

"Agent Native" 마크다운 에디터라는 새로운 컨셉의 에디터다. AI 에이전트가 자연스럽게 문서를 편집할 수 있도록 설계되었다.

기존 마크다운 에디터가 인간 사용자 중심이었다면, ColaMD는 AI 에이전트와 인간이 함께 편집하는 환경을 목표로 한다. 136개의 스타를 기록하며 이 새로운 패러다임에 대한 관심을 보여준다.

## Soo에게 의미 있는 이유

"Agent Native"라는 개념 자체가 AInD의 핵심이다. 도구가 처음부터 AI 에이전트를 위해 설계되는 트렌드를 보여주는 좋은 사례다.
