# "스몰 웹(small web)"은 생각보다 훨씬 크다

- **출처**: [GeekNews](https://news.hada.io/topic?id=27591)

상업적 목적 없이 개인용 웹사이트와 블로그로 구성된 '스몰 웹'이 예상보다 활발히 운영되고 있음. Gemini 프로토콜 같은 비상업적 커뮤니티 중심 대안 웹 환경이 존재하며, 약 6,000개의 '캡슐'이 운영 중. Kagi 검색엔진의 Small Web 이니셔티브와 연결됨.

**왜 중요한가**: AI와 SEO 스팸이 범람하는 웹에서, 인간 중심의 대안적 웹 생태계가 조용히 성장하고 있다는 신호.
