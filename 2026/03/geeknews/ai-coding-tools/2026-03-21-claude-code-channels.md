# Claude Code Channels 공개 - Telegram/Discord로 작업 지시 가능

**출처:** https://news.hada.io/topic?id=27657

Anthropic이 MCP 서버를 통해 Telegram이나 Discord에서 Claude Code에 직접 작업을 지시하고 결과를 받아볼 수 있는 Channels 기능을 공개했다. 양방향 통신을 지원하여 모바일에서도 코딩 에이전트를 제어할 수 있다.

**왜 중요한가:** 코딩 에이전트의 접근성이 메신저 앱으로 확장되면서, 개발 워크플로우의 패러다임이 변화하고 있다.
