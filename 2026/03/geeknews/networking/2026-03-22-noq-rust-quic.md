# Noq: n0의 새로운 Rust 기반 QUIC 구현체

- **출처**: [GeekNews](https://news.hada.io/topic?id=27703)

n0 팀이 개발한 noq는 Rust로 작성된 범용 QUIC 구현체로, 멀티패스와 NAT 트래버설을 지원한다. 기존 iroh의 Quinn 기반 구조 한계를 해결하기 위해 독립 코드베이스로 전환하여 개발되었다.

**왜 중요한가**: QUIC은 HTTP/3의 기반이자 차세대 네트워킹의 핵심 프로토콜. NAT 트래버설과 멀티패스 지원은 P2P, 엣지 컴퓨팅 시나리오에서 중요하다.
