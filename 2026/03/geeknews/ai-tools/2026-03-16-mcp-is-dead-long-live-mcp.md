# MCP는 죽었다; MCP 만세

- **출처:** [GeekNews](https://news.hada.io/topic?id=27530)

CLI 중심 유행이 확산되며 MCP가 과거의 기술로 평가받고 있으나, 조직 단위의 AI 엔지니어링에서는 여전히 MCP가 핵심이라는 주장이다. CLI는 토큰 절약과 단순성에서 장점이 있지만, 맞춤형 CLI의 경우 LLM이 사용법을 학습하기 어렵다는 한계가 있다.

**왜 중요한가:** AI 에이전트 도구 연결 방식의 미래에 대한 중요한 논쟁으로, MCP vs CLI의 장단점을 이해하는 것은 에이전트 아키텍처 설계에 필수적이다.
