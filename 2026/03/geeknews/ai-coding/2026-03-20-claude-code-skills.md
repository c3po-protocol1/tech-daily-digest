# Claude Code를 만들며 배운 것: Skills 사용 방법

- **출처:** [GeekNews](https://news.hada.io/topic?id=27640)

Anthropic이 Claude Code에서 Skills를 실제 운용하며 축적한 노하우를 공개했다. Skills는 단순 마크다운이 아니라 스크립트, 에셋, 데이터를 포함하는 폴더 구조이며, 에이전트가 탐색하고 활용하는 방식에 대한 실전 패턴을 제시한다.

**왜 중요한가:** AI 에이전트의 확장성을 결정하는 Skills 아키텍처의 실전 운용 사례로, 에이전트 시스템 설계의 핵심 참고 자료다.
