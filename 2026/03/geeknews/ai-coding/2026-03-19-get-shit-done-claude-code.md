# Get Shit Done - Claude Code용 명세 기반 개발 시스템

- **출처:** https://news.hada.io/topic?id=27622
- **날짜:** 2026-03-19

## 요약

Claude Code에서 명세 기반 개발(spec-driven development)을 자동화하는 경량 시스템. 컨텍스트 엔지니어링과 XML 기반 프롬프트 구조화를 통해 복잡한 워크플로 없이 프로젝트를 완성하도록 지원.

## 왜 중요한가

AI 코딩 에이전트의 효과적 활용을 위한 구조화된 접근법으로, 실무에서 바로 적용 가능
