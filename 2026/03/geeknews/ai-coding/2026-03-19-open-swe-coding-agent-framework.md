# Open SWE: 사내 코딩 에이전트를 위한 오픈소스 프레임워크

- **출처:** https://news.hada.io/topic?id=27604
- **날짜:** 2026-03-19

## 요약

Stripe, Ramp, Coinbase 등 주요 기업들이 독립적으로 구축한 사내 코딩 에이전트들이 유사한 아키텍처 패턴으로 수렴하고 있으며, 이를 오픈소스로 구현한 프레임워크. Deep Agents 패턴을 기반으로 함.

## 왜 중요한가

기업용 코딩 에이전트의 아키텍처가 표준화되고 있으며, AInD 컨설팅 시 참조할 수 있는 핵심 프레임워크
