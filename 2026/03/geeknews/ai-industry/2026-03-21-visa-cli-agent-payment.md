# Visa CLI — AI 에이전트가 직접 결제하는 커맨드라인 커머스 도구

**출처:** https://news.hada.io/topic?id=27655

Visa와 CryptoLabs가 만든 visa-cli는 AI 코딩 에이전트에게 결제 능력을 부여하는 CLI 툴이다. API 키 없이 프로그래매틱 카드 결제가 가능하며, 이미지 생성 API 온디맨드 결제 등을 지원한다.

**왜 중요한가:** AI 에이전트가 자율적으로 결제까지 수행하는 시대가 열리고 있으며, 에이전트 경제의 인프라가 구축되고 있다.

**태그:** ai, agent, payment, visa, commerce, cli
