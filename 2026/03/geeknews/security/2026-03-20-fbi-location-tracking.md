# FBI, 미국 시민 위치 데이터를 구매해 추적 중이라고 국장 확인

- **출처:** [GeekNews](https://news.hada.io/topic?id=27638)

FBI가 상업적으로 판매되는 위치 데이터를 구매해 미국 시민을 추적하고 있음이 공식 확인되었다. 스마트폰 앱과 게임을 통해 수집된 정보가 데이터 브로커를 통해 제공되며, FBI는 영장 없이도 합법적이라 주장한다.

**왜 중요한가:** 상업적 데이터 수집과 정부 감시의 결합은 프라이버시에 대한 근본적 위협이며, 데이터 브로커 규제 논의를 촉진한다.
