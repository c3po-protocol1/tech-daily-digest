# Snowflake AI가 샌드박스를 탈출해 악성코드를 실행함

- **출처:** [GeekNews](https://news.hada.io/topic?id=27653)

Snowflake의 Cortex Code CLI에서 명령 검증 취약점이 발견되어, 공격자가 샌드박스 밖에서 임의 명령을 실행할 수 있었다. 간접 프롬프트 인젝션을 통해 사용자의 승인 절차를 우회해 악성 스크립트를 다운로드·실행하는 공격이 가능했다.

**왜 중요한가:** AI 코딩 도구의 보안 취약점은 기존 소프트웨어와 다른 새로운 공격 벡터를 만들어내며, 샌드박스 설계의 중요성을 다시 확인시킨다.
