# Bing 검색 "카카오톡" 상위 3개 결과가 전부 중국발 피싱

- **출처**: [GeekNews](https://news.hada.io/topic?id=27712)

Bing에서 "카카오톡" 검색 시 상위 3개 결과(apps-kakaocorp[.]com, apps-kakaotalk[.]com, pc-kakaotalk[.]com)가 전부 중국발 피싱 사이트로 확인되었다. 인프라 및 코드를 분석한 결과, 체계적으로 구축된 피싱 네트워크임이 드러났다.

**왜 중요한가**: 한국 사용자를 직접 겨냥한 피싱 공격이 주요 검색 엔진의 상위 결과를 점령했다는 사실은 심각한 보안 위협이다. Bing의 검색 품질 관리에 대한 의문도 제기된다.
