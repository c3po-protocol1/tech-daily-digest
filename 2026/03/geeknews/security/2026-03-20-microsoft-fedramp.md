# 연방 사이버 보안 전문가들이 의심에도 불구하고 Microsoft 클라우드 서비스를 승인

- **출처:** [GeekNews](https://news.hada.io/topic?id=27633)

미국 정부의 FedRAMP 프로그램이 보안 우려에도 불구하고 Microsoft의 GCC High 서비스를 승인했다. 내부 평가 보고서에서는 "전체 보안 상태를 평가할 자신이 없다"고 명시되었으며, 일부 검토자는 시스템에 우려를 표했다.

**왜 중요한가:** 정부 클라우드 보안 인증 프로세스의 신뢰성에 의문을 제기하며, 공공 부문의 클라우드 보안 거버넌스 문제를 부각시킨다.
