# Cockpit: 서버용 웹 기반 그래픽 인터페이스

- **출처**: [GeekNews](https://news.hada.io/topic?id=27713)

웹 브라우저에서 실행되는 Linux 서버 관리용 그래픽 인터페이스로, 실제 시스템 세션과 직접 상호작용하는 경량형 관리자 도구다. 시스템 자원, 네트워크, 스토리지, 로그 등을 시각적으로 제어할 수 있는 대화형 관리 환경을 제공한다.

**왜 중요한가**: CLI에 익숙하지 않은 팀원도 서버를 관리할 수 있게 하는 접근성 도구. DevOps 문화 확산에서 진입 장벽을 낮추는 역할을 한다.
