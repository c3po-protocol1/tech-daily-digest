# Cluefin-Dure: 증권사/DART API를 활용한 국내주식 멀티 AI 에이전트

- **출처**: [GeekNews](https://news.hada.io/topic?id=27701)

증권사와 DART RestAPI를 AI 에이전트가 사용할 수 있는 도구로 연결한 투자 리서치 멀티 에이전트 프로젝트다. openapi 패키지를 통해 한국 증권 데이터에 접근하며, 여러 에이전트가 협력하여 투자 분석을 수행한다.

**왜 중요한가**: 한국 금융 데이터와 AI 에이전트의 실전 결합 사례. 국내 API를 활용한 도메인 특화 멀티 에이전트 시스템의 좋은 참고 프로젝트다.
