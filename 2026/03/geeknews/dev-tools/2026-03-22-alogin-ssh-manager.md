# alogin: TUI, 멀티홉 게이트웨이, 암호화 저장소를 갖춘 SSH 접속 관리 도구

- **출처**: [GeekNews](https://news.hada.io/topic?id=27716)

20년간 Bash + Expect 스크립트로 유지해온 개인용 SSH 자동화 도구를 Go로 완전히 재작성하여 오픈소스로 공개한 프로젝트다. 수십 대 서버의 사용자, 포트, 게이트웨이 경로, 자격증명을 효율적으로 관리할 수 있는 TUI 인터페이스를 제공한다.

**왜 중요한가**: 서버 접속 관리는 DevOps의 기본이면서도 체계화가 어려운 영역. 20년간의 실전 경험이 녹아든 도구라는 점이 차별점이다.
