# Django에는 토큰이 아니라 시간과 돈을 투자하라

- **출처**: [GeekNews](https://news.hada.io/topic?id=27595)

LLM을 이용해 Django 티켓을 처리하는 방식은 도움이 되지 않으며, Django Software Foundation에 직접 기부하는 편이 더 유익하다는 주장. Django는 품질 기준이 매우 높고 장기적 안정성을 중시하는 프로젝트로, 단순 코드 생성 이상의 깊은 이해가 필요함.

**왜 중요한가**: AI 코드 생성의 한계를 인정하고, 오픈소스 프로젝트에 의미 있게 기여하는 방법에 대한 중요한 논의.
