# Linear MCP 서버를 CLI와 Agent Skill로 다시 만들었습니다

- **출처:** [GeekNews](https://news.hada.io/topic?id=27637)

Python CLI 하나로 Linear API 전체를 감싸는 Agent Skill을 구현했다. .env에 API 키만 넣으면 되고, Agent가 SKILL.md를 읽고 알아서 실행한다. CLI 출력도 AI Agent만 읽는다고 가정하고 최적화했다.

**왜 중요한가:** MCP 서버를 CLI + Agent Skill 패턴으로 대체하는 실용적인 접근법으로, 에이전트 도구 설계의 새로운 패러다임을 제시한다.
