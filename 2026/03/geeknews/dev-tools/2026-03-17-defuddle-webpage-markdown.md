# defuddle - 어떤 웹페이지든 Markdown으로 추출

- **출처**: [https://news.hada.io/topic?id=27540](https://news.hada.io/topic?id=27540)

웹페이지에서 광고, 사이드바, 헤더 등을 제거하고 핵심 본문만 마크다운으로 추출하는 콘텐츠 파서다. Obsidian Web Clipper용으로 개발되었으나 범용적으로 사용 가능하며, 유튜브 영상의 화자별 대화 분리 기능도 지원한다.

**왜 중요한가:** LLM에 웹 콘텐츠를 입력할 때 정제된 텍스트 추출이 필수적이며, RAG 파이프라인 구축 시 유용한 도구다.

**태그:** 마크다운, 웹스크래핑, 콘텐츠파싱, Obsidian, RAG
