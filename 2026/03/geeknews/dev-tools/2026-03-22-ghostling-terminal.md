# Ghostling: 최소 기능 터미널 에뮬레이터

- **출처**: [GeekNews](https://news.hada.io/topic?id=27699)

libghostty C API를 기반으로 한 최소 기능의 터미널 데모로, 단일 C 파일과 Raylib 렌더링을 사용해 동작한다. libghostty-vt의 기능을 검증하기 위한 구조로, GPU 렌더링 대신 2D 그래픽 기반 터미널 구현을 보여준다.

**왜 중요한가**: Ghostty의 핵심 라이브러리가 독립적으로 사용 가능함을 보여주는 데모. 터미널 에뮬레이터 생태계의 모듈화 추세를 반영한다.
