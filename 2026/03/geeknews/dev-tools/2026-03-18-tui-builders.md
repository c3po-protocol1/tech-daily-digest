# tui.builders – 터미널 UI의 Figma, Rust 코드 생성까지

- **출처**: [GeekNews](https://news.hada.io/topic?id=27581)

Rust의 ratatui 기반 TUI(터미널 UI)를 Figma처럼 시각적으로 디자인하고 Rust 코드를 자동 생성하는 도구. CLI/TUI 개발의 진입장벽을 낮추며, 단일 바이너리 배포의 장점을 유지하면서 개발 경험을 개선.

**왜 중요한가**: 터미널 기반 도구 개발의 DX를 혁신적으로 개선하는 접근으로, Rust TUI 생태계의 성장을 촉진.
