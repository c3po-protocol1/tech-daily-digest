# 풀 리퀘스트는 죽었다, 풀 리퀘스트 만세

- **출처**: [GeekNews](https://news.hada.io/topic?id=27700)

AI가 코드를 작성하는 시대에 풀 리퀘스트(PR) 리뷰 방식이 근본적으로 바뀌어야 한다는 주장이다. 리뷰어가 "왜 이렇게 했나?"라고 질문하면 개발자가 AI에게 다시 물어 답변을 붙여넣는 구조로, 인간이 중개인(middleman)이 되는 비효율이 발생하고 있다.

**왜 중요한가**: AI 시대의 코드 리뷰 프로세스 재설계는 AInD의 핵심 주제다. 기존 개발 워크플로우가 AI와 어떻게 공존해야 하는지에 대한 중요한 논의다.
