# Tabularis: 개발자용 크로스플랫폼 DB 관리 앱

- **출처**: [GeekNews](https://news.hada.io/topic?id=27691)

Tauri+React로 제작되어 Windows/macOS/Linux를 지원하는 데이터베이스 관리 도구다. MySQL/MariaDB, PostgreSQL, SQLite를 지원하며, 다중 스키마 및 다중 데이터베이스 연결 기능을 제공한다. 빠른 성능, 보안성, 미려한 UI를 중점으로 설계되었다.

**왜 중요한가**: Tauri 기반 데스크톱 앱의 실용적 사례. Electron의 대안으로서 Tauri 생태계가 성숙해지고 있음을 보여준다.
