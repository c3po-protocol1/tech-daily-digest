# Hammerspoon - Lua로 구현된 강력한 macOS 데스크톱 자동화 도구

- **출처:** [GeekNews](https://news.hada.io/topic?id=27511)

macOS 환경을 Lua 스크립트로 제어할 수 있는 오픈소스 자동화 도구다. 운영체제와 스크립팅 엔진을 연결하는 브리지 역할을 하며, 창 관리, 입력 제어 등 복잡한 작업을 스크립트로 자동화할 수 있다.

**왜 중요한가:** macOS 파워유저와 개발자에게 매우 유용한 자동화 도구이며, AI 에이전트의 데스크톱 자동화와도 연계 가능성이 있다.
