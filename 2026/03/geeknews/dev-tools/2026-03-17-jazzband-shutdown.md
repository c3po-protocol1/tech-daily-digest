# Jazzband - 파이썬 오픈소스 커뮤니티 운영 종료

- **출처**: [https://news.hada.io/topic?id=27564](https://news.hada.io/topic?id=27564)

pip-tools, tablib 등 주요 Python 패키지를 공동 관리하던 Jazzband 오픈소스 커뮤니티가 운영을 종료했다. 오픈소스 유지보수의 지속가능성 문제와 커뮤니티 거버넌스의 한계를 보여주는 사례다.

**왜 중요한가:** 오픈소스 프로젝트의 지속가능성은 기업의 기술 스택 선택 시 반드시 고려해야 할 리스크 요소다.

**태그:** Python, 오픈소스, 커뮤니티, 지속가능성, 거버넌스
