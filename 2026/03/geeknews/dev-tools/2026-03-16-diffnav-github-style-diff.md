# diffnav - GitHub 스타일의 Git diff pager

- **출처:** [GeekNews](https://news.hada.io/topic?id=27515)

delta의 렌더링 기능을 기반으로 GitHub의 파일 트리 UI를 터미널 환경에서 재현하는 도구다. 파일 트리와 diff 내용을 동시에 표시해 대규모 코드 변경 내역을 빠르게 시각적으로 탐색할 수 있다.

**왜 중요한가:** 터미널 기반 개발 워크플로우에서 코드 리뷰 경험을 크게 개선할 수 있는 실용적인 도구다.
