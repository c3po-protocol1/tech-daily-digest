# 지음(ziium) - 한국어 문장 구조로 코드를 쓰는 프로그래밍 언어

- **출처**: [GeekNews](https://news.hada.io/topic?id=27589)

한국어 조사와 서술어를 문법 요소로 활용하는 실험적 프로그래밍 언어. 기존 언어의 키워드를 한글로 번역하는 방식이 아니라, 한국어 화자가 읽고 쓰는 흐름에 가까운 표면 문법을 목표로 함. Rust로 구현, WebAssembly 브라우저 플레이그라운드 포함.

**왜 중요한가**: 프로그래밍 언어 설계에 자연어 문법을 적용하는 독창적 실험으로, 코딩 교육의 새로운 접근법 제시.
