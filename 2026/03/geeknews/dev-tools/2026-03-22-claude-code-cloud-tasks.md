# Claude Code, 클라우드 기반 예약 작업 기능 추가

- **출처**: [GeekNews](https://news.hada.io/topic?id=27690)

로컬 머신에서 클로드 코드를 실행하지 않아도 설정한 일정에 따라 자동으로 작업을 수행할 수 있는 기능이 추가되었다. 레포지토리, 스케줄, 프롬프트를 지정하면 클라우드 인프라에서 Claude가 해당 작업을 반복 실행하며, PR 리뷰 자동화나 이슈 기반 자동 구현이 가능하다.

**왜 중요한가**: AI 코딩 에이전트가 "대화형"에서 "자율 배치형"으로 진화하는 전환점. CI/CD 파이프라인에 AI가 네이티브로 통합되는 미래의 시작이다.
