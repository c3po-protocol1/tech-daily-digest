# 로컬파일을 링크로 즉시 공유하는 MCP 서버

- **출처**: [GeekNews](https://news.hada.io/topic?id=27592)

AI 에이전트에게 파일 공유 작업을 시키거나 메신저로 파일을 보내달라고 할 때 사용할 수 있는 MCP 서버. 구글 드라이브나 이메일 대비 대용량 파일도 업로드 대기 없이 즉시 링크로 공유 가능.

**왜 중요한가**: MCP 프로토콜의 실용적 활용 사례로, AI 에이전트의 파일 조작 능력을 확장하는 도구.
