# oh-my-agent: 실무용 멀티 AI IDE 에이전트 하네스

- **출처**: [https://news.hada.io/topic?id=27560](https://news.hada.io/topic?id=27560)

AI 에이전트가 범위를 벗어나거나 실수를 반복하는 문제를 해결하기 위한 구조적 프레임워크다. AGENTS.md와 Skills 같은 접근을 넘어, 실무 환경에서 여러 AI IDE 에이전트를 체계적으로 관리하고 가이드하는 하네스를 제공한다.

**왜 중요한가:** AI 에이전트 활용의 실무 구조화 문제를 직접 다루며, 에이전트 관리 패턴의 진화를 보여준다.

**태그:** AI에이전트, IDE, 하네스, 개발도구, 구조화
