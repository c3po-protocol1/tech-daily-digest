# macOS 26에서 .internal 등 커스텀 DNS 설정이 작동하지 않음

**출처:** https://news.hada.io/topic?id=27683

macOS 26.3.1에서 /etc/resolver/ 기반 도메인별 DNS 설정이 비표준 TLD에 대해 무효화되는 문제가 발생했다. mDNSResponder가 커스텀 TLD 요청을 mDNS로만 처리하면서 기존 로컬 개발 환경이 중단되고 있다.

**왜 중요한가:** 로컬 개발 환경에 직접적인 영향을 미치는 OS 수준 변경으로, macOS를 사용하는 개발자는 즉각적인 대응이 필요하다.

**태그:** macos, dns, networking, development, breaking-change
