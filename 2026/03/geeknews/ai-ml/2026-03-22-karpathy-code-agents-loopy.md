# Andrej Karpathy가 말하는 코드 에이전트, AutoResearch, 그리고 AI의 루피(Loopy) 시대

- **출처**: [GeekNews](https://news.hada.io/topic?id=27706)

AI 코드 에이전트의 등장으로 소프트웨어 개발 방식이 근본적으로 변화했으며, 2024년 12월을 기점으로 직접 코딩 비중이 80%에서 거의 0%로 급감한 체감적 전환이 발생했다. 개인이 여러 에이전트를 병렬로 운영하며 기능 단위로 작업을 배분하는 방식이 새로운 개발 패러다임으로 자리잡고 있다.

**왜 중요한가**: AI 분야 최고 권위자 중 한 명인 Karpathy의 실제 경험담. AInD 컨설팅 준비에 있어 "에이전트 병렬 운영" 패턴은 핵심 트렌드다.
