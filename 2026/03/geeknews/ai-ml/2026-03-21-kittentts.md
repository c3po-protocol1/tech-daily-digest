# KittenTTS - 25MB 이하의 SOTA 음성 합성(TTS) 모델

**출처:** https://news.hada.io/topic?id=27680

ONNX 기반의 경량 텍스트-음성 변환(TTS) 라이브러리로, CPU만으로 고품질 음성 합성을 수행한다. 모델 크기가 15M~80M 파라미터(25~80MB)로, GPU 없이도 효율적 실행이 가능하며 8가지 내장 음성을 제공한다.

**왜 중요한가:** 경량 TTS 모델은 엣지 디바이스에서의 음성 AI 구현을 가능하게 하며, 에이전트 인터페이스의 핵심 요소다.

**태그:** ai, tts, voice, lightweight, onnx, edge-ai
