# Cursor Composer 2는 Kimi K2.5에 RL을 적용한 모델로 확인

- **출처**: [GeekNews](https://news.hada.io/topic?id=27702)

OpenAI base URL 조작을 통해 요청 경로에 kimi-k2p5-rl이 포함된 것이 발견되며, Cursor의 Composer 2가 Kimi K2.5 + 강화학습(RL) 기반임이 드러났다. 이전 버전에서는 차단되던 요청이 Composer 2에서는 노출되었고, 이후 즉시 차단 패치가 이루어졌다.

**왜 중요한가**: AI 코딩 도구 시장의 이면을 보여주는 발견. Cursor가 자체 모델이 아닌 외부 모델(Kimi K2.5)에 RL을 적용하여 사용한다는 사실은 AI 도구 시장의 생태계 구조를 이해하는 데 중요하다.
