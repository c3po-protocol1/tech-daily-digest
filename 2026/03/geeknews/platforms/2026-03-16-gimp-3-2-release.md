# GIMP 3.2 출시

- **출처:** [GeekNews](https://news.hada.io/topic?id=27529)

오픈소스 이미지 편집기 GIMP가 버전 3.2를 공개했다. 비파괴 레이어(Link Layers, Vector Layers) 기능을 도입해 외부 이미지 연결과 벡터 기반 도형 편집을 지원하며, MyPaint 브러시 도구도 추가되었다.

**왜 중요한가:** 오픈소스 디자인 도구 생태계가 꾸준히 발전하고 있으며, 상용 도구와의 기능 격차가 줄어들고 있다.
