# Remodex - Codex를 위한 리모트 컨트롤

- **출처**: [https://news.hada.io/topic?id=27545](https://news.hada.io/topic?id=27545)

OpenAI Codex를 아이폰에서 원격 제어할 수 있는 오픈소스 브리지 앱이다. Mac에서 Codex 런타임을 실행하고, WebSocket 릴레이로 E2E 암호화 연결을 지원하며, 턴 종료 시 인앱 알림을 제공한다.

**왜 중요한가:** AI 코딩 에이전트의 모바일 제어라는 새로운 워크플로우를 제시하며, 에이전트 사용 패턴의 다양화를 보여준다.

**태그:** Codex, 원격제어, 모바일, 오픈소스, 에이전트
