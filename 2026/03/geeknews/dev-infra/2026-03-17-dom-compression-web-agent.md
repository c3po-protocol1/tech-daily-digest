# 옴니모달 없이 DOM 압축으로 웹을 탐색하는 AI 에이전트 라이브러리

- **출처**: [https://news.hada.io/topic?id=27556](https://news.hada.io/topic?id=27556)

비전 모델 없이 CDP(Chrome DevTools Protocol)로 받은 DOM을 LLM이 이해할 수 있도록 최소한으로 압축·포매팅하여 MCP로 제공하는 웹 서핑 에이전트 라이브러리다. 옴니모달 의존을 제거하여 비용과 불확실성을 줄였다.

**왜 중요한가:** 텍스트 기반 웹 에이전트 접근법으로, 비전 모델 없이도 효과적인 웹 자동화가 가능함을 보여준다.

**태그:** 웹에이전트, DOM, MCP, 자동화, LLM
