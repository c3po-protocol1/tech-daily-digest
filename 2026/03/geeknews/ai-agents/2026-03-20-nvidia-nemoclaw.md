# Nvidia NemoClaw

- **출처:** [GeekNews](https://news.hada.io/topic?id=27648)

OpenClaw 에이전트를 안전하게 설치·실행하기 위한 NVIDIA의 오픈소스 플러그인이다. OpenShell 런타임을 기반으로 한 보안 샌드박스 환경을 제공하며, 모든 네트워크 요청과 파일 접근을 제어한다.

**왜 중요한가:** AI 에이전트의 보안 실행 환경은 에이전트가 프로덕션에 배포되면서 점점 더 중요해지고 있으며, NVIDIA의 참여는 생태계 성숙을 보여준다.
