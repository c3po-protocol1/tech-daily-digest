# 무료 SVG 배경 및 패턴 | SVG Backgrounds

- **출처:** [GeekNews](https://news.hada.io/topic?id=27650)

48종의 가벼운 SVG 배경과 패턴을 제공하는 무료 리소스다. 복사·붙여넣기만으로 웹 프로젝트에 적용 가능하며, 색상, 크기, 블렌드 모드 등을 자유롭게 수정할 수 있다.

**왜 중요한가:** 개발자가 디자인 리소스 없이도 프로페셔널한 UI를 구성할 수 있게 해주는 실용적인 도구다.
