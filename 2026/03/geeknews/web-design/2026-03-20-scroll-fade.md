# 스크롤 페이드여, 사라져라

- **출처:** [GeekNews](https://news.hada.io/topic?id=27645)

웹사이트에서 스크롤 시 요소가 서서히 나타나는 '스크롤 페이드' 효과를 비판하는 글이다. 대부분의 구현이 과도한 전환 효과와 부적절한 타이밍으로 사용자 경험을 해치며, 개발자에게도 불필요한 부담을 준다고 지적한다.

**왜 중요한가:** 웹 UX 트렌드에 대한 비판적 시각으로, 디자인 결정에서 '기술적으로 가능한 것'과 '실제로 좋은 것'을 구분하는 안목을 길러준다.
