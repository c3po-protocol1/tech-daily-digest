# 오픈 소스 보안 작업은 '특별'하지 않다

- **출처:** https://news.hada.io/topic?id=27613
- **날짜:** 2026-03-19

## 요약

보안 업무를 특수한 영역으로 치부하는 '보안 예외주의'를 비판하며, 일반적인 엔지니어링 범주 내에서 다뤄야 함을 강조. 보안 사고를 시스템의 가용성 문제로 통합하여 리스크 관리할 것을 제안.

## 왜 중요한가

보안을 별도 영역이 아닌 소프트웨어 엔지니어링의 일부로 보는 관점 전환이 필요한 시점
