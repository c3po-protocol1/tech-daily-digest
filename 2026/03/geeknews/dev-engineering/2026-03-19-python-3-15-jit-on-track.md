# Python 3.15의 JIT이 다시 정상 궤도에 오르다

- **출처:** https://news.hada.io/topic?id=27618
- **날짜:** 2026-03-19

## 요약

CPython의 JIT 컴파일러가 macOS AArch64에서 11~12%, x86_64 Linux에서 5~6% 빠른 성능을 달성하며 목표를 조기 달성. 3.13~3.14 버전에서 인터프리터보다 느렸던 JIT이 커뮤니티 개발로 크게 향상됨.

## 왜 중요한가

Python의 성능 개선이 AI/ML 워크로드에 직접적으로 영향을 미치며, JIT의 성숙이 Python 생태계의 경쟁력을 강화
