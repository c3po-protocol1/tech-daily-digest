# Slug 알고리듬 10주년

- **출처:** https://news.hada.io/topic?id=27610
- **날짜:** 2026-03-19

## 요약

GPU에서 Bézier 곡선 기반 폰트 렌더링을 수행하는 Slug 알고리듬이 10주년을 맞음. 텍스처 없이 직접 벡터 데이터를 렌더링하며, 게임·CAD·영상 편집·의료 장비 등 다양한 산업에서 사용 중.

## 왜 중요한가

그래픽스 렌더링 기술의 중요한 마일스톤으로, GPU 기반 텍스트 렌더링의 표준이 된 알고리듬
