# 코드 작성 속도가 문제라고 생각했다면 더 큰 문제가 있다

- **출처:** https://news.hada.io/topic?id=27624
- **날짜:** 2026-03-19

## 요약

AI 코딩 도구가 코드 작성 속도를 높이지만, 실제 병목은 조직의 비효율적 프로세스에 있다는 분석. 코드 생산량을 늘리면 리뷰 대기, 배포 지연, 불명확한 요구사항 등 비개발 단계의 정체가 오히려 심화됨.

## 왜 중요한가

AInD 컨설팅에서 핵심 메시지 — AI 도구 도입만으로는 생산성이 향상되지 않으며, 조직 프로세스 개선이 선행되어야 함
