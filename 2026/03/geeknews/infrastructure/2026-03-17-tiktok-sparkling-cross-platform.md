# Sparkling - TikTok의 크로스 플랫폼 인프라

- **출처**: [https://news.hada.io/topic?id=27548](https://news.hada.io/topic?id=27548)

TikTok이 자사 Lynx 엔진 위에서 앱을 프로덕션 규모로 운용하기 위해 구축한 인프라 계층 Sparkling을 오픈소스로 공개했다. React Native에 Expo가 필요하듯, Lynx 엔진의 프로덕션 운용을 위한 스캐폴딩, 빌드, 네이티브 브릿지를 제공한다.

**왜 중요한가:** 대규모 앱의 크로스 플랫폼 인프라 전략을 엿볼 수 있으며, React Native/Expo 생태계와의 비교 관점에서 흥미롭다.

**태그:** TikTok, 크로스플랫폼, Lynx, 오픈소스, 모바일
