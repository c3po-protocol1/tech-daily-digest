# OpenGenerativeUI - AI 기반 생성형 UI 프레임워크 오픈소스

- **출처**: [GeekNews](https://news.hada.io/topic?id=27570)

Claude가 출시한 인터랙티브 시각 자료 생성 기능을 오픈소스로 재현한 프레임워크. 단순 텍스트 응답이 아닌 차트, 다이어그램, 알고리즘 시각화 등을 AI가 자동 생성하는 Generative UI 구현.

**왜 중요한가**: AI 응답이 텍스트를 넘어 인터랙티브 UI로 진화하는 트렌드의 오픈소스 구현체.
