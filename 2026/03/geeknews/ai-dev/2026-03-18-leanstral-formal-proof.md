# Leanstral: 형식 증명 엔지니어링을 위한 오픈소스 에이전트

- **출처**: [GeekNews](https://news.hada.io/topic?id=27586)

Lean 4용으로 설계된 최초의 오픈소스 코드 에이전트로, 형식 증명(formal proof)을 자동화해 인간 검증 부담을 줄이는 것을 목표. Apache 2.0 라이선스로 모델 가중치를 공개하며, Mistral 환경과 무료 API 액세스를 제공.

**왜 중요한가**: 형식 검증의 자동화는 고신뢰성 소프트웨어 개발의 핵심 과제이며, AI가 이 영역에 진입하는 의미 있는 시도.
