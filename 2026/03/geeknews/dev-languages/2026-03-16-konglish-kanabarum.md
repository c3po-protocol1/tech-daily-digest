# Konglish/Kanabarum – 한국어 외래어 발음 변환 라이브러리

- **출처:** [GeekNews](https://news.hada.io/topic?id=27518)

영어→콩글리시, 일본어→콩패니즈 발음 변환 라이브러리다. 학술적 정합성보다 '한국인이 일상적으로 쓰고 읽는 표기'에 최적화된 오픈소스 프로젝트로, "thank you → 땡큐", "かわいい → 카와이" 같은 변환을 지원한다.

**왜 중요한가:** 한국어 자연어 처리의 독특한 영역으로, 검색이나 음성 인식에서 외래어 처리 문제를 해결하는 실용적인 접근이다.
