# Ubuntu 26.04, 46년간의 무음 sudo 비밀번호 입력을 종료

- **출처**: [pbxscience.com](https://pbxscience.com/ubuntu-26-04-ends-46-years-of-silent-sudo-passwords/)
- **HN 점수**: 290 | 댓글 296개

Ubuntu 26.04에서 sudo 비밀번호 입력 시 별표(*) 피드백을 표시하는 변경이 이루어진다. 1980년 이래 46년간 유지되었던 "타이핑해도 아무것도 보이지 않는" 동작이 바뀌는 것이다. 296개의 댓글이 달릴 만큼 개발자들 사이에서 뜨거운 논쟁을 불러일으켰다.

**왜 중요한가**: 보안과 사용성의 오래된 트레이드오프에 대한 재고. 어깨너머 공격(shoulder surfing) 방지와 사용자 경험 사이의 균형에 대한 논쟁은 보안 UX 설계의 핵심 주제다.
