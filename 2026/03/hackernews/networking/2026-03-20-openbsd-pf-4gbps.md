# OpenBSD: PF 큐가 4Gbps 장벽을 돌파

- **출처:** [Undeadly](https://undeadly.org/cgi?action=article;sid=20260319125859)
- **HN 점수:** 170 | 댓글: 53

OpenBSD의 PF(Packet Filter) 방화벽 큐 시스템이 4Gbps 성능 장벽을 넘어섰다. 네트워크 스택의 꾸준한 최적화 작업의 결과로, BSD 기반 방화벽의 실용성이 크게 향상되었다.

**왜 중요한가:** 오픈소스 방화벽의 성능 향상은 보안과 네트워크 인프라 설계에서 상용 솔루션 대비 경쟁력을 높인다.
