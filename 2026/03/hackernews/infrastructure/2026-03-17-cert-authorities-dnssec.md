# 인증기관, 오늘부터 DNSSEC 검증 시작

- **출처**: [https://www.grepular.com/Cert_Authorities_Check_for_DNSSEC_From_Today](https://www.grepular.com/Cert_Authorities_Check_for_DNSSEC_From_Today)

SSL/TLS 인증서 발급 시 인증기관(CA)이 DNSSEC 서명을 검증하기 시작했다. 도메인 소유권 확인 과정에서 DNS 응답의 무결성을 보장하여, DNS 스푸핑을 통한 부정 인증서 발급을 방지하는 보안 강화 조치다.

**왜 중요한가:** 웹 보안 인프라의 중요한 변화로, 도메인 관리자와 DevOps 팀이 DNSSEC 설정을 점검해야 할 시점이다.

**태그:** DNSSEC, SSL, 보안, 인증서, DNS
