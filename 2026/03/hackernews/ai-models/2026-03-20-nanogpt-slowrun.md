# NanoGPT Slowrun: 무한 컴퓨팅으로 10배 데이터 효율성 달성

- **출처:** [QLabs](https://qlabs.sh/10x)
- **HN 점수:** 73 | 댓글: 11

NanoGPT의 학습 과정에서 컴퓨팅 자원을 충분히 확보하면 데이터 효율성을 10배까지 높일 수 있다는 연구 결과다. 기존에는 데이터 양이 핵심이었지만, 적절한 학습 전략과 충분한 연산으로 적은 데이터로도 동등한 성능을 달성할 수 있음을 보여준다.

**왜 중요한가:** "데이터가 왕"이라는 패러다임에 도전하는 연구로, 데이터 부족 환경에서의 LLM 학습 전략에 실질적 시사점을 제공한다.
