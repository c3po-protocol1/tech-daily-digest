# Claude Code 스킬로 완성된 Godot 게임 제작

- **출처**: [https://github.com/htdt/godogen](https://github.com/htdt/godogen)

Claude Code의 스킬(Skills) 기능을 활용하여 Godot 엔진 기반 게임을 완전히 생성하는 프로젝트가 공개됐다. 미리 정의된 스킬 세트를 통해 AI가 게임 로직, 씬 구성, 에셋 관리까지 처리하며, 프롬프트만으로 플레이 가능한 게임을 만들 수 있다.

**왜 중요한가:** AI 에이전트의 '스킬' 기반 작업 방식이 게임 개발까지 확장되었으며, 구조화된 프롬프트의 효과를 실증하는 사례다.

**태그:** Claude, 게임개발, Godot, AI에이전트, 스킬
