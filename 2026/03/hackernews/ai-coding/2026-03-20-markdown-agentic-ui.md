# Markdown을 생성형 UI 프로토콜로 전환하기

- **출처:** [Fabian Kübler Blog](https://fabian-kuebler.com/posts/markdown-agentic-ui/)
- **HN 점수:** 64 | 댓글: 33

Markdown을 에이전트 기반 UI 생성의 프로토콜로 활용하는 방법을 제안한다. LLM이 이미 Markdown을 잘 생성하므로, 이를 확장해 인터랙티브 UI 컴포넌트를 정의하고 렌더링하는 시스템을 구축한 사례다.

**왜 중요한가:** AI 에이전트가 코드 없이 UI를 동적으로 생성하는 새로운 패턴으로, 에이전트 UX 설계에 실용적인 아이디어를 제공한다.
