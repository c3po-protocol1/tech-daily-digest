# AI가 코드베이스를 변화시키는 방식에 의도적이어야 한다

- **출처:** [AI Code](https://aicode.swerdlow.dev)
- **HN 점수:** 27 | 댓글: 14

AI 코딩 도구가 코드베이스에 미치는 영향을 의도적으로 관리해야 한다는 주장이다. AI가 생성한 코드의 일관성, 아키텍처 패턴 준수, 기술 부채 관리 등에 대한 실질적인 가이드라인을 제시한다.

**왜 중요한가:** AI 코딩 도구 도입이 확산되면서, 코드 품질과 아키텍처 일관성을 유지하기 위한 전략적 접근이 필수적이다.
