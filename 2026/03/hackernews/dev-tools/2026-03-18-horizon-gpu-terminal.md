# Horizon – Rust로 만든 GPU 가속 무한 캔버스 터미널

- **출처**: [github.com/peters/horizon](https://github.com/peters/horizon)
- **HN Score**: 37

GPU 가속을 활용한 무한 캔버스 방식의 터미널 에뮬레이터. Rust로 구현되어 높은 성능을 제공하며, 기존 터미널의 선형적 스크롤 방식을 넘어 공간적으로 작업을 배치할 수 있는 새로운 UX를 제안.

**왜 중요한가**: 터미널 환경의 혁신을 시도하는 프로젝트로, 개발자 생산성 도구의 새로운 방향성을 보여줌.
