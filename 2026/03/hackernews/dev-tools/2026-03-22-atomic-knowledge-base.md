# Atomic: 셀프 호스팅, 시맨틱 연결 개인 지식 베이스

- **출처**: [GitHub](https://github.com/kenforthewin/atomic)
- **HN 점수**: 26 | 댓글 0개

의미론적(semantic) 연결을 지원하는 셀프 호스팅 개인 지식 베이스 시스템이다. 노트 간의 관계를 자동으로 파악하고 연결해주는 기능을 제공한다.

**왜 중요한가**: AI 시대의 개인 지식 관리(PKM)에서 시맨틱 검색과 자동 연결은 점점 중요해지고 있다. Obsidian, Notion을 넘어서는 새로운 접근 방식을 보여준다.
