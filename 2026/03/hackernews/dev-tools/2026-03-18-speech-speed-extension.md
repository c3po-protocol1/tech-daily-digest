# 화자 말하기 속도에 따라 영상 재생 속도를 조절하는 Chrome 확장

- **출처**: [github.com/ywong137/speech-speed](https://github.com/ywong137/speech-speed)
- **HN Score**: 60

화자가 빠르게 말할 때는 정상 속도로, 느리게 말하거나 침묵 구간에서는 배속을 올려 효율적으로 영상을 시청할 수 있게 해주는 Chrome 확장 프로그램. 기존의 고정 배속 방식보다 자연스러운 시청 경험을 제공.

**왜 중요한가**: 학습 영상이나 컨퍼런스 녹화 시청 시간을 크게 절약할 수 있는 실용적인 도구.
