# Oxyde: Rust 코어 기반 Pydantic 네이티브 비동기 ORM

- **출처**: [https://github.com/mr-fatalyst/oxyde](https://github.com/mr-fatalyst/oxyde)

Pydantic 모델을 직접 ORM 스키마로 사용할 수 있는 비동기 ORM으로, 핵심 엔진이 Rust로 작성되어 높은 성능을 제공한다. Python의 타입 안전성과 Rust의 속도를 결합하여 FastAPI 등과의 연동이 매끄럽다.

**왜 중요한가:** Python-Rust 하이브리드 패턴이 ORM 영역까지 확장되었으며, 성능이 중요한 AI 백엔드 서비스에서 검토할 가치가 있다.

**태그:** Python, Rust, ORM, Pydantic, 비동기
