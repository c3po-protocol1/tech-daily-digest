# C++26: Oxford Variadic Comma

- **출처:** [sandordargo.com](https://www.sandordargo.com/blog/2026/03/11/cpp26-oxford-variadic-comma)
- **점수:** 92 | **댓글:** 47

C++26에 "Oxford Variadic Comma" 기능이 추가된다. 가변 인자 템플릿에서의 쉼표 처리를 더 일관성 있게 만들어, 매크로와 템플릿 메타프로그래밍에서 흔히 발생하던 문법적 불편함을 해소한다.

**왜 중요한가:** C++ 표준의 지속적 현대화가 이루어지고 있으며, 가변 인자 처리의 개선은 라이브러리 개발자들에게 실질적인 편의를 제공한다.
