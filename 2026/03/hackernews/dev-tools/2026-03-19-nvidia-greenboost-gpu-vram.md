# Nvidia Greenboost: Extend GPU VRAM Using System RAM/NVMe

- **출처:** https://gitlab.com/IsolatedOctopi/nvidia_greenboost
- **날짜:** 2026-03-19

## 요약

GPU VRAM 부족 문제를 시스템 RAM이나 NVMe 스토리지로 투명하게 확장하는 오픈소스 도구. NVIDIA GPU에서 대규모 모델을 제한된 VRAM으로도 실행할 수 있게 해줌.

## 왜 중요한가

로컬 AI 모델 실행 시 VRAM 제약을 완화하는 실용적 솔루션으로, 개발 환경 구축에 유용
