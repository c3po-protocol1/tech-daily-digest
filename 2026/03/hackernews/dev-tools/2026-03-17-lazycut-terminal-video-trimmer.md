# Lazycut: FFmpeg 기반 터미널 비디오 트리머

- **출처**: [https://github.com/emin-ozata/lazycut](https://github.com/emin-ozata/lazycut)

터미널에서 FFmpeg를 활용해 간편하게 비디오를 트리밍할 수 있는 도구 Lazycut이 공개됐다. TUI 인터페이스로 영상의 시작/끝 지점을 설정하고, 재인코딩 없이 빠르게 잘라낼 수 있다.

**왜 중요한가:** CLI 기반 영상 편집 도구로, 개발자 워크플로우에 자연스럽게 통합할 수 있어 자동화 파이프라인 구성 시 유용하다.

**태그:** FFmpeg, CLI, 비디오편집, TUI, 오픈소스
