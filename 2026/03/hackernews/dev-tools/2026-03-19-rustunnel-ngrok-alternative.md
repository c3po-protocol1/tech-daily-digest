# Rustunnel: ngrok-style Secure Tunnel Server in Rust

- **출처:** https://github.com/joaoh82/rustunnel
- **날짜:** 2026-03-19

## 요약

Rust로 작성된 ngrok 스타일의 오픈소스 보안 터널 서버. 로컬 서비스를 외부에 안전하게 노출할 수 있으며, 자체 호스팅이 가능한 경량 솔루션.

## 왜 중요한가

자체 호스팅 가능한 터널링 솔루션으로, 개발 및 데모 환경 구축에 실용적인 도구
