# Termcraft: Rust로 만든 터미널 기반 2D 샌드박스 서바이벌 게임

- **출처**: [GitHub](https://github.com/pagel-s/termcraft)
- **HN 점수**: 62 | 댓글 6개

터미널에서 실행되는 2D 샌드박스 서바이벌 게임 Termcraft가 Show HN으로 공개되었다. Rust로 작성되었으며, Minecraft 스타일의 게임플레이를 터미널 환경에서 구현했다.

**왜 중요한가**: 터미널 기반 UI의 가능성을 보여주는 창의적인 프로젝트. Rust의 TUI 생태계 성장을 반영하며, 개발자 문화에서 터미널의 위상을 보여준다.
