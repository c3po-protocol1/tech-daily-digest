# Tooscut: WebGPU와 WASM으로 브라우저에서 전문 비디오 편집

- **출처**: [tooscut.app](https://tooscut.app/)
- **HN 점수**: 39 | 댓글 10개

WebGPU와 WebAssembly를 활용하여 브라우저 내에서 전문급 비디오 편집을 가능하게 하는 웹 앱이다. 별도 소프트웨어 설치 없이 브라우저만으로 고급 비디오 편집이 가능하다.

**왜 중요한가**: WebGPU의 실제 활용 사례로서, 브라우저 기반 전문 도구의 가능성을 보여준다. 클라우드가 아닌 클라이언트 사이드 컴퓨팅의 미래를 엿볼 수 있다.
