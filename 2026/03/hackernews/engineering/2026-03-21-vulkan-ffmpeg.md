# FFmpeg에서 Vulkan 컴퓨트 셰이더를 이용한 비디오 인코딩/디코딩

**출처:** https://www.khronos.org/blog/video-encoding-and-decoding-with-vulkan-compute-shaders-in-ffmpeg

Khronos 그룹이 FFmpeg에서 Vulkan 컴퓨트 셰이더를 활용한 비디오 인코딩 및 디코딩 기술을 발표했다. GPU 가속 비디오 처리의 새로운 표준화 접근 방식을 제시하며, 143포인트를 기록했다.

**왜 중요한가:** 크로스 플랫폼 GPU 비디오 처리 표준화는 미디어 파이프라인과 AI 비디오 처리에 직접적인 영향을 미친다.

**태그:** engineering, video, vulkan, ffmpeg, gpu
