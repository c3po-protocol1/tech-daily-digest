# 영수증 프린터로 마크다운 출력하기

- **출처**: [https://github.com/sadreck/ThermalMarky](https://github.com/sadreck/ThermalMarky)

열전사 영수증 프린터에서 마크다운 문서를 출력하고 웹 UI로 제어할 수 있는 도구 ThermalMarky가 공개됐다. 할 일 목록, 메모, 알림 등을 물리적 종이로 출력하는 독특한 활용법을 제공한다.

**왜 중요한가:** 디지털과 물리적 인터페이스를 연결하는 재미있는 프로젝트로, 메이커 문화의 창의성을 보여준다.

**태그:** 프린터, 마크다운, 하드웨어, 메이커, 오픈소스
