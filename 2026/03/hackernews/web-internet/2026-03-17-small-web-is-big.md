# '작은 웹'은 생각보다 크다

- **출처**: [https://kevinboone.me/small_web_is_big.html](https://kevinboone.me/small_web_is_big.html)

개인 블로그, 소규모 포럼, 정적 사이트 등으로 구성된 '스몰 웹'의 규모가 실제로는 상당히 크다는 분석이다. 대형 플랫폼 밖에서도 활발한 웹 생태계가 존재하며, 검색 엔진이 이를 제대로 인덱싱하지 못하고 있다는 점을 지적한다.

**왜 중요한가:** 플랫폼 중심 인터넷에 대한 반성과 분산형 웹의 가치를 재조명하는 시의적절한 논의다.

**태그:** 스몰웹, 인터넷, 분산형, 블로그, 웹생태계
