# Signet - 위성/기상 데이터 기반 자율 산불 추적 시스템

- **출처:** [signet.watch](https://signet.watch)
- **점수:** 102 | **댓글:** 28

위성 이미지와 기상 데이터를 활용하여 자율적으로 산불을 추적하는 시스템 Signet이 공개되었다. 실시간 데이터 분석을 통해 산불의 확산 경로를 예측하고, 조기 경보를 제공하는 것을 목표로 한다.

**왜 중요한가:** 기후변화로 산불 위험이 증가하는 상황에서, AI/데이터 기반의 재난 대응 기술은 사회적으로 매우 중요한 응용 분야다.
