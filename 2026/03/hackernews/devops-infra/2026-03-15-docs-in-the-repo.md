# 이제 문서를 저장소 안으로 옮길 때다

- **출처:** [Hacker News](https://www.dein.fr/posts/2026-03-13-its-time-to-move-your-docs-in-the-repo)
- **점수:** 83 | **댓글:** 54

프로젝트 문서를 별도 위키나 플랫폼이 아닌 코드 저장소 내에 함께 관리해야 한다는 주장이다. 코드와 문서의 동기화, 버전 관리, PR 리뷰 프로세스 통합 등의 이점을 설명한다. 특히 AI 에이전트가 코드와 문서를 함께 이해할 수 있다는 점에서 시의적절한 논의다.

**왜 중요한가:** AI 코딩 에이전트 시대에 문서와 코드의 공존은 에이전트의 컨텍스트 이해도를 크게 높여준다.
