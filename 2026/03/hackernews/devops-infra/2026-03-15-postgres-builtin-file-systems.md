# Postgres에 내장 파일 시스템 도입

- **출처:** [Hacker News](https://db9.ai/)
- **점수:** 8 | **댓글:** 0

PostgreSQL에 내장 파일 시스템을 통합하는 프로젝트다. 데이터베이스와 파일 스토리지를 하나의 시스템에서 관리할 수 있게 하여, 애플리케이션 아키텍처를 단순화하는 것을 목표로 한다.

**왜 중요한가:** PostgreSQL 생태계의 확장은 데이터 인프라 설계에서 새로운 선택지를 제공한다.
