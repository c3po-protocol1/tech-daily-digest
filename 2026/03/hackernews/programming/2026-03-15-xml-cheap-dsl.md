# XML은 값싼 DSL이다

- **출처:** [Hacker News](https://unplannedobsolescence.com/blog/xml-cheap-dsl/)
- **점수:** 220 | **댓글:** 232

XML을 도메인 특화 언어(DSL)로 활용하는 방법에 대한 실용적 분석이다. 커스텀 파서를 만드는 대신 XML의 기존 도구 생태계를 활용하면 비용 효율적으로 DSL을 구축할 수 있다는 주장을 펼친다. 232개의 활발한 댓글이 달릴 정도로 개발자 커뮤니티에서 큰 논쟁을 불러일으켰다.

**왜 중요한가:** DSL 설계는 복잡한 도메인 로직을 다루는 모든 프로젝트에서 중요한 아키텍처 결정이다.
