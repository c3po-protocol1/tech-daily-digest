# Get Shit Done: Meta-Prompting, Context Engineering & Spec-Driven Dev System

- **출처**: [github.com/gsd-build](https://github.com/gsd-build/get-shit-done)
- **HN Score**: 118

메타 프롬프팅과 컨텍스트 엔지니어링을 활용한 스펙 기반 개발 시스템. AI 에이전트에게 명확한 스펙을 제공하고, 컨텍스트를 체계적으로 관리하여 대규모 프로젝트에서도 일관된 코드를 생성하도록 하는 프레임워크.

**왜 중요한가**: 프롬프트 엔지니어링을 넘어 '컨텍스트 엔지니어링'이라는 개념이 실무 개발 워크플로우에 녹아드는 사례.
