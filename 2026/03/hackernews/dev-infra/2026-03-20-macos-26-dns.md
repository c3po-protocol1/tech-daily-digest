# macOS 26, .internal 포함 커스텀 DNS 설정 깨뜨림

- **출처:** [GitHub Gist](https://gist.github.com/adamamyl/81b78eced40feae50eae7c4f3bec1f5a)
- **HN 점수:** 296 | 댓글: 143

macOS 26 업데이트가 .internal TLD를 포함한 커스텀 DNS 설정을 깨뜨리는 문제가 보고되었다. 로컬 개발 환경이나 사내 네트워크에서 .internal 도메인을 사용하는 개발자들에게 직접적인 영향을 미친다.

**왜 중요한가:** macOS 업데이트가 개발 환경을 파괴하는 사례가 반복되고 있으며, 로컬 DNS 설정에 의존하는 워크플로우를 점검해야 한다.
