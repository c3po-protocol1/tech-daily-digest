# SSH Has No Host Header

- **출처:** https://blog.exe.dev/ssh-host-header
- **날짜:** 2026-03-19

## 요약

SSH 프로토콜에 HTTP의 Host 헤더에 해당하는 메커니즘이 없어 발생하는 문제들을 분석한 기술 글. 하나의 IP에서 여러 SSH 서비스를 운영할 때의 제약과 해결 방법을 다룸.

## 왜 중요한가

인프라 엔지니어링에서 간과하기 쉬운 SSH 프로토콜의 근본적 한계를 이해하는 데 유용
