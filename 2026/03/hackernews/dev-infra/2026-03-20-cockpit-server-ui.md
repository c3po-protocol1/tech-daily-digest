# Cockpit: 서버를 위한 웹 기반 관리 인터페이스

- **출처:** [GitHub - cockpit-project/cockpit](https://github.com/cockpit-project/cockpit)
- **HN 점수:** 100 | 댓글: 52

Cockpit은 웹 브라우저를 통해 Linux 서버를 관리할 수 있는 오픈소스 도구다. 시스템 모니터링, 스토리지 관리, 네트워크 설정, 컨테이너 관리 등을 GUI로 제공하며, SSH 터미널도 내장되어 있다.

**왜 중요한가:** 서버 관리의 진입 장벽을 낮추면서도 전문가에게도 유용한 도구로, 홈랩이나 소규모 인프라 관리에 특히 효과적이다.
