# //go:fix inline과 소스 레벨 인라이너

- **출처:** [go.dev/blog/inliner](https://go.dev/blog/inliner)
- **점수:** 82 | **댓글:** 21

Go 언어에 `//go:fix inline` 디렉티브와 소스 레벨 인라이너가 도입되었다. 이를 통해 API 마이그레이션 시 함수 호출을 자동으로 인라인 변환할 수 있어, 대규모 코드베이스의 점진적 리팩토링이 크게 수월해진다.

**왜 중요한가:** Go의 도구 체계가 더욱 강화되고 있으며, 대규모 코드베이스 관리에서 자동화된 리팩토링 지원이 점점 중요해지고 있다.
