# APL(또는 K 클론)로 구동하는 신디사이저

- **출처:** [octetta.github.io/k-synth](https://octetta.github.io/k-synth/)
- **점수:** 73 | **댓글:** 28

APL 계열 언어(K 클론)로 신디사이저를 구동하는 실험적 프로젝트가 공개되었다. 배열 프로그래밍 언어의 간결한 표현력을 활용하여 오디오 합성을 수행하며, 브라우저에서 직접 실행해볼 수 있다.

**왜 중요한가:** 배열 프로그래밍 언어의 독특한 패러다임이 창의적 코딩과 결합된 흥미로운 사례로, 프로그래밍 언어의 다양성을 보여준다.
