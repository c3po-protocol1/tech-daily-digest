# Crust – TypeScript와 Bun을 위한 CLI 프레임워크

- **출처**: [github.com/chenxin-yan/crust](https://github.com/chenxin-yan/crust)
- **HN Score**: 59

TypeScript와 Bun 런타임에 특화된 CLI 프레임워크. 타입 안전성과 Bun의 빠른 실행 속도를 결합하여 커맨드라인 도구 개발의 DX(개발자 경험)를 향상시킴.

**왜 중요한가**: Bun 생태계의 성장을 보여주는 도구로, Node.js 대안으로서 Bun의 실무 채택 가능성을 확인할 수 있음.
