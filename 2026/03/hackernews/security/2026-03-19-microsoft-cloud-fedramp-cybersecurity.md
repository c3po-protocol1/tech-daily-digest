# Despite Doubts, Federal Cyber Experts Approved Microsoft Cloud Service

- **출처:** https://www.propublica.org/article/microsoft-cloud-fedramp-cybersecurity-government
- **날짜:** 2026-03-19

## 요약

연방 사이버보안 전문가들이 보안 우려에도 불구하고 Microsoft 클라우드 서비스를 승인한 내막을 ProPublica가 보도. 정치적 압력과 관료적 프로세스가 보안 판단을 왜곡할 수 있음을 시사함.

## 왜 중요한가

클라우드 보안 인증 프로세스의 신뢰성에 대한 의문을 제기하며, 정부 IT 조달의 보안 거버넌스 문제를 부각
