# 머신러닝 시각적 입문서 (2015)

- **출처:** [r2d3.us](https://r2d3.us/visual-intro-to-machine-learning-part-1/)
- **점수:** 301 | **댓글:** 29

2015년에 공개된 머신러닝 시각적 입문 가이드가 다시 주목받고 있다. 의사결정 트리를 중심으로 ML의 핵심 개념을 인터랙티브 시각화로 설명하며, 데이터 분류와 과적합 문제를 직관적으로 이해할 수 있게 해준다.

**왜 중요한가:** LLM 시대에도 기초 ML 개념의 중요성은 변하지 않으며, 이 자료는 여전히 최고의 ML 입문 콘텐츠 중 하나로 평가받고 있다.
