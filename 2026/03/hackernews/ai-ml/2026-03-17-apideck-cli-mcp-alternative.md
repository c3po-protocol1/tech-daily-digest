# Apideck CLI: MCP보다 컨텍스트 소비가 적은 AI 에이전트 인터페이스

- **출처**: [https://www.apideck.com/blog/mcp-server-eating-context-window-cli-alternative](https://www.apideck.com/blog/mcp-server-eating-context-window-cli-alternative)

MCP(Model Context Protocol) 서버가 컨텍스트 윈도우를 과도하게 소비하는 문제를 지적하며, CLI 기반의 대안 접근법을 제시하는 Apideck CLI가 공개됐다. API 통합을 위한 에이전트 인터페이스로, MCP 대비 훨씬 적은 토큰으로 동일한 기능을 수행한다.

**왜 중요한가:** MCP의 실질적 한계를 짚으며 대안을 제시하는 점에서, 에이전트 도구 통합 아키텍처의 진화 방향을 보여준다.

**태그:** MCP, 에이전트, CLI, API통합, 컨텍스트윈도우
