# Kagi Small Web

- **출처**: [kagi.com](https://kagi.com/smallweb/)
- **HN Score**: 665

Kagi 검색엔진의 Small Web 이니셔티브. 개인 블로그와 소규모 웹사이트를 우선적으로 검색 결과에 노출시키는 프로젝트로, 상업적 SEO에 지배되지 않는 인간 중심의 웹 생태계를 지향함.

**왜 중요한가**: AI 시대에 오히려 개인 콘텐츠와 인간이 직접 작성한 글의 가치가 재조명되는 흐름을 보여줌.
