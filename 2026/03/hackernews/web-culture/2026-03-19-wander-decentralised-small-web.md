# Wander – A Tiny, Decentralised Tool to Explore the Small Web

- **출처:** https://susam.net/wander/
- **날짜:** 2026-03-19

## 요약

분산형으로 설계된 '스몰 웹' 탐색 도구. 대형 플랫폼에 종속되지 않는 개인 웹사이트와 블로그를 발견하고 탐험할 수 있게 해주는 경량 도구.

## 왜 중요한가

인디 웹 운동의 일환으로, 대형 플랫폼 중심의 인터넷에 대한 대안적 접근
