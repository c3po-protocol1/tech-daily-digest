# Home Assistant로 식물 자동 급수 시스템 구축

- **출처**: [https://finnian.io/blog/home-assistant-waters-my-plants/](https://finnian.io/blog/home-assistant-waters-my-plants/)

Home Assistant와 토양 수분 센서를 연동하여 식물에 자동으로 물을 주는 시스템을 구축한 프로젝트다. 센서 데이터 기반 자동화 규칙 설정, 하드웨어 구성, 실제 운용 경험을 공유한다.

**왜 중요한가:** IoT 자동화의 실용적 활용 사례로, 센서-액추에이터 자동화 패턴을 이해하는 좋은 입문 프로젝트다.

**태그:** HomeAssistant, IoT, 자동화, 센서, 스마트홈
