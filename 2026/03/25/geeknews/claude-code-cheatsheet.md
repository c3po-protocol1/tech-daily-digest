## 핵심 요약

Claude Code v2.1.81의 주요 명령어, 단축키, 설정, 환경 변수, MCP 서버 및 에이전트 구성을 정리한 개발자용 요약 문서이다. 새 버전의 **헤드리스 모드(--bare)**, **채널 미리보기(--channels)** 등 실용적 기능을 한눈에 파악할 수 있다.

특히 유용한 항목: 환경 변수 설정(CLAUDE_CODE_MAX_TURNS, CLAUDE_CODE_USE_BEDROCK 등), MCP 서버 연동 방법, 에이전트 모드 설정, 서브에이전트 관리 등이 체계적으로 정리되어 있다.

Claude Code를 일상적으로 사용하는 개발자에게 **"필요할 때 바로 찾아볼 수 있는"** 실전 참고 자료이다.

## Soo에게 의미 있는 이유

AInD 컨설팅에서 Claude Code 교육 시 이런 치트시트가 온보딩 자료로 유용하다. 새로 AI 코딩 도구를 도입하는 팀에게 핵심 명령어와 설정을 한 페이지로 제공하면 도입 초기의 마찰을 줄일 수 있다.
