# Claude, 컴퓨터의 마우스·키보드·화면 직접 제어 기능 출시

## 핵심 요약

Anthropic이 Claude Code Desktop과 Cowork을 통해 **실제 컴퓨터의 마우스, 키보드, 화면을 직접 제어하는 기능**을 출시했다. Dispatch와 결합하면 자리를 비운 상태에서도 Claude가 원격으로 컴퓨터를 조작할 수 있다. macOS 먼저 지원.

이것은 AI 에이전트가 API/CLI 수준이 아닌 **GUI 수준에서 컴퓨터를 제어**하는 "Computer Use"의 실질적 상용화이다. 기존에는 CLI 명령과 코드 실행으로 제한되었던 에이전트의 행동 범위가, 이제 인간과 동일한 방식(마우스 클릭, 키보드 입력, 화면 읽기)으로 모든 GUI 애플리케이션을 조작할 수 있게 되었다. 이는 API가 없는 레거시 소프트웨어, 웹 애플리케이션, 데스크톱 앱 등에 대한 자동화를 가능하게 한다.

보안과 안전 측면에서의 함의도 크다. 에이전트가 화면을 보고 마우스를 움직이는 능력은 강력하지만, 의도하지 않은 클릭이나 잘못된 입력의 위험도 높아진다. Anthropic이 이 기능에 어떤 안전 장치를 적용했는지가 향후 에이전트 보안의 기준이 될 것이다.

## Soo에게 의미 있는 이유

Computer Use는 RPA(Robotic Process Automation)의 AI 버전으로, 기업의 레거시 시스템 자동화에 직접 적용 가능하다. API가 없는 사내 시스템을 AI로 자동화하는 것은 AInD 컨설팅의 즉시 활용 가능한 유스케이스이다.
