# 📚 Articles — 별도 요청 심층 분석 자료

Soo가 요청한 개별 주제에 대한 상세 분석 문서.

## Structure
```
articles/
├── {대분류}/
│   └── {YYYY-MM-DD}_{제목-slug}.md
└── README.md
```

## Categories
- **ai-engineering** — AI 에이전트, 하네스 엔지니어링, AInD 관련
- **dev-tools** — 개발 도구, 플러그인, 메타 도구
- **on-device-ai** — 온디바이스/엣지 AI, 로컬 추론
- **productivity** — 생산성, 전략, 리더십, 관리

## Lifecycle
- 매월 마지막 날: 지난 달 작성 문서 리스팅 → Soo 확인 → 삭제 요청 처리
- 삭제 요청이 없는 문서는 영구 보관
