## Monthly Review Lifecycle
- **매월 마지막 날:** 지난 달에 작성된 문서를 리스팅
  - 예: 4월 30일 → 3월 작성 문서 리스팅
  - 예: 5월 31일 → 4월 작성 문서 리스팅
- **Soo가 확인 후:** 삭제 요청한 문서만 제거, 나머지는 영구 보관
- **한 달 숙성 기간** 후 정리 — 작성 직후 판단하지 않음
